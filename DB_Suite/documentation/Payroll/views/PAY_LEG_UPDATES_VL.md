# PAY_LEG_UPDATES_VL

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

- LAST_UPDATE_LOGIN

## Query

```sql
SELECT plu.PAY_STAT_UPDATE_ID PAY_STAT_UPDATE_ID, plu.LEGISLATION_CODE LEGISLATION_CODE, plu.UPDATE_TYPE_CODE UPDATE_TYPE_CODE, plu.UPDATE_FOR_YEAR UPDATE_FOR_YEAR, plu.UPDATE_FOR_MONTH UPDATE_FOR_MONTH, plu.SEQ_FOR_YEAR_MON SEQ_FOR_YEAR_MON, plu.MODULE_ID MODULE_ID, plu.MANDATORY_FLAG MANDATORY_FLAG, plu.APPLY_FLAG APPLY_FLAG, plu.SEED_DATA_SOURCE SEED_DATA_SOURCE, plu.ORA_SEED_SET1 ORA_SEED_SET1, plu.ORA_SEED_SET2 ORA_SEED_SET2, plu.ENTERPRISE_ID ENTERPRISE_ID, plu.CREATED_BY CREATED_BY, plu.CREATION_DATE CREATION_DATE, plu.LAST_UPDATED_BY LAST_UPDATED_BY, plu.LAST_UPDATE_DATE LAST_UPDATE_DATE, plu.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, plutl.NAME NAME FROM PAY_LEG_UPDATES plu, PAY_LEG_UPDATES_TL plutl WHERE plu.PAY_STAT_UPDATE_ID = plutl.PAY_STAT_UPDATE_ID AND plutl.LANGUAGE = USERENV('LANG')
```
