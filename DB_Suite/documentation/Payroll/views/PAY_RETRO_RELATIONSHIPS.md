# PAY_RETRO_RELATIONSHIPS

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payretrorelationships-5670.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payretrorelationships-5670.html)

## Columns

- RETRO_RELATIONSHIP_ID
- PAYROLL_RELATIONSHIP_ID
- REPROCESS_DATE
- START_DATE
- OVERRIDE_PROCESS_DATE
- APPROVAL_STATUS
- RETRO_REL_ACTION_ID
- SUPERSEDING_RETRO_REL_ID
- OWNER_TYPE
- ORIGINAL_OWNER_TYPE
- ENTERPRISE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LAST_UPDATED_BY

## Query

```sql
SELECT PAY_EVENT_RELATIONSHIPS.EVENT_RELATIONSHIP_ID RETRO_RELATIONSHIP_ID, PAY_EVENT_RELATIONSHIPS.PAYROLL_RELATIONSHIP_ID PAYROLL_RELATIONSHIP_ID, PAY_EVENT_RELATIONSHIPS.PROCESS_DATE REPROCESS_DATE, PAY_EVENT_RELATIONSHIPS.START_DATE START_DATE, PAY_EVENT_RELATIONSHIPS.OVERRIDE_PROCESS_DATE OVERRIDE_PROCESS_DATE, PAY_EVENT_RELATIONSHIPS.APPROVAL_STATUS APPROVAL_STATUS, PAY_EVENT_RELATIONSHIPS.PAYROLL_REL_ACTION_ID RETRO_REL_ACTION_ID, PAY_EVENT_RELATIONSHIPS.SUPERSEDING_EVENT_REL_ID SUPERSEDING_RETRO_REL_ID, PAY_EVENT_RELATIONSHIPS.OWNER_TYPE OWNER_TYPE, PAY_EVENT_RELATIONSHIPS.ORIGINAL_OWNER_TYPE ORIGINAL_OWNER_TYPE, PAY_EVENT_RELATIONSHIPS.ENTERPRISE_ID ENTERPRISE_ID, PAY_EVENT_RELATIONSHIPS.CREATED_BY CREATED_BY, PAY_EVENT_RELATIONSHIPS.CREATION_DATE CREATION_DATE, PAY_EVENT_RELATIONSHIPS.LAST_UPDATE_DATE LAST_UPDATE_DATE, PAY_EVENT_RELATIONSHIPS.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, PAY_EVENT_RELATIONSHIPS.LAST_UPDATED_BY LAST_UPDATED_BY FROM PAY_EVENT_RELATIONSHIPS PAY_EVENT_RELATIONSHIPS WHERE PAY_EVENT_RELATIONSHIPS.EVENT_REPORT_TYPE = 'RETRONOT'
```
