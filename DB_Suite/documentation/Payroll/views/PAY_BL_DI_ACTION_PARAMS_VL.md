# PAY_BL_DI_ACTION_PARAMS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybldiactionparamsvl-4901.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybldiactionparamsvl-4901.html)

## Columns

- TASK_ACTION_ID
- ACTION_PARAMETER_ID
- ACTION_PARAMETER_NAME
- DISPLAY_FLAG

## Query

```sql
SELECT PARAMS.TASK_ACTION_ID TASK_ACTION_ID, PARAM_LOVS.ACTION_PARAMETER_ID ACTION_PARAMETER_ID, PARAM_LOVS.DISPLAY_ACTION_PARAMETER_NAME ACTION_PARAMETER_NAME, PARAM_LOVS.DISPLAY_FLAG DISPLAY_FLAG FROM PAY_BL_ACTION_PARAMETERS_VL PARAMS, PAY_BL_ACTION_PARAMETERS_VL PARAM_LOVS WHERE (PARAMS.PARAMETER_NODE_TYPE IN ('L', 'R') AND PARAMS.ACTION_PARAMETER_ID = PARAM_LOVS.ACTION_PARAMETER_ID and PARAMS.TASK_ACTION_ID = PARAM_LOVS.TASK_ACTION_ID) ORDER BY PARAMS.TASK_ACTION_ID, PARAMS.PARAMETER_SEQUENCE, PARAM_LOVS.PARAMETER_SEQUENCE
```
