# PAY_TMPLT_VERSION_F_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paytmpltversionfvl-4291.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paytmpltversionfvl-4291.html)

## Columns

- TMPLT_VERSION_ID
- TEMPLATE_ID
- VERSION
- VERSION_INFO
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- ENTERPRISE_ID

## Query

```sql
SELECT PTV.TMPLT_VERSION_ID TMPLT_VERSION_ID, PTV.TEMPLATE_ID TEMPLATE_ID, PTV.VERSION VERSION, PTVTL.VERSION_INFO VERSION_INFO, PTV.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, PTV.CREATED_BY CREATED_BY, PTV.CREATION_DATE CREATION_DATE, PTV.LAST_UPDATED_BY LAST_UPDATED_BY, PTV.LAST_UPDATE_DATE LAST_UPDATE_DATE, PTV.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, PTV.ENTERPRISE_ID ENTERPRISE_ID FROM PAY_TMPLT_VERSION_F PTV, PAY_TMPLT_VERSION_F_TL PTVTL WHERE PTV.TMPLT_VERSION_ID = PTVTL.TMPLT_VERSION_ID AND PTVTL.LANGUAGE = USERENV('LANG')
```
