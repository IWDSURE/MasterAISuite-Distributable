# PAY_TMPLT_RULE_GROUPS_F_VL

**Object owner:** PAY

**Object type:** VIEW

- EFFECTIVE_END_DATE
- BASE_RULE_GROUP_NAME

- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE

- LAST_UPDATE_LOGIN

- CREATION_DATE
- GROUP_LOCATION
- TEMPLATE_ID

- LEGISLATION_CODE
- ENTERPRISE_ID
- MODULE_ID
- SGUID

## Query

```sql
SELECT PTRG.RULE_GROUP_ID RULE_GROUP_ID, PTRG.EFFECTIVE_START_DATE, PTRG.EFFECTIVE_END_DATE, PTRG.BASE_RULE_GROUP_NAME BASE_RULE_GROUP_NAME, PTRGTL.GROUP_TEXT GROUP_TEXT, PTRG.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, PTRG.LAST_UPDATE_DATE LAST_UPDATE_DATE, PTRG.LAST_UPDATED_BY LAST_UPDATED_BY, PTRG.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, PTRG.CREATED_BY CREATED_BY, PTRG.CREATION_DATE CREATION_DATE, PTRG.GROUP_LOCATION, PTRG.TEMPLATE_ID, PTRG.LEGISLATIVE_DATA_GROUP_ID, PTRG.LEGISLATION_CODE, PTRG.ENTERPRISE_ID, PTRG.MODULE_ID, PTRG.SGUID SGUID FROM PAY_TMPLT_RULE_GROUPS_F PTRG, PAY_TMPLT_RULE_GROUPS_F_TL PTRGTL WHERE PTRG.RULE_GROUP_ID = PTRGTL.RULE_GROUP_ID AND PTRGTL.LANGUAGE = USERENV('LANG')
```
