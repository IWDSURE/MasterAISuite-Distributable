# PAY_CALC_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paycalctypesvl-8570.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paycalctypesvl-8570.html)

## Columns

- CALC_TYPE_ID
- BASE_NAME
- TYPE
- FORMULA_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENTERPRISE_ID
- MODULE_ID
- NAME
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LAST_UPDATED_BY
- SEED_STATUS
- SGUID

## Query

```sql
SELECT pct.CALC_TYPE_ID CALC_TYPE_ID, pct.BASE_NAME BASE_NAME, pct.TYPE TYPE, pct.FORMULA_ID FORMULA_ID, pct.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pct.LEGISLATION_CODE LEGISLATION_CODE, pct.ENTERPRISE_ID ENTERPRISE_ID, pct.MODULE_ID MODULE_ID, pcttl.NAME NAME, pct.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pct.CREATED_BY CREATED_BY, pct.CREATION_DATE CREATION_DATE, pct.LAST_UPDATE_DATE LAST_UPDATE_DATE, pct.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pct.LAST_UPDATED_BY LAST_UPDATED_BY, pct.SEED_STATUS SEED_STATUS, pct.SGUID SGUID FROM PAY_CALC_TYPES pct, PAY_CALC_TYPES_TL pcttl WHERE pcttl.CALC_TYPE_ID = pct.CALC_TYPE_ID AND pcttl.LANGUAGE = USERENV('LANG')
```
