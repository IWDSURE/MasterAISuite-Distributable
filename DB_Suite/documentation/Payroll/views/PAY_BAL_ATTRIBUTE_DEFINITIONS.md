# PAY_BAL_ATTRIBUTE_DEFINITIONS

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalattributedefinitions-6725.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalattributedefinitions-6725.html)

## Columns

- ATTRIBUTE_ID
- BASE_ATTRIBUTE_NAME
- USER_ATTRIBUTE_NAME
- DESCRIPTION
- ALTERABLE
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- MODULE_ID
- BALANCE_CATEGORY_ID
- BALANCE_DIMENSION_ID
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT pbad.ATTRIBUTE_ID ATTRIBUTE_ID, pbad.BASE_ATTRIBUTE_NAME BASE_ATTRIBUTE_NAME, pbadtl.USER_ATTRIBUTE_NAME USER_ATTRIBUTE_NAME, pbadtl.DESCRIPTION DESCRIPTION, pbad.ALTERABLE ALTERABLE, pbad.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pbad.LEGISLATION_CODE LEGISLATION_CODE, pbad.MODULE_ID MODULE_ID, pbad.BALANCE_CATEGORY_ID BALANCE_CATEGORY_ID, pbad.BALANCE_DIMENSION_ID BALANCE_DIMENSION_ID, pbad.LAST_UPDATE_DATE LAST_UPDATE_DATE, pbad.LAST_UPDATED_BY LAST_UPDATED_BY, pbad.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pbad.CREATED_BY CREATED_BY, pbad.CREATION_DATE CREATION_DATE, pbad.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM PAY_BAL_ATT_DEFINITIONS pbad, PAY_BAL_ATT_DEFINITIONS_TL pbadtl WHERE pbadtl.ATTRIBUTE_ID = pbad.ATTRIBUTE_ID AND pbadtl.LANGUAGE = USERENV('LANG') WITH CHECK OPTION
```
