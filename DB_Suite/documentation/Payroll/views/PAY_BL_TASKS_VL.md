# PAY_BL_TASKS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybltasksvl-4655.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybltasksvl-4655.html)

## Columns

- TASK_ID
- TASK_NAME
- TASK_TYPE
- TASK_DETAILS
- TASK_CONTEXT
- DISPLAY_TASK_NAME
- LEGISLATIVE_DATA_GROUP_ID

## Query

```sql
SELECT pay_tasks.base_task_id task_id, MAX(DECODE(PAY_TASK_PROPERTIES.TASK_PROPERTY_TYPE, 'PLSQL',PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE, 'SERVICE_NAME', DECODE (DECODE(PAY_TASK_ACTIONS.ACTION_TYPE,'VIEW_OBJECT', 'APPLICATION_MODULE', 'LOV_VIEW_OBJECT', 'APPLICATION_MODULE', 'TABLE_VIEW', 'APPLICATION_MODULE', PAY_TASK_ACTIONS.ACTION_TYPE),'APPLICATION_MODULE', NULL,PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE), 'AM_INSTANCE_NAME', PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE,'VIEW_NAME1',PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE, NULL)) TASK_NAME, DECODE(PAY_TASK_ACTIONS.ACTION_TYPE,'VIEW_OBJECT', 'APPLICATION_MODULE', 'LOV_VIEW_OBJECT', 'APPLICATION_MODULE', PAY_TASK_ACTIONS.ACTION_TYPE) TASK_TYPE, MAX(DECODE(PAY_TASK_PROPERTIES.TASK_PROPERTY_TYPE, 'SERVICE_STATIC_WSDL_LOCATION', PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE, 'AM_FULL_NAME', PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE, NULL)) TASK_DETAILS, MAX(DECODE(PAY_TASK_PROPERTIES.TASK_PROPERTY_TYPE, 'SERVICE_NAME_SPACE', DECODE(DECODE(PAY_TASK_ACTIONS.ACTION_TYPE, 'VIEW_OBJECT', 'APPLICATION_MODULE', 'LOV_VIEW_OBJECT', 'APPLICATION_MODULE', 'TABLE_VIEW', 'APPLICATION_MODULE', PAY_TASK_ACTIONS.ACTION_TYPE),'APPLICATION_MODULE', NULL,PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE), 'AM_CONFIGURATION_NAME', PAY_TASK_PROPERTIES.TASK_PROPERTY_VALUE, NULL)) TASK_CONTEXT, PAY_TASKS_TL.TASK_NAME DISPLAY_TASK_NAME, MAX(PAY_TASKS.LEGISLATIVE_DATA_GROUP_ID) LEGISLATIVE_DATA_GROUP_ID FROM PAY_TASKS, PAY_TASK_USAGE, PAY_TASKS_TL, PAY_TASK_ACTIONS, PAY_TASK_PROPERTIES WHERE pay_tasks.base_task_id = pay_task_usage.base_task_id AND pay_task_usage.usage_type = 'BATCH_LOADER' AND pay_tasks_tl.task_id = pay_tasks.task_id AND pay_tasks_tl.language = userenv('LANG') AND pay_task_actions.base_task_id = pay_tasks.base_task_id AND pay_task_properties.base_task_action_id = pay_task_actions.base_task_action_id AND pay_task_properties.task_property_type IN ('PLSQL','VIEW_NAME', 'SERVICE_NAME', 'SERVICE_STATIC_WSDL_LOCATION', 'SERVICE_NAME_SPACE', 'AM_INSTANCE_NAME', 'AM_FULL_NAME', 'AM_CONFIGURATION_NAME') GROUP BY pay_tasks.base_task_id, DECODE(PAY_TASK_ACTIONS.ACTION_TYPE,'VIEW_OBJECT', 'APPLICATION_MODULE', 'LOV_VIEW_OBJECT', 'APPLICATION_MODULE', PAY_TASK_ACTIONS.ACTION_TYPE), PAY_TASKS_TL.TASK_NAME
```
