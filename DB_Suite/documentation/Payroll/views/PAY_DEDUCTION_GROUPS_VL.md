# PAY_DEDUCTION_GROUPS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydeductiongroupsvl-7632.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydeductiongroupsvl-7632.html)

## Columns

- DEDUCTION_GROUP_ID
- BASE_NAME
- NAME
- ENTERPRISE_ID
- LEGISLATION_CODE
- BASE_DEDUCTION_GROUP_ID
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE
- OBJECT_VERSION_NUMBER
- SEED_STATUS

## Query

```sql
SELECT pdg.DEDUCTION_GROUP_ID DEDUCTION_GROUP_ID, pdg.BASE_NAME BASE_NAME, pdgtl.NAME NAME, pdg.ENTERPRISE_ID ENTERPRISE_ID, pdg.LEGISLATION_CODE LEGISLATION_CODE, pdg.BASE_DEDUCTION_GROUP_ID BASE_DEDUCTION_GROUP_ID, pdg.MODULE_ID MODULE_ID, pdg.CREATED_BY CREATED_BY, pdg.CREATION_DATE CREATION_DATE, pdg.LAST_UPDATED_BY LAST_UPDATED_BY, pdg.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pdg.LAST_UPDATE_DATE LAST_UPDATE_DATE, pdg.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pdg.SEED_STATUS SEED_STATUS FROM PAY_DEDUCTION_GROUPS pdg, PAY_DEDUCTION_GROUPS_TL pdgtl WHERE pdg.DEDUCTION_GROUP_ID = pdgtl.DEDUCTION_GROUP_ID AND pdgtl.LANGUAGE = USERENV('LANG')
```
