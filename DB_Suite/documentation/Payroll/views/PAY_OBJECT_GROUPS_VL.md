# PAY_OBJECT_GROUPS_VL

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payobjectgroupsvl-6290.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payobjectgroupsvl-6290.html)

## Query

```sql
SELECT pog.OBJECT_GROUP_ID OBJECT_GROUP_ID, pog.OBJECT_GROUP_TYPE_ID OBJECT_GROUP_TYPE_ID, pog.BASE_OBJECT_GROUP_NAME BASE_OBJECT_GROUP_NAME, pog.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pog.LEGISLATION_CODE LEGISLATION_CODE, pog.ENTERPRISE_ID ENTERPRISE_ID, pog.START_DATE START_DATE, pog.END_DATE END_DATE, pog.FORMULA_ID FORMULA_ID, pog.STATIC_DYNAMIC_FLAG STATIC_DYNAMIC_FLAG, pog.SOURCE_TYPE SOURCE_TYPE, pog.SOURCE_ID SOURCE_ID, pog.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pog.CREATED_BY CREATED_BY, pog.CREATION_DATE CREATION_DATE, pog.LAST_UPDATE_DATE LAST_UPDATE_DATE, pog.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pog.LAST_UPDATED_BY LAST_UPDATED_BY, pog.MODULE_ID MODULE_ID, pogtl.OBJECT_GROUP_NAME OBJECT_GROUP_NAME, pogtl.DESCRIPTION DESCRIPTION FROM PAY_OBJECT_GROUPS pog, PAY_OBJECT_GROUPS_TL pogtl WHERE pogtl.OBJECT_GROUP_ID = pog.OBJECT_GROUP_ID AND pogtl.LANGUAGE = USERENV('LANG')
```
