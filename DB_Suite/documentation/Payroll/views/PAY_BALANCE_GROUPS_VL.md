# PAY_BALANCE_GROUPS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalancegroupsvl-7235.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalancegroupsvl-7235.html)

## Columns

- BALANCE_GROUP_ID
- BASE_GROUP_NAME
- GROUP_NAME
- DESCRIPTION
- ENTERPRISE_ID
- LEGISLATION_CODE
- LEGISLATIVE_DATA_GROUP_ID
- MODULE_ID
- BALANCE_CATEGORY_ATT_FLAG
- BALANCE_DIMENSION_ATT_FLAG
- GROUP_LEVEL_FLAG
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT pbg.BALANCE_GROUP_ID BALANCE_GROUP_ID, pbg.BASE_GROUP_NAME BASE_GROUP_NAME, pbgtl.GROUP_NAME GROUP_NAME, pbgtl.DESCRIPTION DESCRIPTION, pbg.ENTERPRISE_ID ENTERPRISE_ID, pbg.LEGISLATION_CODE LEGISLATION_CODE, pbg.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pbg.MODULE_ID MODULE_ID, pbg.BALANCE_CATEGORY_ATT_FLAG BALANCE_CATEGORY_ATT_FLAG, pbg.BALANCE_DIMENSION_ATT_FLAG BALANCE_DIMENSION_ATT_FLAG, pbg.GROUP_LEVEL_FLAG GROUP_LEVEL_FLAG, pbg.LAST_UPDATE_DATE LAST_UPDATE_DATE, pbg.LAST_UPDATED_BY LAST_UPDATED_BY, pbg.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pbg.CREATED_BY CREATED_BY, pbg.CREATION_DATE CREATION_DATE, pbg.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM PAY_BALANCE_GROUPS pbg, PAY_BALANCE_GROUPS_TL pbgtl WHERE pbgtl.BALANCE_GROUP_ID = pbg.BALANCE_GROUP_ID AND pbgtl.LANGUAGE = USERENV('LANG')
```
