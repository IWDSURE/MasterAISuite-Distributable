# PAY_EVENT_GROUPS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payeventgroupsvl-3940.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payeventgroupsvl-3940.html)

## Columns

- EVENT_GROUP_ID
- BASE_EVENT_GROUP_NAME
- EVENT_GROUP_NAME
- EVENT_GROUP_TYPE
- PRORATION_TYPE
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENT_SUBSCRIPTION_SET_ID
- LDG_SUBSCRIPTION_SET_ID
- ENTERPRISE_ID
- TIME_DEFINITION_ID
- CREATION_DATE
- CREATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LAST_UPDATED_BY
- OBJECT_VERSION_NUMBER
- MODULE_ID
- DESCRIPTION

## Query

```sql
SELECT B.EVENT_GROUP_ID, B.BASE_EVENT_GROUP_NAME, TL.EVENT_GROUP_NAME, B.EVENT_GROUP_TYPE, B.PRORATION_TYPE, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.ENT_SUBSCRIPTION_SET_ID ENT_SUBSCRIPTION_SET_ID, B.LDG_SUBSCRIPTION_SET_ID LDG_SUBSCRIPTION_SET_ID, B.ENTERPRISE_ID, B.TIME_DEFINITION_ID, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.MODULE_ID MODULE_ID, TL.DESCRIPTION FROM PAY_EVENT_GROUPS B, PAY_EVENT_GROUPS_TL TL WHERE B.EVENT_GROUP_ID = TL.EVENT_GROUP_ID AND TL.LANGUAGE = USERENV('LANG')
```
