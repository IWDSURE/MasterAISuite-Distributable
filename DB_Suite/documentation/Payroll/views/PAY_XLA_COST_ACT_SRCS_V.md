# PAY_XLA_COST_ACT_SRCS_V

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payxlacostactsrcsv-3859.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/payxlacostactsrcsv-3859.html)

## Columns

- PAY_XLA_EVENT_ID
- COST_PROCESS_FLOW_NAME
- COST_PROCESS_NAME
- BASE_PROCESS_FLOW_NAME
- BASE_PROCESS_NAME

## Query

```sql
SELECT PXE.EVENT_ID PAY_XLA_EVENT_ID, PAY_SLA_PKG.GET_FLOW_NAME(COST_PA.PAYROLL_ACTION_ID) COST_PROCESS_FLOW_NAME, PAY_SLA_PKG.GET_PROCESS_NAME(COST_PA.ACTION_TYPE) COST_PROCESS_NAME, PAY_SLA_PKG.GET_FLOW_NAME(RUN_PA.PAYROLL_ACTION_ID) BASE_PROCESS_FLOW_NAME, PAY_SLA_PKG.GET_PROCESS_NAME(RUN_PA.ACTION_TYPE) BASE_PROCESS_NAME FROM PAY_XLA_EVENTS PXE, PAY_PAYROLL_REL_ACTIONS COST_RA, PAY_PAYROLL_ACTIONS COST_PA, PAY_PAYROLL_REL_ACTIONS RUN_RA, PAY_PAYROLL_ACTIONS RUN_PA WHERE COST_RA.PAYROLL_REL_ACTION_ID = PXE.COST_ACTION_ID AND COST_PA.PAYROLL_ACTION_ID = COST_RA.PAYROLL_ACTION_ID AND RUN_RA.PAYROLL_REL_ACTION_ID = PXE.RUN_ACTION_ID AND RUN_PA.PAYROLL_ACTION_ID = RUN_RA.PAYROLL_ACTION_ID AND PXE.COST_TYPE IN ('C', 'S', 'CA')
```
