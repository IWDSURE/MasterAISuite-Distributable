# PAY_PAYMENT_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paypaymenttypesvl-3392.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paypaymenttypesvl-3392.html)

## Columns

- PAYMENT_TYPE_ID
- BASE_PAYMENT_TYPE_NAME
- PAYMENT_TYPE_NAME
- DESCRIPTION
- LEGISLATION_CODE
- ENTERPRISE_ID
- CATEGORY
- ALLOW_AS_DEFAULT
- BASE_PAYMENT_TYPE_ID
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT ppt.PAYMENT_TYPE_ID PAYMENT_TYPE_ID, ppt.BASE_PAYMENT_TYPE_NAME BASE_PAYMENT_TYPE_NAME, ppttl.PAYMENT_TYPE_NAME PAYMENT_TYPE_NAME , ppttl.DESCRIPTION DESCRIPTION, ppt.LEGISLATION_CODE LEGISLATION_CODE, ppt.ENTERPRISE_ID ENTERPRISE_ID, ppt.CATEGORY CATEGORY, ppt.ALLOW_AS_DEFAULT ALLOW_AS_DEFAULT, ppt.BASE_PAYMENT_TYPE_ID BASE_PAYMENT_TYPE_ID, ppt.MODULE_ID MODULE_ID, ppt.CREATED_BY CREATED_BY, ppt.CREATION_DATE CREATION_DATE, ppt.LAST_UPDATED_BY LAST_UPDATED_BY, ppt.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, ppt.LAST_UPDATE_DATE LAST_UPDATE_DATE, ppt.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM PAY_PAYMENT_TYPES ppt, PAY_PAYMENT_TYPES_TL ppttl WHERE ppttl.PAYMENT_TYPE_ID = ppt.PAYMENT_TYPE_ID AND ppttl.LANGUAGE = USERENV('LANG')
```
