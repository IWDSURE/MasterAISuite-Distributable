# PAY_CHKLST_ANNOTATIONS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paychklstannotationsvl-8313.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paychklstannotationsvl-8313.html)

## Columns

- CHECKLIST_ANNOTATION_ID
- BASE_ANNOTATION_TXT
- BASE_ANNOTATION_ID
- BASE_CHECKLIST_ID
- OWNER_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENTERPRISE_ID
- OBJECT_VERSION_NUMBER
- ANNOTATION_TEXT

## Query

```sql
SELECT B.CHECKLIST_ANNOTATION_ID, B.BASE_ANNOTATION_TXT, B.BASE_ANNOTATION_ID, B.BASE_CHECKLIST_ID, B.OWNER_ID, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE, B.ENTERPRISE_ID, B.OBJECT_VERSION_NUMBER, TL.ANNOTATION_TEXT FROM PAY_CHKLST_ANNOTATIONS B, PAY_CHKLST_ANNOTATIONS_TL TL WHERE B.CHECKLIST_ANNOTATION_ID = TL.CHECKLIST_ANNOTATION_ID AND USERENV('LANG') = TL.LANGUAGE
```
