# PAY_DEDUCTION_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydeductiontypesvl-8051.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydeductiontypesvl-8051.html)

## Columns

- DEDUCTION_TYPE_ID
- DEDUCTION_CODE
- BASE_NAME
- NAME
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENTERPRISE_ID
- DEDUCTION_GROUP_ID
- DISPLAY_SEQUENCE
- USE_ALL_TAX_UNITS
- BASE_DEDUCTION_TYPE_ID
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE
- OBJECT_VERSION_NUMBER
- SEED_STATUS

## Query

```sql
SELECT pdt.DEDUCTION_TYPE_ID DEDUCTION_TYPE_ID, pdt.DEDUCTION_CODE DEDUCTION_CODE, pdt.BASE_NAME BASE_NAME, pdttl.NAME NAME, pdt.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pdt.LEGISLATION_CODE LEGISLATION_CODE, pdt.ENTERPRISE_ID ENTERPRISE_ID, pdt.DEDUCTION_GROUP_ID DEDUCTION_GROUP_ID, pdt.DISPLAY_SEQUENCE DISPLAY_SEQUENCE, pdt.USE_ALL_TAX_UNITS USE_ALL_TAX_UNITS, pdt.BASE_DEDUCTION_TYPE_ID BASE_DEDUCTION_TYPE_ID, pdt.MODULE_ID MODULE_ID, pdt.CREATED_BY CREATED_BY, pdt.CREATION_DATE CREATION_DATE, pdt.LAST_UPDATED_BY LAST_UPDATED_BY, pdt.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pdt.LAST_UPDATE_DATE LAST_UPDATE_DATE, pdt.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pdt.SEED_STATUS SEED_STATUS FROM PAY_DEDUCTION_TYPES pdt, PAY_DEDUCTION_TYPES_TL pdttl WHERE pdt.DEDUCTION_TYPE_ID = pdttl.DEDUCTION_TYPE_ID AND pdttl.LANGUAGE = USERENV('LANG')
```
