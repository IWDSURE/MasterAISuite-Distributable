# PER_ETHNICITIES_

This table stores legislation-specific ethnicity data for a person. May be multiple rows within a Legislation Code

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** TABLE

**Tablespace:** APPS_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perethnicities-17913.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perethnicities-17913.html)

## Primary Key

| Name | Columns |
|------|----------|
| PER_ETHNICITIES_PK_ | LAST_UPDATE_DATE, LAST_UPDATED_BY, ETHNICITY_ID |
