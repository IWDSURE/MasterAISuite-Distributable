# PER_BOOKINGS

This table stores the booking details made for a person to attend an event or interview.

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** TABLE

**Tablespace:** APPS_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perbookings-20454.html](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perbookings-20454.html)

## Primary Key

| Name | Columns |
|------|----------|
| PER_BOOKINGS_PK | BOOKING_ID |
