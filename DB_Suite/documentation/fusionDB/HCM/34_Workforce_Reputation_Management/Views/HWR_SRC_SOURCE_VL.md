# HWR_SRC_SOURCE_VL

## Details

**Schema:** FUSION

**Object owner:** HWR

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrsrcsourcevl-5994.html#hwrsrcsourcevl-5994](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrsrcsourcevl-5994.html#hwrsrcsourcevl-5994)

## Columns

- SOURCE_ID
- SOURCE_TYPE_ID
- SOURCE_TYPE_VERSION_ID
- CONNECTION_SPEC
- IS_INTERNAL
- IS_RECRUITING
- SYNC_STATE_CLOB
- NAME
- DESCRIPTION
- APPLICATION_NAME
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.SOURCE_ID, B.SOURCE_TYPE_ID, B.SOURCE_TYPE_VERSION_ID, B.CONNECTION_SPEC, B.IS_INTERNAL, B.IS_RECRUITING, B.SYNC_STATE_CLOB, T.NAME, T.DESCRIPTION, SRCTYPEB.SOURCE_TYPE_KEY as APPLICATION_NAME, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM HWR_SRC_SOURCE_B B, HWR_SRC_SOURCE_TL T, HWR_SRC_SRCTYPE_B SRCTYPEB WHERE B.SOURCE_ID = T.SOURCE_ID AND SRCTYPEB.SOURCE_TYPE_ID = B.SOURCE_TYPE_ID AND T.LANGUAGE = USERENV('LANG')
```
