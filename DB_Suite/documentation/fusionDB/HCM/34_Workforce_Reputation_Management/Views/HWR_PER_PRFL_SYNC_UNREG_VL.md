## Query

```sql
SELECT B.PRFL_ID, B.SOURCE_ID, B.AUTH_TOKEN, B.REFRESH_TOKEN, B.TOKEN_TYPE, B.AUTH_SECRET, B.AUTH_EXPIRATION_DATE, B.STATE_DATA, B.META_DATA, B.STATE, S.DETAIL_STATE_DATA, B.LAST_DWNLD_DT_PRFL_CORE, B.LAST_DWNLD_DT_PRFL_RLTNS, B.LAST_DWNLD_DT_PRFL_POSTS, B.GBL_PRFL_ID, B.LINKED_PRFL_ID, B.LINKED_SOURCE_ID, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM HWR_PER_PRFL_SYNC B LEFT JOIN HWR_PER_PRFL_SYNC_STATE S ON B.PRFL_ID = S.PRFL_ID AND B.SOURCE_ID = S.SOURCE_ID WHERE B.LINKED_PRFL_ID IS NOT NULL AND B.LINKED_SOURCE_ID IS NOT NULL
```
