# HWR_ISSUE_VL

## Details

**Schema:** FUSION

**Object owner:** HWR

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrissuevl-8401.html#hwrissuevl-8401](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrissuevl-8401.html#hwrissuevl-8401)

## Columns

- GBL_PRFL_ID
- GBL_PRFL_FULL_NAME
- PRFL_ID
- SOURCE_ID
- SOURCE_NAME
- CONTROL_KEY
- CONTROL_NAME
- POST_ID
- POST_MESSAGE
- POST_DATE
- RELEVANCE
- NUM_OCCURRENCES
- ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- ORG_ID

## Query

```sql
SELECT GBL_VL.GBL_PRFL_ID, GBL_VL.NAME AS GBL_PRFL_FULL_NAME, REP_ISSUE_B.PRFL_ID, REP_ISSUE_B.SOURCE_ID, SRC_VL.NAME AS SOURCE_NAME, REP_ISSUE_B.CONTROL_KEY, REP_CNTRL_VL.NAME AS CONTROL_NAME, REP_ISSUE_B.SRC_POST_ID AS POST_ID, REP_ISSUE_B.MESSAGE AS POST_MESSAGE, REP_ISSUE_B.POST_DATE, REP_ISSUE_B.RELEVANCE, REP_ISSUE_B.NUM_OCCURRENCES, REP_ISSUE_B.ID, REP_ISSUE_B.CREATED_BY, REP_ISSUE_B.CREATION_DATE, REP_ISSUE_B.LAST_UPDATED_BY, REP_ISSUE_B.LAST_UPDATE_DATE, REP_ISSUE_B.LAST_UPDATE_LOGIN, REP_ISSUE_B.ORG_ID FROM HWR_PER_GBL_PRFL_VL GBL_VL, HWR_PER_GBL_PRFL_PRFL_X GBL_PRFL_X, HWR_ISSUE_B REP_ISSUE_B, HWR_REP_CONTROL_VL REP_CNTRL_VL, HWR_SRC_SOURCE_VL SRC_VL WHERE REP_ISSUE_B.PRFL_ID = GBL_PRFL_X.PRFL_ID AND REP_ISSUE_B.SOURCE_ID = GBL_PRFL_X.SOURCE_ID AND GBL_PRFL_X.GBL_PRFL_ID = GBL_VL.GBL_PRFL_ID AND REP_ISSUE_B.SOURCE_ID = SRC_VL.SOURCE_ID AND REP_ISSUE_B.CONTROL_KEY = REP_CNTRL_VL.REPUTATION_CONTROL_KEY
```
