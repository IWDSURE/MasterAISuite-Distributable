## Query

```sql
SELECT B.VOLUNTEER_ID VOLUNTEER_ID, T.FIRST_NAME FIRST_NAME, T.MIDDLE_NAME MIDDLE_NAME, T.LAST_NAME LAST_NAME, T.ALIAS ALIAS, T.GENDER_CODE GENDER_CODE, T.EMAIL EMAIL, T.CURRENT_STATUS CURRENT_STATUS, T.INTERESTS INTERESTS, T.LANGUAGES LANGUAGES, T.SKILLS SKILLS, T.SPECIALTIES SPECIALTIES, T.TITLE TITLE, T.PROJECT_LEAD_INTEREST, B.USER_NAME USER_NAME, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.IS_INTERESTED_IN_PROJECT_LEAD IS_INTERESTED_IN_PROJECT_LEAD, B.HCM_PERSON_ID HCM_PERSON_ID, B.T_SIZE T_SIZE, B.T_STYLE T_STYLE, B.CT_USER_ID, B.IS_AGRMNT_ACCEPTED FROM HWR_VLTR_PER_PROFILE_B B, HWR_VLTR_PER_PROFILE_TL T WHERE B.VOLUNTEER_ID = T.VOLUNTEER_ID AND T.LANGUAGE = USERENV('LANG')
```
