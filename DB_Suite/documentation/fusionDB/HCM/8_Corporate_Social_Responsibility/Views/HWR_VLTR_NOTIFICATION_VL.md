## Query

```sql
SELECT A.RUN_RECIPIENT_ID RUN_RECIPIENT_ID, A.RECIPIENT_ID RECIPIENT_ID, A.STATUS RECIPIENT_ALERT_STATUS, A.SENT_TO SENT_TO, A.SENT_ON SENT_ON, A.ENTRY_ID ENTRY_ID, A.HASH_KEY HASH_KEY, B.RUN_MESSAGE_ID RUN_MESSAGE_ID, B.MESSAGE_SEQ MESSAGE_SEQ, B.RUN_ID RUN_ID, B.MESSAGE_LANGUAGE MESSAGE_LANGUAGE, B.SHORT_MESSAGE SHORT_MESSAGE, B.SUBJECT SUBJECT, B.BODY BODY, B.TEMPLATE_ID TEMPLATE_ID, B.STATUS MESSAGE_ALERT_STATUS, B.TOTAL_SUCCESS_RECIPIENTS TOTAL_SUCCESS_RECIPIENTS, B.TOTAL_ERROR_RECIPIENTS TOTAL_ERROR_RECIPIENTS, A.CREATED_BY CREATED_BY, C.ALERT_ID ALERT_ID, C.NAME NAME FROM HRC_ALERT_RUN_RECIPIENTS A, HRC_ALERT_RUN_MESSAGES B, HRC_ALERTS_TL C, HRC_ALERT_TEMPLATES_VL D WHERE A.RUN_MESSAGE_ID = B.RUN_MESSAGE_ID AND B.TEMPLATE_ID = D.TEMPLATE_ID AND D.ALERT_ID = C.ALERT_ID
```
