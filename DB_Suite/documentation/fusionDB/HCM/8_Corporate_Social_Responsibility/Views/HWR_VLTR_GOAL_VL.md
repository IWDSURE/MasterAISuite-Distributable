# HWR_VLTR_GOAL_VL

## Details

**Schema:** FUSION

**Object owner:** HHR

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrvltrgoalvl-7298.html#hwrvltrgoalvl-7298](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwrvltrgoalvl-7298.html#hwrvltrgoalvl-7298)

## Columns

- GOAL_ID
- LOWER_BOUND
- UPPER_BOUND
- EFFECTIVE_START_DATE
- EFFECTIVE_END_DATE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- GOAL_METRIC
- UNITS
- FREQUENCY
- HCM_PERSON_ID

## Query

```sql
SELECT A.GOAL_ID GOAL_ID, A.LOWER_BOUND LOWER_BOUND, A.UPPER_BOUND UPPER_BOUND, A.EFFECTIVE_START_DATE EFFECTIVE_START_DATE, A.EFFECTIVE_END_DATE EFFECTIVE_END_DATE, A.CREATED_BY CREATED_BY, A.CREATION_DATE CREATION_DATE, A.LAST_UPDATED_BY LAST_UPDATED_BY, A.LAST_UPDATE_DATE LAST_UPDATE_DATE, A.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.GOAL_METRIC GOAL_METRIC, B.UNITS UNITS, B.FREQUENCY FREQUENCY , A.HCM_PERSON_ID HCM_PERSON_ID FROM HWR_VLTR_GOAL_B A, HWR_VLTR_GOAL_TL B WHERE A.GOAL_ID = B.GOAL_ID AND B.LANGUAGE = USERENV('LANG')
```
