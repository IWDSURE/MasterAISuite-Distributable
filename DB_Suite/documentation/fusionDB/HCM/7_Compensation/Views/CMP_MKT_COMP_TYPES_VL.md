# CMP_MKT_COMP_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktcomptypesvl-7837.html#cmpmktcomptypesvl-7837](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktcomptypesvl-7837.html#cmpmktcomptypesvl-7837)

## Columns

- COMP_TYPE_ID
- COMP_TYPE_NAME
- COMP_TYPE_SEQ_NUM
- COMP_TYPE_CODE
- BUSINESS_GROUP_ID
- MARKET_TARGET
- GROUP_CODE
- DATA_TYPE_CODE
- STATUS_CODE
- SEED_DATA
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.COMP_TYPE_ID, T.COMP_TYPE_NAME, B.COMP_TYPE_SEQ_NUM, B.COMP_TYPE_CODE, B.BUSINESS_GROUP_ID, B.MARKET_TARGET, B.GROUP_CODE, B.DATA_TYPE_CODE, B.STATUS_CODE, B.SEED_DATA, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM CMP_MKT_COMP_TYPES_B B, CMP_MKT_COMP_TYPES_TL T WHERE B.COMP_TYPE_ID = T.COMP_TYPE_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
