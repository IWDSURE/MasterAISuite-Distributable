## Query

```sql
SELECT B.LOOKUP_USAGE_ID, B.KEY_TYPE, B.KEY_VALUE, B.ATTRIBUTE_NAME, B.ATTRIBUTE_VALUE, B.ENABLED_FLAG, B.ORDER_NUM, B.NUMBER_VALUE1, B.NUMBER_VALUE2, B.NUMBER_VALUE3, B.TEXT_VALUE1, B.TEXT_VALUE2, T.DISPLAY_NAME, T.DESCRIPTION, B.TEXT_VALUE3, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.BUSINESS_GROUP_ID, B.NUMBER_VALUE4, B.TEXT_VALUE4 FROM CMP_LOOKUP_USAGES_B B, CMP_LOOKUP_USAGES_TL T WHERE B.LOOKUP_USAGE_ID = T.LOOKUP_USAGE_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
