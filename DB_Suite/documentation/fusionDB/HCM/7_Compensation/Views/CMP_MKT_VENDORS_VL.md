# CMP_MKT_VENDORS_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvendorsvl-5207.html#cmpmktvendorsvl-5207](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvendorsvl-5207.html#cmpmktvendorsvl-5207)

## Columns

- VENDOR_ID
- VENDOR_NAME
- VENDOR_SEQ_NUM
- VENDOR_CODE
- BUSINESS_GROUP_ID
- STATUS
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- CONTACT_NAME_1
- CONTACT_EMAIL_1
- CONTACT_PHONE_1
- CONTACT_NAME_2
- CONTACT_EMAIL_2
- CONTACT_PHONE_2
- WEBSITE
- VENDOR_SPECIALTY
- NOTES

## Query

```sql
SELECT B.VENDOR_ID, T.VENDOR_NAME, B.VENDOR_SEQ_NUM, B.VENDOR_CODE, B.BUSINESS_GROUP_ID, B.STATUS, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.CONTACT_NAME_1, B.CONTACT_EMAIL_1, B.CONTACT_PHONE_1, B.CONTACT_NAME_2, B.CONTACT_EMAIL_2, B.CONTACT_PHONE_2, B.WEBSITE, T.VENDOR_SPECIALTY, T.NOTES FROM CMP_MKT_VENDORS_B B, CMP_MKT_VENDORS_TL T WHERE B.VENDOR_ID = T.VENDOR_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
