# CMP_MKT_COMP_TYPE_CONFIG_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktcomptypeconfigvl-7799.html#cmpmktcomptypeconfigvl-7799](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktcomptypeconfigvl-7799.html#cmpmktcomptypeconfigvl-7799)

## Columns

- COMP_TYPE_CONFIG_ID
- COMPENSATION_OBJECT
- COMP_TYPE_ID
- DISPLAY_SEQUENCE
- COMP_TYPE_DISPLAY_NAME
- MKT_TARGET_DISPLAY_NAME
- ENABLE_COMP_TYPE_FLAG
- ENABLE_MKT_TARGET_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.COMP_TYPE_CONFIG_ID, B.COMPENSATION_OBJECT, B.COMP_TYPE_ID, B.DISPLAY_SEQUENCE, TL.COMP_TYPE_DISPLAY_NAME, TL.MKT_TARGET_DISPLAY_NAME, B.ENABLE_COMP_TYPE_FLAG, B.ENABLE_MKT_TARGET_FLAG, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM CMP_MKT_COMP_TYPE_CONFIG_B B, CMP_MKT_COMP_TYPE_CONFIG_TL TL WHERE B.COMP_TYPE_CONFIG_ID = TL.COMP_TYPE_CONFIG_ID AND TL.LANGUAGE = USERENV('LANG')
```
