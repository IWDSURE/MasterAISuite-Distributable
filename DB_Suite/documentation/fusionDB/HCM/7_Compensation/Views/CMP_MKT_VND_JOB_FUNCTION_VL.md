# CMP_MKT_VND_JOB_FUNCTION_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvndjobfunctionvl-7443.html#cmpmktvndjobfunctionvl-7443](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvndjobfunctionvl-7443.html#cmpmktvndjobfunctionvl-7443)

## Columns

- JOB_FUNCTION_ID
- JOB_FUNCTION_SEQ_NUM
- JOB_FUNCTION_CODE
- VENDOR_ID
- STATUS
- JOB_FUNCTION_NAME
- JOB_FUNCTION_DESCR
- DESCRIPTION
- BUSINESS_GROUP_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.JOB_FUNCTION_ID, B.JOB_FUNCTION_SEQ_NUM, B.JOB_FUNCTION_CODE, B.VENDOR_ID, B.STATUS, T.JOB_FUNCTION_NAME, T.JOB_FUNCTION_DESCR, T.DESCRIPTION, B.BUSINESS_GROUP_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM CMP_MKT_VND_JOB_FUNCTION_B B, CMP_MKT_VND_JOB_FUNCTION_TL T where B.JOB_FUNCTION_ID = T.JOB_FUNCTION_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
