## Query

```sql
SELECT B.JOB_LIST_ID, T.VENDOR_JOB_TITLE, T.VENDOR_JOB_DESCR, T.VENDOR_JOB_DESCR_CHAR, T.DESCRIPTION, B.VENDOR_ID, B.VENDOR_JOB_CODE, B.JOB_FUNCTION_ID, B.JOB_FAMILY_ID, B.CAREER_STREAM_ID, B.CAREER_LEVEL_ID, B.OTHER_LEVEL_ID, B.VENDOR_JOB_MAPPING_CODE, B.JOB_ID, B.POSITION_ID, B.STATUS, T.MISC_TEXT_1, T.MISC_TEXT_2, T.MISC_TEXT_3, T.NOTES, B.BUSINESS_GROUP_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.JOB_MATCH_PERCENT FROM CMP_MKT_VND_JOBS_B B, CMP_MKT_VND_JOBS_TL T WHERE B.JOB_LIST_ID = T.JOB_LIST_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
