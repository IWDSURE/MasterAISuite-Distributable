# CMP_TCS_ITEM_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmptcsitemvl-4460.html#cmptcsitemvl-4460](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmptcsitemvl-4460.html#cmptcsitemvl-4460)

## Columns

- ITEM_ID
- NAME
- DATA_TYPE_CD
- ESTIMATED_FLAG
- LEGISLATIVE_DATA_GROUP_ID
- NNMNTRY_UOM
- ROUNDING_CD
- SOURCE_CD
- SOURCE_KEY
- UOM
- BUSINESS_GROUP_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- SOURCE_KEY_CHAR1
- SOURCE_KEY_CHAR2
- SELECTION_CRITERIA
- STAT_CD
- PRORATION_FLAG
- CALC_EXPR

## Query

```sql
SELECT ITEM.ITEM_ID, ITEM_TL.NAME, ITEM.DATA_TYPE_CD, ITEM.ESTIMATED_FLAG, ITEM.LEGISLATIVE_DATA_GROUP_ID, ITEM.NNMNTRY_UOM, ITEM.ROUNDING_CD, ITEM.SOURCE_CD, ITEM.SOURCE_KEY, ITEM.UOM, ITEM.BUSINESS_GROUP_ID, ITEM.CREATED_BY, ITEM.CREATION_DATE, ITEM.LAST_UPDATED_BY, ITEM.LAST_UPDATE_DATE, ITEM.LAST_UPDATE_LOGIN, ITEM.OBJECT_VERSION_NUMBER, ITEM.SOURCE_KEY_CHAR1, ITEM.SOURCE_KEY_CHAR2, ITEM.SELECTION_CRITERIA, ITEM.STAT_CD , ITEM.PRORATION_FLAG , ITEM.CALC_EXPR FROM CMP_TCS_ITEM ITEM, CMP_TCS_ITEM_TL ITEM_TL WHERE ITEM.ITEM_ID = ITEM_TL.ITEM_ID AND ITEM_TL.LANGUAGE = USERENV('LANG') AND ITEM.BUSINESS_GROUP_ID = ITEM_TL.BUSINESS_GROUP_ID
```
