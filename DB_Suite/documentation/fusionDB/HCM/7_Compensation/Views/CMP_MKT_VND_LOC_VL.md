# CMP_MKT_VND_LOC_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvndlocvl-4870.html#cmpmktvndlocvl-4870](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktvndlocvl-4870.html#cmpmktvndlocvl-4870)

## Columns

- SURVEY_LOCATION_ID
- VENDOR_ID
- SURVEY_LOCATION_CODE
- STATUS
- SURVEY_LOCATION_NAME
- MISC_TEXT_1
- MISC_TEXT_2
- MISC_TEXT_3
- DESCRIPTION
- BUSINESS_GROUP_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.SURVEY_LOCATION_ID, B.VENDOR_ID, B.SURVEY_LOCATION_CODE, B.STATUS, T.SURVEY_LOCATION_NAME, T.MISC_TEXT_1, T.MISC_TEXT_2, T.MISC_TEXT_3, T.DESCRIPTION, B.BUSINESS_GROUP_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM CMP_MKT_VND_LOC_B B, CMP_MKT_VND_LOC_TL T WHERE B.SURVEY_LOCATION_ID = T.SURVEY_LOCATION_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
