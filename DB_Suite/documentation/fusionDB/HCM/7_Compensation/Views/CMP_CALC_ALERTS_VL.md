# CMP_CALC_ALERTS_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpcalcalertsvl-5057.html#cmpcalcalertsvl-5057](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpcalcalertsvl-5057.html#cmpcalcalertsvl-5057)

## Columns

- ALERT_ID
- ALERT_KEY
- KEY_TYPE
- ALERT_TYPE
- ENABLED_FLAG
- ALERT_NAME
- ALERT_DESCRIPTION
- QUICK_ALERT_CODE
- QUICK_ALERT_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- PARTI_PROCESS_ENA
- REFRESH_PROCESS_ENA
- WRKSHT_SAVE_ENA
- WRKSHT_TAB_OUT_ENA
- BUSINESS_GROUP_ID

## Query

```sql
SELECT B.ALERT_ID, B.ALERT_KEY, B.KEY_TYPE, B.ALERT_TYPE, B.ENABLED_FLAG, T.ALERT_NAME, T.ALERT_DESCRIPTION, B.QUICK_ALERT_CODE, B.QUICK_ALERT_FLAG, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.PARTI_PROCESS_ENA, B.REFRESH_PROCESS_ENA, B.WRKSHT_SAVE_ENA, B.WRKSHT_TAB_OUT_ENA, B.BUSINESS_GROUP_ID FROM CMP_CALC_ALERTS_B B, CMP_CALC_ALERTS_TL T WHERE B.ALERT_ID = T.ALERT_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
