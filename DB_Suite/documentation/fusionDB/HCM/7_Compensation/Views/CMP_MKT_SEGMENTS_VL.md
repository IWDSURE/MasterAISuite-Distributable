# CMP_MKT_SEGMENTS_VL

## Details

**Schema:** FUSION

**Object owner:** CMP

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktsegmentsvl-5761.html#cmpmktsegmentsvl-5761](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/cmpmktsegmentsvl-5761.html#cmpmktsegmentsvl-5761)

## Columns

- SEGMENT_ID
- SEGMENT_CODE
- SEGMENT_NAME
- SEGMENT_DESCR
- MISC_TEXT_1
- MISC_TEXT_2
- MISC_TEXT_3
- COUNTRY_CODE
- STATUS_CODE
- BUSINESS_GROUP_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.SEGMENT_ID SEGMENT_ID, B.SEGMENT_CODE SEGMENT_CODE, T.SEGMENT_NAME SEGMENT_NAME, T.SEGMENT_DESCR SEGMENT_DESCR, T.MISC_TEXT_1, T.MISC_TEXT_2, T.MISC_TEXT_3, B.COUNTRY_CODE COUNTRY_CODE, B.STATUS_CODE STATUS_CODE, B.BUSINESS_GROUP_ID BUSINESS_GROUP_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM CMP_MKT_SEGMENTS_B B, CMP_MKT_SEGMENTS_TL T WHERE B.SEGMENT_ID = T.SEGMENT_ID AND T.LANGUAGE = USERENV('LANG') AND B.BUSINESS_GROUP_ID = T.BUSINESS_GROUP_ID
```
