# FAI_SUPPT_OBJECT_FIELDS_B

This table stores metadata of fields in supported business objects used by an agent tool.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectfieldsb-15110.html#faisupptobjectfieldsb-15110](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectfieldsb-15110.html#faisupptobjectfieldsb-15110)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_SUPPT_OBJECT_FIELDS_B_PK | FIELD_ID |
