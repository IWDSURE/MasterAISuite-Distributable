# FAI_DEEP_LINK_TOOLS_B

This table stores an association of a deep link to a generative AI tool.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinktoolsb-24248.html#faideeplinktoolsb-24248](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinktoolsb-24248.html#faideeplinktoolsb-24248)
