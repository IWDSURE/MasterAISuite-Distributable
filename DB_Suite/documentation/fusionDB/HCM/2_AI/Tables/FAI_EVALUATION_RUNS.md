# FAI_EVALUATION_RUNS

This table stores information about evaluation runs.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faievaluationruns-13770.html#faievaluationruns-13770](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faievaluationruns-13770.html#faievaluationruns-13770)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_EVALUATION_RUNS_PK | EVAL_RUN_ID |
