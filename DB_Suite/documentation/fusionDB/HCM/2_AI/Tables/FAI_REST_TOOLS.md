# FAI_REST_TOOLS

This table stores information needed by an agent tool to make a REST call.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/fairesttools-29124.html#fairesttools-29124](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/fairesttools-29124.html#fairesttools-29124)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_REST_TOOLS_PK | REST_TOOL_ID |
