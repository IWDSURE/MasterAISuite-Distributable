# FAI_DEEP_LINKS_B_

This table stores deep links that can be used by a generative AI tool.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinksb-30050.html#faideeplinksb-30050](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinksb-30050.html#faideeplinksb-30050)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_DEEP_LINKS_B_PK_ | LAST_UPDATE_DATE, LAST_UPDATED_BY, DEEP_LINK_ID |
