# FAI_MSG_DLVY_OPTNS_B

This table stores message delivery options for a generative AI tool.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faimsgdlvyoptnsb-29229.html#faimsgdlvyoptnsb-29229](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faimsgdlvyoptnsb-29229.html#faimsgdlvyoptnsb-29229)
