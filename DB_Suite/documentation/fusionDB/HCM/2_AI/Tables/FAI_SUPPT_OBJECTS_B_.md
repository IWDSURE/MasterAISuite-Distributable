# FAI_SUPPT_OBJECTS_B_

This table stores information required by an agent tool to make a REST call on supported business objects.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectsb-16364.html#faisupptobjectsb-16364](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectsb-16364.html#faisupptobjectsb-16364)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_SUPPT_OBJECTS_B_PK_ | LAST_UPDATE_DATE, LAST_UPDATED_BY, SUPPT_OBJECT_ID |
