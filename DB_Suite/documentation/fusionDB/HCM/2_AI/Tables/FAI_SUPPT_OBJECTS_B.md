# FAI_SUPPT_OBJECTS_B

This table stores information required by an agent tool to make a REST call on supported business objects.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectsb-7955.html#faisupptobjectsb-7955](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faisupptobjectsb-7955.html#faisupptobjectsb-7955)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_SUPPT_OBJECTS_B_PK | SUPPT_OBJECT_ID |
