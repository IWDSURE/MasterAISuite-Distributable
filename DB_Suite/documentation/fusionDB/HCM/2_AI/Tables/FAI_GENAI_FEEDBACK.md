# FAI_GENAI_FEEDBACK

This table stores feedback submitted for a generative AI response.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faigenaifeedback-20006.html#faigenaifeedback-20006](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faigenaifeedback-20006.html#faigenaifeedback-20006)

## Primary Key

| Name | Columns |
|------|----------|
| FAI_GENAI_FEEDBACK_PK | FEEDBACK_ID |

## Columns

| Name | Datatype | Length | Precision | Not-null | Comments |
|---|---|---|---|---|---|
| FEEDBACK_ID | NUMBER |  | 18 | Yes | Autogenerated number and primary key for this table. |
| ORCHESTRATION_FLOW_ID | VARCHAR2 | 200 |  |  | Foreign key referencing the ORCHESTRATION_FLOW_ID column in the FAI_GENAI_REQUESTS table. |
| RESPONSE_INDEX | NUMBER |  | 2 |  | The index of the response for which feedback is provided. |
| ACTUAL_TEXT | CLOB |  |  |  | The text submitted by the end user. |
| EXECUTION_GROUP_NUMBER | VARCHAR2 | 200 |  |  | This column indicates the reference to an agents execution turn |
| TRACE | VARCHAR2 | 200 |  |  | This column indicates the refernece to an agents conversation trace |
| EVENT_TYPE | VARCHAR2 | 80 |  | Yes | This column indicates the event type collected as feedback record |
| EVENT_PAYLOAD | CLOB |  |  |  | This column captures full event payload |

| CREATION_DATE | TIMESTAMP |  |  | Yes | Who column: indicates the date and time of the creation of the row. |

| LAST_UPDATE_DATE | TIMESTAMP |  |  | Yes | Who column: indicates the date and time of the last update of the row. |
| LAST_UPDATE_LOGIN | VARCHAR2 | 32 |  |  | Who column: indicates the session login associated to the user who last updated the row. |
| OBJECT_VERSION_NUMBER | NUMBER |  | 9 | Yes | Used to implement optimistic locking. This number is incremented every time that the row is updated. The number is compared at the start and end of a transaction to detect whether another session has updated the row since it was queried. |
