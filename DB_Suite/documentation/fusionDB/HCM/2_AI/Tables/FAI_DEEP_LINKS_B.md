# FAI_DEEP_LINKS_B

This table stores deep links that can be used by a generative AI tool.

## Details

**Schema:** FUSION

**Object owner:** FAI

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinksb-30563.html#faideeplinksb-30563](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/faideeplinksb-30563.html#faideeplinksb-30563)

| Name | Columns |

| FAI_DEEP_LINKS_B_PK | DEEP_LINK_ID |
