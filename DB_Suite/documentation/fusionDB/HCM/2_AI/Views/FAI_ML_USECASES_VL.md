**Schema:** FUSION

## Query

```sql
SELECT B.ID ID, B.USECASE_CODE USECASE_CODE, B.FAMILY FAMILY, B.INTRODUCED_IN_RELEASE INTRODUCED_IN_RELEASE, B.ADDN_INFO ADDN_INFO, B.ADDN_TXN_INFO ADDN_TXN_INFO, B.STATUS STATUS, B.STATUS_DETAILS STATUS_DETAILS, B.ML_APP_NAME ML_APP_NAME, B.ML_APP_IMPL_NAME ML_APP_IMPL_NAME, B.ML_APP_VERSION ML_APP_VERSION, B.ML_APP_INSTANCE_ID ML_APP_INSTANCE_ID, B.LAST_ACTIVATION_DATE LAST_ACTIVATION_DATE, B.LAST_DEACTIVATION_DATE LAST_DEACTIVATION_DATE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.MODULE_ID MODULE_ID, T.USECASE_NAME USECASE_NAME FROM FAI_ML_USECASES_B B, FAI_ML_USECASES_TL T WHERE B.ID=T.ID AND T.LANGUAGE=USERENV('LANG')
```
