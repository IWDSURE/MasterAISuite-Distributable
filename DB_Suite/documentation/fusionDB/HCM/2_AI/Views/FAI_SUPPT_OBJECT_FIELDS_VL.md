## Query

```sql
SELECT b.FIELD_ID FIELD_ID, b.SUPPT_OBJECT_ID SUPPT_OBJECT_ID, b.FIELD_NAME FIELD_NAME, b.DATA_TYPE DATA_TYPE, b.QUERYABLE_FLAG QUERYABLE_FLAG, b.UPDATABLE_FLAG UPDATABLE_FLAG, b.REQUIRED_FOR_CREATE_FLAG REQUIRED_FOR_CREATE_FLAG, b.FIELD_PROPERTIES FIELD_PROPERTIES, b.MANDATORY_FLAG MANDATORY_FLAG, b.SEEDED_FLAG SEEDED_FLAG, b.CREATED_BY CREATED_BY, b.CREATION_DATE CREATION_DATE, b.LAST_UPDATED_BY LAST_UPDATED_BY, b.LAST_UPDATE_DATE LAST_UPDATE_DATE, b.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, b.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, tl.FIELD_DISPLAY_NAME FIELD_DISPLAY_NAME, tl.DESCRIPTION DESCRIPTION FROM FAI_SUPPT_OBJECT_FIELDS_B b, FAI_SUPPT_OBJECT_FIELDS_TL tl WHERE b.FIELD_ID = tl.FIELD_ID AND tl.LANGUAGE = USERENV('LANG')
```
