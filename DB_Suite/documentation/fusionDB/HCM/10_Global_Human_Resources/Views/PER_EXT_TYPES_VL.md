# PER_EXT_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perexttypesvl-5794.html#perexttypesvl-5794](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perexttypesvl-5794.html#perexttypesvl-5794)

## Columns

- EXT_TYPE_ID
- EXT_TYPE_CODE
- TYPE_NAME
- DESCRIPTION
- RUN_ELEVATED
- RUN_ELEVATED_OVERRIDE
- ALLOW_USER_CREATION
- MODULE_ID
- LEGISLATION_CODE
- LEGISLATIVE_DATA_GROUP_ID
- ENTERPRISE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- EXT_TYPE_GRP_CODE

## Query

```sql
SELECT PER_EXT_TYPES_B.EXT_TYPE_ID EXT_TYPE_ID, PER_EXT_TYPES_B.EXT_TYPE_CODE EXT_TYPE_CODE, PER_EXT_TYPES_TL.TYPE_NAME TYPE_NAME, PER_EXT_TYPES_TL.DESCRIPTION DESCRIPTION, PER_EXT_TYPES_B.RUN_ELEVATED, PER_EXT_TYPES_B.RUN_ELEVATED_OVERRIDE, PER_EXT_TYPES_B.ALLOW_USER_CREATION, PER_EXT_TYPES_B.MODULE_ID MODULE_ID, PER_EXT_TYPES_B.LEGISLATION_CODE LEGISLATION_CODE, PER_EXT_TYPES_B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, PER_EXT_TYPES_B.ENTERPRISE_ID ENTERPRISE_ID, PER_EXT_TYPES_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, PER_EXT_TYPES_B.CREATED_BY CREATED_BY, PER_EXT_TYPES_B.CREATION_DATE CREATION_DATE, PER_EXT_TYPES_B.LAST_UPDATED_BY LAST_UPDATED_BY, PER_EXT_TYPES_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, PER_EXT_TYPES_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, PER_EXT_TYPES_B.EXT_TYPE_GRP_CODE EXT_TYPE_GRP_CODE FROM PER_EXT_TYPES_B, PER_EXT_TYPES_TL WHERE PER_EXT_TYPES_B.EXT_TYPE_ID = PER_EXT_TYPES_TL.EXT_TYPE_ID AND PER_EXT_TYPES_TL.LANGUAGE = USERENV('LANG')
```
