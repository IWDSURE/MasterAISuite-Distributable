# PER_PORTRAIT_MESSAGES_VL

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perportraitmessagesvl-7149.html#perportraitmessagesvl-7149](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perportraitmessagesvl-7149.html#perportraitmessagesvl-7149)

## Columns

- PORTRAIT_MESSAGE_ID
- PORTRAIT_MESSAGE_CONTENT
- SHOW_CAL_EVT_CATEGORY
- PRIORITY
- MESSAGE_TYPE
- TIME_FROM
- TIME_TO
- TARGET_PERSON_ID
- TARGET_DEPARTMENT_ID
- TARGET_LOCATION_ID
- TARGET_ORGANIZATION_TREE_CODE
- TARGET_ORGANIZATION_TOP_NODE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- ENTERPRISE_ID

## Query

```sql
SELECT B.PORTRAIT_MESSAGE_ID, T.PORTRAIT_MESSAGE_CONTENT, B.SHOW_CAL_EVT_CATEGORY, B.PRIORITY, B.MESSAGE_TYPE, B.TIME_FROM, B.TIME_TO, B.TARGET_PERSON_ID, B.TARGET_DEPARTMENT_ID, B.TARGET_LOCATION_ID, B.TARGET_ORGANIZATION_TREE_CODE, B.TARGET_ORGANIZATION_TOP_NODE, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.ENTERPRISE_ID FROM PER_PORTRAIT_MESSAGES_B B, PER_PORTRAIT_MESSAGES_TL T WHERE B.PORTRAIT_MESSAGE_ID = T.PORTRAIT_MESSAGE_ID AND T.LANGUAGE = USERENV('LANG')
```
