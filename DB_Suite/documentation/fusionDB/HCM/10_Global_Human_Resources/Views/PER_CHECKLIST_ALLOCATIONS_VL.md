# PER_CHECKLIST_ALLOCATIONS_VL

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perchecklistallocationsvl-5404.html#perchecklistallocationsvl-5404](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perchecklistallocationsvl-5404.html#perchecklistallocationsvl-5404)

## Columns

- CHECKLIST_ALLOCATION_ID
- NAME
- DESCRIPTION
- CHECKLIST_ALLOCATION_CODE
- CHECKLIST_ID
- INITIATOR_PERSON_ID
- COMMENTS
- START_DATE
- ALLOCATION_TYPE
- CHECKLIST_SCHEDULE_ID
- STATUS
- ENTERPRISE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.CHECKLIST_ALLOCATION_ID, T.NAME, T.DESCRIPTION, B.CHECKLIST_ALLOCATION_CODE, B.CHECKLIST_ID, B.INITIATOR_PERSON_ID, B.COMMENTS, B.START_DATE, B.ALLOCATION_TYPE, B.CHECKLIST_SCHEDULE_ID, B.STATUS, B.ENTERPRISE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM PER_CHECKLIST_ALLOCATIONS B, PER_CHECKLIST_ALLOCATIONS_TL T WHERE B.CHECKLIST_ALLOCATION_ID = T.CHECKLIST_ALLOCATION_ID AND T.LANGUAGE = USERENV('LANG')
```
