# HR_GEN_AI_DOCUMENTS_VL

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrgenaidocumentsvl-4128.html#hrgenaidocumentsvl-4128](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrgenaidocumentsvl-4128.html#hrgenaidocumentsvl-4128)

## Columns

- DOCUMENT_ID
- TOOL_ID
- STATUS
- DOCUMENT_CREATED_DATE
- DOCUMENT_PUBLISHED_DATE
- DOCUMENT_DELETED_DATE
- OWNER_ID
- OWNER_NAME
- INTERNAL_NAME
- INTERNAL_DESCRIPTION
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- NAME
- DESCRIPTION

## Query

```sql
SELECT B.DOCUMENT_ID, B.TOOL_ID, B.STATUS, B.DOCUMENT_CREATED_DATE, B.DOCUMENT_PUBLISHED_DATE, B.DOCUMENT_DELETED_DATE, B.OWNER_ID, B.OWNER_NAME, B.INTERNAL_NAME, B.INTERNAL_DESCRIPTION, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.OBJECT_VERSION_NUMBER, T.NAME, T.DESCRIPTION FROM HR_GEN_AI_DOCUMENTS_B B, HR_GEN_AI_DOCUMENTS_TL T WHERE B.DOCUMENT_ID=T.DOCUMENT_ID AND T.LANGUAGE=USERENV('LANG')
```
