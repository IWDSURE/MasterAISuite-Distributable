## Query

```sql
SELECT 'ORA_PERSON' as PERFORMER_CATEGORY from DUAL UNION SELECT 'ORA_USER' as PERFORMER_CATEGORY from DUAL UNION SELECT 'ORA_SUPPLIER' as PERFORMER_CATEGORY from DUAL
```
