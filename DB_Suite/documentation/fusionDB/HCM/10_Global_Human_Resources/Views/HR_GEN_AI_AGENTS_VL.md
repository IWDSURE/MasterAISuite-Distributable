## Query

```sql
SELECT B.AGENT_ID, B.AGENT_CODE, B.FAMILY, B.PRODUCT, B.AGENT_CREATED_DATE, B.OWNER_ID, B.OWNER_NAME, B.STATUS, B.PROMPT_CODE, B.NAMESPACE, B.VERSION, B.SPECIFICATION, B.MAX_INTERACTIONS, B.AGENT_TYPE, B.PROMPT, B.MODEL_CONFIG_ID, B.INTERNAL_NAME, B.INTERNAL_DESCRIPTION, B.REUSABLE_FLAG, B.SEEDED_FLAG, B.MODULE_ID, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.OBJECT_VERSION_NUMBER, T.NAME, T.DESCRIPTION FROM HR_GEN_AI_AGENTS_B B, HR_GEN_AI_AGENTS_TL T WHERE B.AGENT_ID=T.AGENT_ID AND T.LANGUAGE=USERENV('LANG')
```
