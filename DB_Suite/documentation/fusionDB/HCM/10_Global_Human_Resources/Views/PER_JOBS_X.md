# PER_JOBS_X

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perjobsx-5673.html#perjobsx-5673](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perjobsx-5673.html#perjobsx-5673)

## Columns

- JOB_ID
- EFFECTIVE_START_DATE
- EFFECTIVE_END_DATE
- JOB_CODE
- NAME
- BUSINESS_GROUP_ID
- SET_ID
- JOB_FAMILY_ID
- BENCHMARK_JOB_ID
- ACTION_OCCURRENCE_ID
- ACTIVE_STATUS
- BENCHMARK_JOB_FLAG
- APPROVAL_AUTHORITY
- MANAGER_LEVEL
- MED_CHECKUP_REQ
- REGULAR_TEMPORARY
- FULL_PART_TIME
- JOB_FUNCTION_CODE

## Query

```sql
SELECT JOBS.JOB_ID, JOBS.EFFECTIVE_START_DATE, JOBS.EFFECTIVE_END_DATE, JOBS.JOB_CODE, JOBS_TL.NAME, JOBS.BUSINESS_GROUP_ID, JOBS.SET_ID, JOBS.JOB_FAMILY_ID, JOBS.BENCHMARK_JOB_ID, JOBS.ACTION_OCCURRENCE_ID, JOBS.ACTIVE_STATUS, JOBS.BENCHMARK_JOB_FLAG, JOBS.APPROVAL_AUTHORITY, JOBS.MANAGER_LEVEL, JOBS.MED_CHECKUP_REQ, JOBS.REGULAR_TEMPORARY, JOBS.FULL_PART_TIME, JOBS.JOB_FUNCTION_CODE FROM PER_JOBS_F JOBS, PER_JOBS_F_TL JOBS_TL WHERE JOBS.JOB_ID = JOBS_TL.JOB_ID AND JOBS.EFFECTIVE_START_DATE = JOBS_TL.EFFECTIVE_START_DATE AND JOBS.EFFECTIVE_END_DATE = JOBS_TL.EFFECTIVE_END_DATE AND JOBS_TL.LANGUAGE = SYS_CONTEXT('USERENV', 'LANG') AND TRUNC(SYSDATE) BETWEEN JOBS.EFFECTIVE_START_DATE AND JOBS.EFFECTIVE_END_DATE
```
