# PER_CHECKLIST_CATEGORIES_VL

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perchecklistcategoriesvl-4165.html#perchecklistcategoriesvl-4165](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perchecklistcategoriesvl-4165.html#perchecklistcategoriesvl-4165)

## Columns

- CHECKLIST_CATEGORY_ID
- CATEGORY_CODE
- CATEGORY_TYPE
- SUB_CATEGORY_ALLOWED
- SUBJECT_TYPE_CODE
- PARENT_CATEGORY_CODE
- ENABLED_FLAG
- TAG
- NAME
- DESCRIPTION
- SEED_DATA_SOURCE
- MODULE_ID
- ENTERPRISE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.CHECKLIST_CATEGORY_ID, B.CATEGORY_CODE, B.CATEGORY_TYPE, B.SUB_CATEGORY_ALLOWED, B.SUBJECT_TYPE_CODE, B.PARENT_CATEGORY_CODE, B.ENABLED_FLAG, B.TAG, T.NAME, T.DESCRIPTION, B.SEED_DATA_SOURCE, B.MODULE_ID, B.ENTERPRISE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM PER_CHECKLIST_CATEGORIES_B B, PER_CHECKLIST_CATEGORIES_TL T WHERE B.CHECKLIST_CATEGORY_ID = T.CHECKLIST_CATEGORY_ID AND T.LANGUAGE = USERENV('LANG')
```
