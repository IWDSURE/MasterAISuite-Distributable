# PER_PERSONS_

This table is new for Fusion, and is created to assist BC4J architecture by providing a non-date tracked parent table to act as the parent for all tables in the Person Model, even PER_ALL_PEOPLE_F itself

## Details

**Schema:** FUSION

**Object owner:** PER

**Object type:** TABLE

**Tablespace:** APPS_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perpersons-14892.html#perpersons-14892](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/perpersons-14892.html#perpersons-14892)

## Primary Key

| Name | Columns |
|------|----------|
| PER_PERSONS_PK_ | LAST_UPDATE_DATE, LAST_UPDATED_BY, PERSON_ID |
