# HRC_MESSAGE_CONFIG_VL

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcmessageconfigvl-7078.html#hrcmessageconfigvl-7078](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcmessageconfigvl-7078.html#hrcmessageconfigvl-7078)

- DISABLE_SMS_CONFIG_EDIT
- DISABLE_EMAIL_CONFIG_EDIT
- DISABLE_PUSH_CONFIG_EDIT
- MAKE_BIP_MANDATE

- SUBJECT

## Query

```sql
SELECT HMCB.MESSAGE_CONFIG_ID MESSAGE_CONFIG_ID, HMCB.CONFIG_ID CONFIG_ID, HMCB.DISABLE_CONFIG_EDIT DISABLE_CONFIG_EDIT, HMCB.DISABLE_SMS_CONFIG_EDIT DISABLE_SMS_CONFIG_EDIT, HMCB.DISABLE_EMAIL_CONFIG_EDIT DISABLE_EMAIL_CONFIG_EDIT, HMCB.DISABLE_PUSH_CONFIG_EDIT DISABLE_PUSH_CONFIG_EDIT, HMCB.MAKE_BIP_MANDATE MAKE_BIP_MANDATE, HMCB.ENTERPRISE_ID ENTERPRISE_ID, HMCB.MODULE_ID MODULE_ID, HMCB.CREATED_BY CREATED_BY, HMCB.CREATION_DATE CREATION_DATE, HMCB.LAST_UPDATED_BY LAST_UPDATED_BY, HMCB.LAST_UPDATE_DATE LAST_UPDATE_DATE, HMCB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, HMCB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, HMCTL.SUBJECT SUBJECT, HMCTL.CONTENT CONTENT, HMCTL.FROM_ADDRESS_DISPLAY_NAME FROM_ADDRESS_DISPLAY_NAME FROM HRC_MESSAGE_CONFIG_B HMCB, HRC_MESSAGE_CONFIG_TL HMCTL WHERE HMCB.MESSAGE_CONFIG_ID = HMCTL.MESSAGE_CONFIG_ID AND HMCTL.LANGUAGE = USERENV('LANG')
```
