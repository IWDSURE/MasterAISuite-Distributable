## Query

```sql
SELECT B.BUS_OBJECT_ATTR_ID BUS_OBJECT_ATTR_ID, TL.ATTRIBUTE_LABEL ATTRIBUTE_LABEL, TL.ATTRIBUTE_DESCRIPTION ATTRIBUTE_DESCRIPTION, B.VO_ATTRIBUTE_NAME VO_ATTRIBUTE_NAME, B.COMMON_KEY COMMON_KEY, B.BUSINESS_OBJECT_ID BUSINESS_OBJECT_ID, B.MODULE_ID MODULE_ID, B.ENTERPRISE_ID ENTERPRISE_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.SGUID SGUID FROM HRC_DL_BUS_OBJECT_ATTRS_B B, HRC_DL_BUS_OBJECT_ATTRS_TL TL WHERE B.BUS_OBJECT_ATTR_ID = TL.BUS_OBJECT_ATTR_ID AND USERENV('LANG') = TL.LANGUAGE
```
