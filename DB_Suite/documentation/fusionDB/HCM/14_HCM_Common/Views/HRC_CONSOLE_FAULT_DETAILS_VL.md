# HRC_CONSOLE_FAULT_DETAILS_VL

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcconsolefaultdetailsvl-4257.html#hrcconsolefaultdetailsvl-4257](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcconsolefaultdetailsvl-4257.html#hrcconsolefaultdetailsvl-4257)

## Columns

- FAULT_DETAILS_ID
- ISSUE_TYPE_CODE
- ISSUE_TYPE
- ISSUE_TITLE
- DESCRIPTION
- RESOLUTION
- ENTERPRISE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- FAULT_CODE
- ERROR_CODE
- INTERNAL_CODE
- SGUID
- MODULE_ID

## Query

```sql
SELECT HCFD.FAULT_DETAILS_ID, HCFD.ISSUE_TYPE_CODE, HCFDTL.ISSUE_TYPE, HCFDTL.ISSUE_TITLE, HCFDTL.DESCRIPTION, HCFDTL.RESOLUTION, HCFD.ENTERPRISE_ID, HCFD.CREATED_BY, HCFD.CREATION_DATE, HCFD.LAST_UPDATED_BY, HCFD.LAST_UPDATE_DATE, HCFD.LAST_UPDATE_LOGIN, HCFD.OBJECT_VERSION_NUMBER, HCFD.FAULT_CODE, HCFD.ERROR_CODE, HCFD.INTERNAL_CODE, HCFD.SGUID, HCFD.MODULE_ID FROM HRC_CONSOLE_FAULT_DETAILS_B HCFD, HRC_CONSOLE_FAULT_DETAILS_TL HCFDTL WHERE HCFD.FAULT_DETAILS_ID = HCFDTL.FAULT_DETAILS_ID AND HCFDTL.LANGUAGE = USERENV('LANG')
```
