# HRC_ARM_COMPOSITES_VL

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcarmcompositesvl-6107.html#hrcarmcompositesvl-6107](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcarmcompositesvl-6107.html#hrcarmcompositesvl-6107)

## Columns

- COMPOSITE_ID
- COMPOSITE_NAME
- ENTERPRISE_ID
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- NAME
- DESCRIPTION

## Query

```sql
SELECT HAC.COMPOSITE_ID COMPOSITE_ID, HAC.COMPOSITE_NAME COMPOSITE_NAME, HAC.ENTERPRISE_ID ENTERPRISE_ID, HAC.MODULE_ID MODULE_ID, HAC.CREATED_BY CREATED_BY, HAC.CREATION_DATE CREATION_DATE, HAC.LAST_UPDATED_BY LAST_UPDATED_BY, HAC.LAST_UPDATE_DATE LAST_UPDATE_DATE, HAC.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, HAC.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, HACTL.NAME NAME, HACTL.DESCRIPTION DESCRIPTION FROM HRC_ARM_COMPOSITES_B HAC, HRC_ARM_COMPOSITES_TL HACTL WHERE HAC.COMPOSITE_ID = HACTL.COMPOSITE_ID AND HACTL.LANGUAGE = USERENV('LANG')
```
