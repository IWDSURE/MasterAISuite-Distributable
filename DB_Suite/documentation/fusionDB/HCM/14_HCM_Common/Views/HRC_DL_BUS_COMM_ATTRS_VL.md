# HRC_DL_BUS_COMM_ATTRS_VL

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcdlbuscommattrsvl-8028.html#hrcdlbuscommattrsvl-8028](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcdlbuscommattrsvl-8028.html#hrcdlbuscommattrsvl-8028)

## Columns

- BUS_COMM_ATTR_ID
- ATTRIBUTE_LABEL
- ATTRIBUTE_DESCRIPTION
- COMMON_KEY
- MODULE_ID
- ENTERPRISE_ID
- CREATED_BY
- CREATION_DATE
- OBJECT_VERSION_NUMBER
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.BUS_COMM_ATTR_ID BUS_COMM_ATTR_ID, T.ATTRIBUTE_LABEL ATTRIBUTE_LABEL, T.ATTRIBUTE_DESCRIPTION ATTRIBUTE_DESCRIPTION, B.COMMON_KEY COMMON_KEY, B.MODULE_ID MODULE_ID, B.ENTERPRISE_ID ENTERPRISE_ID, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM HRC_DL_BUS_COMM_ATTRS_B B, HRC_DL_BUS_COMM_ATTRS_TL T WHERE B.BUS_COMM_ATTR_ID = T.BUS_COMM_ATTR_ID AND T.LANGUAGE = USERENV('LANG')
```
