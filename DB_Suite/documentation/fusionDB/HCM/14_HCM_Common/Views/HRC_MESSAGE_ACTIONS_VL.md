# HRC_MESSAGE_ACTIONS_VL

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcmessageactionsvl-6286.html#hrcmessageactionsvl-6286](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcmessageactionsvl-6286.html#hrcmessageactionsvl-6286)

## Columns

- ACTION_ID
- CONFIG_ID
- IS_AFFIRMATIVE_ACTION
- ACTION_CODE
- IS_ACTIVE
- ENTERPRISE_ID
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- NAME
- DESCRIPTION

## Query

```sql
SELECT HMAB.ACTION_ID ACTION_ID, HMAB.CONFIG_ID CONFIG_ID, HMAB.IS_AFFIRMATIVE_ACTION, HMAB.ACTION_CODE ACTION_CODE, HMAB.IS_ACTIVE IS_ACTIVE, HMAB.ENTERPRISE_ID ENTERPRISE_ID, HMAB.MODULE_ID MODULE_ID, HMAB.CREATED_BY CREATED_BY, HMAB.CREATION_DATE CREATION_DATE, HMAB.LAST_UPDATED_BY LAST_UPDATED_BY, HMAB.LAST_UPDATE_DATE LAST_UPDATE_DATE, HMAB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, HMAB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, HMATL.NAME NAME, HMATL.DESCRIPTION DESCRIPTION FROM HRC_MESSAGE_ACTIONS_B HMAB, HRC_MESSAGE_ACTIONS_TL HMATL WHERE HMAB.ACTION_ID = HMATL.ACTION_ID AND HMATL.LANGUAGE = USERENV('LANG')
```
