# HRC_DL_STG_PHYSICAL_LINES

Each row in this table corresponds to a Fusion Row that will be MERGED/DELETED on Fusion tables. It Stores actual fusion data using generic columns. It also holds information of its status during IMPORT,LOAD and VALIDATE which are used to calculate status counts of its logical row.

## Details

**Schema:** FUSION

**Object owner:** HRC

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcdlstgphysicallines-21910.html#hrcdlstgphysicallines-21910](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrcdlstgphysicallines-21910.html#hrcdlstgphysicallines-21910)
