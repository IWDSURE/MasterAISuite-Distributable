# HTS_WORK_PATTERN_BREAKS_VL

## Details

**Schema:** FUSION

**Object owner:** HTS

**Object type:** VIEW

- WORK_PATTERN_TYPE_BREAK_ID

- LAST_UPDATE_LOGIN

## Query

```sql
SELECT WPBB.WORK_PATTERN_BREAK_ID WORK_PATTERN_BREAK_ID, WPBB.WORK_PATTERN_BREAK_CODE WORK_PATTERN_BREAK_CODE, WPBTL.BREAK_NAME BREAK_NAME, WPBB.PARENT_RECORD_ID PARENT_RECORD_ID, WPBB.PARENT_RECORD_TYPE PARENT_RECORD_TYPE, WPBB.WORK_PATTERN_TYPE_BREAK_ID WORK_PATTERN_TYPE_BREAK_ID, WPBB.BREAK_TYPE_CODE BREAK_TYPE_CODE, WPBB.BREAK_PAID_CODE BREAK_PAID_CODE, WPBB.START_TIME_OFFSET START_TIME_OFFSET, WPBB.END_TIME_OFFSET END_TIME_OFFSET, WPBB.DURATION DURATION, WPBB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, WPBB.CREATION_DATE CREATION_DATE, WPBB.CREATED_BY CREATED_BY, WPBB.LAST_UPDATED_BY LAST_UPDATED_BY, WPBB.LAST_UPDATE_DATE LAST_UPDATE_DATE, WPBB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, WPBB.ENTERPRISE_ID ENTERPRISE_ID FROM HTS_WORK_PATTERN_BREAKS WPBB, HTS_WORK_PATTERN_BREAKS_TL WPBTL WHERE WPBB.WORK_PATTERN_BREAK_ID = WPBTL.WORK_PATTERN_BREAK_ID AND WPBTL.LANGUAGE = USERENV('LANG')
```
