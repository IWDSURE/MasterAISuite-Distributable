## Query

```sql
SELECT R.CONTENT_ID, R.LEARNING_ITEM_ID, C.NAME, C.DESCRIPTION, C.DESCRIPTION_LONG, C.LANGUAGE_ID, C.DURATION, C.DURATION_UOM, C.STATUS, C.START_DATE, C.END_DATE, R.TRACKING_TYPE, R.AUTHOR, R.KEYWORDS, R.VERSION, R.PRICE, R.CURRENCY, R.IS_COPYRIGHTED, R.COPYRIGHT_TEXT, R.IS_VISIBLE, R.STARTING_URL, R.MASTERY_SCORE, R.LAUNCH_DATA, R.LAST_UPDATE_METHOD, R.EXTERNAL_IDENTIFIER, R.CATALOG, R.CATALOG_NUMBER, R.TIME_LIMIT_ACTION, R.CREATED_BY, R.CREATION_DATE, R.LAST_UPDATED_BY, R.LAST_UPDATE_DATE, R.LAST_UPDATE_LOGIN, R.ENTERPRISE_ID, R.OBJECT_VERSION_NUMBER FROM WLF_LEARNING_ITEMS_F_VL C, WLF_LI_CONTENT_F R WHERE C.LEARNING_ITEM_ID = R.LEARNING_ITEM_ID AND C.STATUS = 'ACTIVE'
```
