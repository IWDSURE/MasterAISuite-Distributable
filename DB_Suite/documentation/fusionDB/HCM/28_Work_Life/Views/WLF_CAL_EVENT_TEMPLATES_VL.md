# WLF_CAL_EVENT_TEMPLATES_VL

**Object owner:** WLF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfcaleventtemplatesvl-5238.html#wlfcaleventtemplatesvl-5238](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfcaleventtemplatesvl-5238.html#wlfcaleventtemplatesvl-5238)

- EVENT_TEMPLATE_ID

- EVENT_TEMPLATE_TYPE
- LANGUAGE
- SOURCE_LANG
- TEMPLATE_BODY_RICH
- TEMPLATE_BODY_PLAIN
- CREATED_BY

- LAST_UPDATE_LOGIN

## Query

```sql
SELECT b.EVENT_TEMPLATE_ID EVENT_TEMPLATE_ID, b.EVENT_TEMPLATE_CODE EVENT_TEMPLATE_CODE, b.EVENT_TEMPLATE_TYPE EVENT_TEMPLATE_TYPE, tl.LANGUAGE LANGUAGE, tl.SOURCE_LANG SOURCE_LANG, tl.TEMPLATE_BODY_RICH TEMPLATE_BODY_RICH, tl.TEMPLATE_BODY_PLAIN TEMPLATE_BODY_PLAIN, b.CREATED_BY CREATED_BY, b.CREATION_DATE CREATION_DATE, b.LAST_UPDATED_BY LAST_UPDATED_BY, b.LAST_UPDATE_DATE LAST_UPDATE_DATE, b.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, b.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM WLF_CAL_EVENT_TEMPLATES_B b, WLF_CAL_EVENT_TEMPLATES_TL tl WHERE b.EVENT_TEMPLATE_ID = tl.EVENT_TEMPLATE_ID AND tl.LANGUAGE = USERENV('LANG')
```
