# WLF_RECOM_CONSUMER_CONFIG_VL

## Details

**Schema:** FUSION

**Object owner:** WLF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfrecomconsumerconfigvl-4323.html#wlfrecomconsumerconfigvl-4323](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfrecomconsumerconfigvl-4323.html#wlfrecomconsumerconfigvl-4323)

- RECOM_CONSUMER_CONFIG_ID
- OBJECT_CATEGORY
- NAME
- OBJECT_TYPE_LOOKUP
- OBJECT_SUB_TYPE_LOOKUP
- CONSUMER_IMPLEMENTATION_CLASS
- ENABLED_FLAG
- DESCRIPTION
- LANGUAGE
- SOURCE_LANG
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- MODULE_ID

## Query

```sql
SELECT b.RECOM_CONSUMER_CONFIG_ID RECOM_CONSUMER_CONFIG_ID, b.OBJECT_CATEGORY OBJECT_CATEGORY, tl.NAME NAME, b.OBJECT_TYPE_LOOKUP OBJECT_TYPE_LOOKUP, b.OBJECT_SUB_TYPE_LOOKUP OBJECT_SUB_TYPE_LOOKUP, b.CONSUMER_IMPLEMENTATION_CLASS CONSUMER_IMPLEMENTATION_CLASS, b.ENABLED_FLAG ENABLED_FLAG, tl.DESCRIPTION DESCRIPTION, tl.LANGUAGE LANGUAGE, tl.SOURCE_LANG SOURCE_LANG, b.CREATED_BY CREATED_BY, b.CREATION_DATE CREATION_DATE, b.LAST_UPDATED_BY LAST_UPDATED_BY, b.LAST_UPDATE_DATE LAST_UPDATE_DATE, b.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, b.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, b.MODULE_ID MODULE_ID FROM WLF_RECOM_CONSUMER_CONFIG_B b, WLF_RECOM_CONSUMER_CONFIG_TL tl WHERE b.RECOM_CONSUMER_CONFIG_ID = tl.RECOM_CONSUMER_CONFIG_ID AND tl.LANGUAGE = USERENV('LANG')
```
