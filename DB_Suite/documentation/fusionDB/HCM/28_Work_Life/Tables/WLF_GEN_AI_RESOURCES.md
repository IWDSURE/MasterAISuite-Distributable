# WLF_GEN_AI_RESOURCES

This table stores the details of the resources used to generate summary using Gen AI.

## Details

**Schema:** FUSION

**Object owner:** WLF

**Object type:** TABLE

**Tablespace:** FUSION_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfgenairesources-8529.html#wlfgenairesources-8529](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfgenairesources-8529.html#wlfgenairesources-8529)

## Primary Key

| Name | Columns |
|------|----------|
| WLF_GEN_AI_RESOURCES_PK | GEN_AI_RESOURCE_ID |
