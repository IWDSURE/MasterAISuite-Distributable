# WLF_SKILL_VARIANTS

This table stores the skill variants of talent profile skills. For example it can be used to store skills variants generated by AI Skill Variant api

## Details

**Schema:** FUSION

**Object owner:** WLF

**Object type:** TABLE

**Tablespace:** FUSION_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfskillvariants-27867.html#wlfskillvariants-27867](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/wlfskillvariants-27867.html#wlfskillvariants-27867)

## Primary Key

| Name | Columns |
|------|----------|
| WLF_SKILL_VARIANTS_PK | SKILL_VARIANT_ID |
