# HRR_TMPL_METRIC_CONFIG_VL

## Details

**Schema:** FUSION

**Object owner:** HRR

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrrtmplmetricconfigvl-6571.html#hrrtmplmetricconfigvl-6571](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrrtmplmetricconfigvl-6571.html#hrrtmplmetricconfigvl-6571)

## Columns

- ENTERPRISE_ID
- METRIC_ID
- DASHBOARD_TMPL_ID
- METRIC_CODE
- CONTENT_TYPE_ID
- DASHBOARD_COLUMN_NAME
- RATING_MODEL_ID
- USE_AS_AXIS_FLAG
- USE_AS_UNDERLAY_FLAG
- READ_ONLY_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LABEL
- SECTION_ID

## Query

```sql
SELECT MCB.ENTERPRISE_ID ENTERPRISE_ID, MCB.METRIC_ID METRIC_ID, MCB.DASHBOARD_TMPL_ID DASHBOARD_TMPL_ID, MCB.METRIC_CODE METRIC_CODE, MCB.CONTENT_TYPE_ID CONTENT_TYPE_ID, MCB.DASHBOARD_COLUMN_NAME DASHBOARD_COLUMN_NAME, MCB.RATING_MODEL_ID RATING_MODEL_ID, MCB.USE_AS_AXIS_FLAG USE_AS_AXIS_FLAG, MCB.USE_AS_UNDERLAY_FLAG USE_AS_UNDERLAY_FLAG, MCB.READ_ONLY_FLAG READ_ONLY_FLAG, MCB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, MCB.CREATED_BY CREATED_BY, MCB.CREATION_DATE CREATION_DATE, MCB.LAST_UPDATED_BY LAST_UPDATED_BY, MCB.LAST_UPDATE_DATE LAST_UPDATE_DATE, MCB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, MCTL.LABEL LABEL, MCB.SECTION_ID FROM HRR_TMPL_METRIC_CONFIG MCB, HRR_TMPL_METRIC_CONFIG_TL MCTL WHERE MCB.METRIC_ID = MCTL.METRIC_ID AND MCB.ENTERPRISE_ID = MCTL.ENTERPRISE_ID AND MCTL.LANGUAGE = USERENV('LANG')
```
