# BEN_BILL_CALENDAR

This table is used to hold bill calendar periods for a given billing cycle

## Details

**Schema:** FUSION

**Object owner:** BEN

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/benbillcalendar-9407.html#benbillcalendar-9407](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/benbillcalendar-9407.html#benbillcalendar-9407)

| Name | Columns |

| BEN_BILL_CALENDAR_PK | BILL_CAL_ID |
