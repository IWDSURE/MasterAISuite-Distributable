## Query

```sql
SELECT prt.RUN_TYPE_ID RUN_TYPE_ID, prt.EFFECTIVE_START_DATE EFFECTIVE_START_DATE, prt.EFFECTIVE_END_DATE EFFECTIVE_END_DATE, prt.BASE_RUN_TYPE_NAME BASE_RUN_TYPE_NAME, prt.RUN_METHOD RUN_METHOD, prt.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, prt.LEGISLATION_CODE LEGISLATION_CODE, prt.ENTERPRISE_ID ENTERPRISE_ID, prt.SHORTNAME SHORTNAME, prt.INCLUSION_FLAG INCLUSION_FLAG, prt.MODULE_ID MODULE_ID, prttl.RUN_TYPE_NAME RUN_TYPE_NAME, prt.LAST_UPDATE_DATE LAST_UPDATE_DATE, prt.LAST_UPDATED_BY LAST_UPDATED_BY, prt.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, prt.CREATED_BY CREATED_BY, prt.CREATION_DATE CREATION_DATE, prt.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, prt.SRS_FLAG SRS_FLAG, prt.TRANSIENT_FLAG, prt.ELEMENT_RECOVER_FLAG, prt.COSTING_FLAG, prt.RETROPAY_FLAG, prt.SGUID SGUID FROM PAY_RUN_TYPES_F prt, PAY_RUN_TYPES_TL prttl WHERE prt.RUN_TYPE_ID = prttl.RUN_TYPE_ID AND LANGUAGE = USERENV('LANG')
```
