# PAY_BAL_EXCEPTION_REPORTS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalexceptionreportsvl-3826.html#paybalexceptionreportsvl-3826](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalexceptionreportsvl-3826.html#paybalexceptionreportsvl-3826)

## Columns

- BAL_EXCEPTION_REPORT_ID
- REPORT_DEFINITION_ID
- BASE_BAL_EXCEP_REPORT_NAME
- BAL_EXCEP_REPORT_NAME
- ENTERPRISE_ID
- LEGISLATION_CODE
- LEGISLATIVE_DATA_GROUP_ID
- CONSOLIDATION_SET_ID
- PAYROLL_ID
- OUTPUT_FORMAT
- OBJECT_VERSION_NUMBER
- CREATED_BY
- LAST_UPDATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- MODULE_ID

## Query

```sql
SELECT PBER.BAL_EXCEPTION_REPORT_ID, PBER.REPORT_DEFINITION_ID, PBER.BASE_BAL_EXCEP_REPORT_NAME, PBERTL.BAL_EXCEP_REPORT_NAME, PBER.ENTERPRISE_ID, PBER.LEGISLATION_CODE, PBER.LEGISLATIVE_DATA_GROUP_ID, PBER.CONSOLIDATION_SET_ID, PBER.PAYROLL_ID, PBER.OUTPUT_FORMAT, PBER.OBJECT_VERSION_NUMBER, PBER.CREATED_BY, PBER.LAST_UPDATED_BY, PBER.CREATION_DATE, PBER.LAST_UPDATE_DATE, PBER.LAST_UPDATE_LOGIN, PBER.MODULE_ID FROM PAY_BAL_EXCEPTION_REPORTS_B PBER, PAY_BAL_EXCEPTION_REPORTS_TL PBERTL WHERE PBER.BAL_EXCEPTION_REPORT_ID = PBERTL.BAL_EXCEPTION_REPORT_ID AND PBERTL.LANGUAGE = USERENV('LANG')
```
