# PAY_DIMENSION_USAGES_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydimensionusagesvl-7732.html#paydimensionusagesvl-7732](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paydimensionusagesvl-7732.html#paydimensionusagesvl-7732)

## Columns

- DIMENSION_USAGE_ID
- BALANCE_DIMENSION_ID
- DATABASE_ITEM_SUFFIX
- DIMENSION_NAME
- DESCRIPTION
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENTERPRISE_ID
- USE_IN_EXCEP_REPORTING_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- MODULE_ID
- SGUID

## Query

```sql
SELECT pdu.DIMENSION_USAGE_ID DIMENSION_USAGE_ID, pdu.BALANCE_DIMENSION_ID BALANCE_DIMENSION_ID, pdu.DATABASE_ITEM_SUFFIX DATABASE_ITEM_SUFFIX, pdutl.DIMENSION_NAME DIMENSION_NAME, pdutl.DESCRIPTION DESCRIPTION, pdu.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pdu.LEGISLATION_CODE LEGISLATION_CODE, pdu.ENTERPRISE_ID ENTERPRISE_ID, pdu.USE_IN_EXCEP_REPORTING_FLAG USE_IN_EXCEP_REPORTING_FLAG, pdu.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pdu.CREATED_BY CREATED_BY, pdu.CREATION_DATE CREATION_DATE, pdu.LAST_UPDATE_DATE LAST_UPDATE_DATE, pdu.LAST_UPDATED_BY LAST_UPDATED_BY, pdu.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pdu.MODULE_ID, pdu.SGUID SGUID FROM PAY_DIMENSION_USAGES pdu, PAY_DIMENSION_USAGES_TL pdutl WHERE pdutl.DIMENSION_USAGE_ID = pdu.DIMENSION_USAGE_ID AND pdutl.LANGUAGE = USERENV('LANG')
```
