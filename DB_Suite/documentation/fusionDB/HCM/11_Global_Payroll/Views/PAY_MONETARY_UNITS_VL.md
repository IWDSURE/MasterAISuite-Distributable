# PAY_MONETARY_UNITS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paymonetaryunitsvl-4544.html#paymonetaryunitsvl-4544](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paymonetaryunitsvl-4544.html#paymonetaryunitsvl-4544)

## Columns

- MONETARY_UNIT_ID
- CURRENCY_CODE
- MONETARY_UNIT_NAME
- LEGISLATION_CODE
- ENTERPRISE_ID
- BASE_MONETARY_UNIT_NAME
- RELATIVE_VALUE
- COMMENTS
- LEGISLATIVE_DATA_GROUP_ID
- MODULE_ID
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- OBJECT_VERSION_NUMBER
- LANGUAGE
- SOURCE_LANG

## Query

```sql
SELECT PAY_MONETARY_UNITS.MONETARY_UNIT_ID MONETARY_UNIT_ID, PAY_MONETARY_UNITS.CURRENCY_CODE CURRENCY_CODE, PAY_MONETARY_UNITS_TL.MONETARY_UNIT_NAME MONETARY_UNIT_NAME, PAY_MONETARY_UNITS.LEGISLATION_CODE LEGISLATION_CODE, PAY_MONETARY_UNITS.ENTERPRISE_ID ENTERPRISE_ID, PAY_MONETARY_UNITS.BASE_MONETARY_UNIT_NAME BASE_MONETARY_UNIT_NAME, PAY_MONETARY_UNITS.RELATIVE_VALUE RELATIVE_VALUE, PAY_MONETARY_UNITS.COMMENTS COMMENTS, PAY_MONETARY_UNITS.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, PAY_MONETARY_UNITS.MODULE_ID MODULE_ID, PAY_MONETARY_UNITS.LAST_UPDATE_DATE LAST_UPDATE_DATE, PAY_MONETARY_UNITS.LAST_UPDATED_BY LAST_UPDATED_BY, PAY_MONETARY_UNITS.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, PAY_MONETARY_UNITS.CREATED_BY CREATED_BY, PAY_MONETARY_UNITS.CREATION_DATE CREATION_DATE, PAY_MONETARY_UNITS.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, PAY_MONETARY_UNITS_TL.LANGUAGE LANGUAGE, PAY_MONETARY_UNITS_TL.SOURCE_LANG SOURCE_LANG FROM PAY_MONETARY_UNITS, PAY_MONETARY_UNITS_TL WHERE PAY_MONETARY_UNITS.MONETARY_UNIT_ID = PAY_MONETARY_UNITS_TL.MONETARY_UNIT_ID AND PAY_MONETARY_UNITS_TL.LANGUAGE = USERENV('LANG')
```
