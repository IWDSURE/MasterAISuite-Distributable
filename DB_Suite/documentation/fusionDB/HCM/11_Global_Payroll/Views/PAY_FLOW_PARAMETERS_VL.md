## Query

```sql
SELECT b.FLOW_PARAMETER_ID, b.BASE_FLOW_PARAMETER_ID, b.BASE_FLOW_PARAMETER_NAME, b.MODULE_ID, b.OBJECT_VERSION_NUMBER, b.BASE_FLOW_ID, b.DISPLAY_FLAG, b.PARAM_DISP_TYPE, b.PARAM_USAGE_TYPE, b.PARAM_VALUE_SET, b.CONTEXT_PARAM_FLAG, b.PARAM_LOOKUP, b.PARAM_SEQUENCE, b.DEFAULT_TYPE, b.DEFAULT_VAL, b.LEGISLATIVE_DATA_GROUP_ID, b.LEGISLATION_CODE, b.ENTERPRISE_ID, b.LAST_UPDATE_DATE, b.LAST_UPDATED_BY, b.LAST_UPDATE_LOGIN, b.CREATED_BY, b.CREATION_DATE, b.PARAMETER_CODE, tl.PARAMETER_NAME, tl.DESCRIPTION, b.LAZY_LOAD, b.SGUID SGUID, b.MULTI_VALUED_FLAG FROM PAY_FLOW_PARAMETERS b, PAY_FLOW_PARAMETERS_TL tl WHERE b.FLOW_PARAMETER_ID = tl.FLOW_PARAMETER_ID AND USERENV('LANG') = tl.LANGUAGE
```
