# PAY_BAL_ATT_DEFINITIONS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalattdefinitionsvl-8322.html#paybalattdefinitionsvl-8322](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paybalattdefinitionsvl-8322.html#paybalattdefinitionsvl-8322)

## Columns

- ATTRIBUTE_ID
- BASE_ATTRIBUTE_NAME
- ALTERABLE
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ENTERPRISE_ID
- USER_ATTRIBUTE_NAME
- DESCRIPTION
- BALANCE_CATEGORY_ID
- BALANCE_DIMENSION_ID
- GROUP_LEVEL_FLAG
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- OBJECT_VERSION_NUMBER
- MODULE_ID

## Query

```sql
SELECT pbad.ATTRIBUTE_ID ATTRIBUTE_ID, pbad.BASE_ATTRIBUTE_NAME BASE_ATTRIBUTE_NAME, pbad.ALTERABLE ALTERABLE, pbad.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pbad.LEGISLATION_CODE LEGISLATION_CODE, pbad.ENTERPRISE_ID ENTERPRISE_ID, pbadtl.USER_ATTRIBUTE_NAME USER_ATTRIBUTE_NAME, pbadtl.DESCRIPTION DESCRIPTION, pbad.BALANCE_CATEGORY_ID BALANCE_CATEGORY_ID, pbad.BALANCE_DIMENSION_ID BALANCE_DIMENSION_ID, pbad.GROUP_LEVEL_FLAG GROUP_LEVEL_FLAG, pbad.LAST_UPDATE_DATE LAST_UPDATE_DATE, pbad.LAST_UPDATED_BY LAST_UPDATED_BY, pbad.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pbad.CREATED_BY CREATED_BY, pbad.CREATION_DATE CREATION_DATE, pbad.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pbad.MODULE_ID MODULE_ID FROM PAY_BAL_ATT_DEFINITIONS pbad, PAY_BAL_ATT_DEFINITIONS_TL pbadtl WHERE pbadtl.ATTRIBUTE_ID = pbad.ATTRIBUTE_ID AND pbadtl.LANGUAGE = USERENV('LANG')
```
