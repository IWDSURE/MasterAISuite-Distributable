## Query

```sql
SELECT B.EVENT_ACTION_TYPE_ID, B.BASE_NAME, B.REPORT_TYPE_REQUIRED, B.OVERRIDE_ACT_SUBMISSION_FLAG, B.APPLICATION_ID, B.ACTION_SUBMISSION, B.JAVA_CLASS, B.DELETE_EVENTS_FLAG, B.PROCESSING_LEVEL, B.SEGMENT_NAME, B.ENTERPRISE_ID, B.LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.MODULE_ID, B.SEED_DATA_SOURCE, B.SGUID, B.ALLOW_MANUAL_NOTIFICATION_FLAG, TL.EVENT_ACTION_TYPE_NAME FROM PAY_EVENT_ACTION_TYPES B, PAY_EVENT_ACTION_TYPES_TL TL WHERE B.EVENT_ACTION_TYPE_ID = TL.EVENT_ACTION_TYPE_ID AND TL.LANGUAGE = USERENV('LANG')
```
