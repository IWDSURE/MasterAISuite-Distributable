# PAY_COST_CLASSIFICATIONS_VL

## Details

**Schema:** FUSION

**Object owner:** PAY

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paycostclassificationsvl-5065.html#paycostclassificationsvl-5065](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/paycostclassificationsvl-5065.html#paycostclassificationsvl-5065)

## Columns

- CLASSIFICATION_NAME
- COST_CLASSIFICATION_ID
- LEGISLATIVE_DATA_GROUP_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER
- ENTERPRISE_ID

## Query

```sql
SELECT pcctl.CLASSIFICATION_NAME CLASSIFICATION_NAME, pcc.COST_CLASSIFICATION_ID COST_CLASSIFICATION_ID, pcc.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, pcc.CREATED_BY CREATED_BY, pcc.CREATION_DATE CREATION_DATE, pcc.LAST_UPDATE_DATE LAST_UPDATE_DATE, pcc.LAST_UPDATED_BY LAST_UPDATED_BY, pcc.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, pcc.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, pcc.ENTERPRISE_ID ENTERPRISE_ID FROM PAY_COST_CLASSIFICATIONS pcc, PAY_COST_CLASSIFICATIONS_TL pcctl WHERE pcctl.COST_CLASSIFICATION_ID = pcc.COST_CLASSIFICATION_ID AND pcctl.LANGUAGE = USERENV('LANG')
```
