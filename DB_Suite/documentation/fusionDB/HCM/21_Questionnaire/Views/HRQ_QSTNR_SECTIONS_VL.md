# HRQ_QSTNR_SECTIONS_VL

## Details

**Schema:** FUSION

**Object owner:** HRQ

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqqstnrsectionsvl-3146.html#hrqqstnrsectionsvl-3146](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqqstnrsectionsvl-3146.html#hrqqstnrsectionsvl-3146)

## Columns

- QSTNR_SECTION_ID
- BUSINESS_GROUP_ID
- SECTION_SEQ_NUM
- NEW_PAGE
- NAME
- QUESTIONNAIRE_ID
- QSTNR_VERSION_NUM
- DESCRIPTION
- INTRO_TEXT
- QUESTION_ORDER
- RESPONSE_ORDER
- MANDATORY
- ALLOW_ADHOC
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- RANDOM_SAMPLE_ALL_FLAG
- RANDOM_SAMPLE_SIZE

## Query

```sql
SELECT B.QSTNR_SECTION_ID, B.BUSINESS_GROUP_ID, B.SECTION_SEQ_NUM, NEW_PAGE, L.NAME, B.QUESTIONNAIRE_ID, B.QSTNR_VERSION_NUM, L.DESCRIPTION, L.INTRO_TEXT, B.QUESTION_ORDER, B.RESPONSE_ORDER, B.MANDATORY, B.ALLOW_ADHOC, B.MODULE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.RANDOM_SAMPLE_ALL_FLAG, B.RANDOM_SAMPLE_SIZE FROM HRQ_QSTNR_SECTIONS_B B, HRQ_QSTNR_SECTIONS_TL L WHERE B.BUSINESS_GROUP_ID = L.BUSINESS_GROUP_ID AND B.QSTNR_SECTION_ID = L.QSTNR_SECTION_ID AND L.LANGUAGE = USERENV('LANG')
```
