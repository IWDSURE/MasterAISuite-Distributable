# HRQ_SUBSCRIBERS_VL

## Details

**Schema:** FUSION

**Object owner:** HRQ

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqsubscribersvl-4615.html#hrqsubscribersvl-4615](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqsubscribersvl-4615.html#hrqsubscribersvl-4615)

## Columns

- SUBSCRIBER_ID
- BUSINESS_GROUP_ID
- SUBSCRIBER_CODE
- NAME
- INCLUDE_NA_OPT
- INCLUDE_NA_TEXT
- INTERFACE_CLASS
- DEFAULT_QUESTIONNAIRE_ID
- DEFAULT_RATING_MODEL_ID
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.SUBSCRIBER_ID, B.BUSINESS_GROUP_ID, B.SUBSCRIBER_CODE, L.NAME, B.INCLUDE_NA_OPT, L.INCLUDE_NA_TEXT, B.INTERFACE_CLASS, B.DEFAULT_QUESTIONNAIRE_ID, B.DEFAULT_RATING_MODEL_ID, B.MODULE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM HRQ_SUBSCRIBERS_B B, HRQ_SUBSCRIBERS_TL L WHERE B.SUBSCRIBER_ID = L.SUBSCRIBER_ID AND B.BUSINESS_GROUP_ID = L.BUSINESS_GROUP_ID AND L.LANGUAGE = USERENV('LANG')
```
