# HRQ_QSTNR_TEMPLATES_VL

## Details

**Schema:** FUSION

**Object owner:** HRQ

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqqstnrtemplatesvl-5621.html#hrqqstnrtemplatesvl-5621](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrqqstnrtemplatesvl-5621.html#hrqqstnrtemplatesvl-5621)

## Columns

- BUSINESS_GROUP_ID
- QSTNR_TEMPLATE_ID
- NAME
- QUESTIONNAIRE_ID
- ALLOW_FORMAT_CHG
- ALLOW_INTRO_CHG
- ALLOW_SECTION_CHG
- ALLOW_QSTN_ORDER_CHG
- ALLOW_RESP_ORDER_CHG
- BASIC_SETUP_FLAG
- RANDOM_QSTNS
- RANDOM_ANSWERS
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.BUSINESS_GROUP_ID, B.QSTNR_TEMPLATE_ID, L.NAME, B.QUESTIONNAIRE_ID, B.ALLOW_FORMAT_CHG, B.ALLOW_INTRO_CHG, B.ALLOW_SECTION_CHG, B.ALLOW_QSTN_ORDER_CHG, B.ALLOW_RESP_ORDER_CHG, B.BASIC_SETUP_FLAG, B.RANDOM_QSTNS, B.RANDOM_ANSWERS, B.MODULE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM HRQ_QSTNR_TEMPLATES_B B, HRQ_QSTNR_TEMPLATES_TL L WHERE B.BUSINESS_GROUP_ID = L.BUSINESS_GROUP_ID AND B.QSTNR_TEMPLATE_ID = L.QSTNR_TEMPLATE_ID AND L.LANGUAGE = USERENV('LANG')
```
