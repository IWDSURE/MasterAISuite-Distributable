# FF_USER_ROWS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffuserrowsvl-6132.html#ffuserrowsvl-6132](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffuserrowsvl-6132.html#ffuserrowsvl-6132)

## Columns

- USER_ROW_ID
- EFFECTIVE_START_DATE
- EFFECTIVE_END_DATE
- ROW_LOW_RANGE_OR_NAME
- ROW_NAME
- USER_TABLE_ID
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- ROW_HIGH_RANGE
- DISPLAY_SEQUENCE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID
- SGUID

## Query

```sql
SELECT B.USER_ROW_ID USER_ROW_ID, B.EFFECTIVE_START_DATE EFFECTIVE_START_DATE, B.EFFECTIVE_END_DATE EFFECTIVE_END_DATE, B.ROW_LOW_RANGE_OR_NAME ROW_LOW_RANGE_OR_NAME, TL.ROW_NAME ROW_NAME, B.USER_TABLE_ID USER_TABLE_ID, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.ROW_HIGH_RANGE ROW_HIGH_RANGE, B.DISPLAY_SEQUENCE DISPLAY_SEQUENCE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID, B.SGUID SGUID FROM FF_USER_ROWS_F B, FF_USER_ROWS_TL TL WHERE B.USER_ROW_ID = TL.USER_ROW_ID AND TL.LANGUAGE = USERENV('LANG')
```
