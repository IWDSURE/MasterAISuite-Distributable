# FF_GLOBALS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffglobalsvl-8556.html#ffglobalsvl-8556](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffglobalsvl-8556.html#ffglobalsvl-8556)

## Columns

- GLOBAL_ID
- EFFECTIVE_START_DATE
- EFFECTIVE_END_DATE
- BASE_GLOBAL_NAME
- GLOBAL_NAME
- DESCRIPTION
- GLOBAL_VALUE
- DATA_TYPE
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID
- SGUID

## Query

```sql
SELECT B.GLOBAL_ID GLOBAL_ID, B.EFFECTIVE_START_DATE EFFECTIVE_START_DATE, B.EFFECTIVE_END_DATE EFFECTIVE_END_DATE, B.BASE_GLOBAL_NAME BASE_GLOBAL_NAME, TL.GLOBAL_NAME GLOBAL_NAME, TL.DESCRIPTION DESCRIPTION, B.GLOBAL_VALUE GLOBAL_VALUE, B.DATA_TYPE DATA_TYPE, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID, B.SGUID SGUID FROM FF_GLOBALS_B_F B, FF_GLOBALS_TL TL WHERE B.GLOBAL_ID = TL.GLOBAL_ID AND TL.LANGUAGE = USERENV('LANG')
```
