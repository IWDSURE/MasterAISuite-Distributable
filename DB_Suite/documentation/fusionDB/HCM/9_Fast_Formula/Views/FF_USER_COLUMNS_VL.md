# FF_USER_COLUMNS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffusercolumnsvl-6373.html#ffusercolumnsvl-6373](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffusercolumnsvl-6373.html#ffusercolumnsvl-6373)

## Columns

- USER_COLUMN_ID
- BASE_USER_COLUMN_NAME
- USER_COLUMN_NAME
- USER_TABLE_ID
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- DATA_TYPE
- FORMULA_ID
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID
- SGUID

## Query

```sql
SELECT B.USER_COLUMN_ID USER_COLUMN_ID, B.BASE_USER_COLUMN_NAME BASE_USER_COLUMN_NAME, TL.USER_COLUMN_NAME, B.USER_TABLE_ID USER_TABLE_ID, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.DATA_TYPE DATA_TYPE, B.FORMULA_ID FORMULA_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID, B.SGUID SGUID FROM FF_USER_COLUMNS B, FF_USER_COLUMNS_TL TL WHERE B.USER_COLUMN_ID = TL.USER_COLUMN_ID AND TL.LANGUAGE = USERENV('LANG')
```
