# FF_FUNCTIONS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/fffunctionsvl-4727.html#fffunctionsvl-4727](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/fffunctionsvl-4727.html#fffunctionsvl-4727)

## Columns

- FUNCTION_ID
- FUNCTION_KEY
- BASE_FUNCTION_NAME
- DESCRIPTION
- ALIAS_NAME
- INTERPRETER_CALL_FLAG
- DEFINITION
- DATA_TYPE
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATION_DATE
- CREATED_BY
- MODULE_ID

## Query

```sql
SELECT B.FUNCTION_ID FUNCTION_ID, B.FUNCTION_KEY FUNCTION_KEY, B.BASE_FUNCTION_NAME BASE_FUNCTION_NAME, TL.DESCRIPTION DESCRIPTION, B.ALIAS_NAME ALIAS_NAME, B.INTERPRETER_CALL_FLAG INTERPRETER_CALL_FLAG, B.DEFINITION DEFINITION, B.DATA_TYPE DATA_TYPE, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.MODULE_ID MODULE_ID FROM FF_FUNCTIONS_B B, FF_FUNCTIONS_TL TL WHERE B.FUNCTION_ID = TL.FUNCTION_ID AND TL.LANGUAGE = USERENV('LANG') WITH CHECK OPTION
```
