# FF_USER_ENTITIES_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffuserentitiesvl-6389.html#ffuserentitiesvl-6389](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffuserentitiesvl-6389.html#ffuserentitiesvl-6389)

## Columns

- USER_ENTITY_ID
- ROUTE_ID
- CREATOR_TYPE
- CREATOR_ID
- BASE_USER_ENTITY_NAME
- USER_ENTITY_NAME
- DESCRIPTION
- NOTFOUND_ALLOWED_FLAG
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATION_DATE
- CREATED_BY
- MODULE_ID

## Query

```sql
SELECT B.USER_ENTITY_ID USER_ENTITY_ID, B.ROUTE_ID ROUTE_ID, B.CREATOR_TYPE CREATOR_TYPE, B.CREATOR_ID CREATOR_ID, B.BASE_USER_ENTITY_NAME BASE_USER_ENTITY_NAME, TL.USER_ENTITY_NAME USER_ENTITY_NAME, TL.DESCRIPTION DESCRIPTION, B.NOTFOUND_ALLOWED_FLAG NOTFOUND_ALLOWED_FLAG, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.MODULE_ID MODULE_ID FROM FF_USER_ENTITIES_B B, FF_USER_ENTITIES_TL TL WHERE B.USER_ENTITY_ID = TL.USER_ENTITY_ID AND TL.LANGUAGE = USERENV('LANG')
```
