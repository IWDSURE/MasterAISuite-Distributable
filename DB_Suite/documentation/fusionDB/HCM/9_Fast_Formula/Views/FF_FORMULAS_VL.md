# FF_FORMULAS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffformulasvl-5069.html#ffformulasvl-5069](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffformulasvl-5069.html#ffformulasvl-5069)

## Columns

- FORMULA_ID
- EFFECTIVE_START_DATE
- EFFECTIVE_END_DATE
- FORMULA_TYPE_ID
- BASE_FORMULA_NAME
- FORMULA_NAME
- DESCRIPTION
- EDIT_STATUS
- FORMULA_TEXT
- COMPILE_FLAG
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID
- SGUID

## Query

```sql
SELECT B.FORMULA_ID FORMULA_ID, B.EFFECTIVE_START_DATE EFFECTIVE_START_DATE, B.EFFECTIVE_END_DATE EFFECTIVE_END_DATE, B.FORMULA_TYPE_ID FORMULA_TYPE_ID, B.BASE_FORMULA_NAME BASE_FORMULA_NAME, TL.FORMULA_NAME FORMULA_NAME, TL.DESCRIPTION DESCRIPTION, B.EDIT_STATUS EDIT_STATUS, B.FORMULA_TEXT FORMULA_TEXT, B.COMPILE_FLAG COMPILE_FLAG, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID, B.SGUID SGUID FROM FF_FORMULAS_B_F B, FF_FORMULAS_TL TL WHERE B.FORMULA_ID = TL.FORMULA_ID AND TL.LANGUAGE = USERENV('LANG')
```
