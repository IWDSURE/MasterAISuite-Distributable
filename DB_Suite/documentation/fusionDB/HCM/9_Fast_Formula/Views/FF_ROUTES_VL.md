# FF_ROUTES_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffroutesvl-3623.html#ffroutesvl-3623](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffroutesvl-3623.html#ffroutesvl-3623)

## Columns

- ROUTE_ID
- BASE_ROUTE_NAME
- DESCRIPTION
- TEXT
- OPTIMIZER_HINT
- MULTI_ROW_FLAG
- USER_DEFINED_FLAG
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID

## Query

```sql
SELECT B.ROUTE_ID ROUTE_ID, B.BASE_ROUTE_NAME BASE_ROUTE_NAME, TL.DESCRIPTION DESCRIPTION, B.TEXT TEXT, B.OPTIMIZER_HINT OPTIMIZER_HINT, B.MULTI_ROW_FLAG MULTI_ROW_FLAG, B.USER_DEFINED_FLAG USER_DEFINED_FLAG, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID FROM FF_ROUTES_B B, FF_ROUTES_TL TL WHERE B.ROUTE_ID = TL.ROUTE_ID AND TL.LANGUAGE = USERENV('LANG') WITH CHECK OPTION
```
