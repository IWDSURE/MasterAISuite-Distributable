# FF_FORMULA_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffformulatypesvl-6044.html#ffformulatypesvl-6044](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffformulatypesvl-6044.html#ffformulatypesvl-6044)

## Columns

- FORMULA_TYPE_ID
- BASE_FORMULA_TYPE_NAME
- EDIT_TYPE
- PARTITION_TYPE
- FORMULA_TYPE_NAME
- DESCRIPTION
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID
- ENTERPRISE_ID

## Query

```sql
SELECT B.FORMULA_TYPE_ID FORMULA_TYPE_ID, B.BASE_FORMULA_TYPE_NAME BASE_FORMULA_TYPE_NAME, B.EDIT_TYPE EDIT_TYPE, B.PARTITION_TYPE PARTITION_TYPE, TL.FORMULA_TYPE_NAME FORMULA_TYPE_NAME, TL.DESCRIPTION DESCRIPTION, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE , B.MODULE_ID, B.ENTERPRISE_ID ENTERPRISE_ID FROM FF_FORMULA_TYPES_B B, FF_FORMULA_TYPES_TL TL WHERE B.FORMULA_TYPE_ID = TL.FORMULA_TYPE_ID AND TL.LANGUAGE = USERENV('LANG')
```
