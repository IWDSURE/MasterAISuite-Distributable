# FF_CONTEXTS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffcontextsvl-7744.html#ffcontextsvl-7744](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffcontextsvl-7744.html#ffcontextsvl-7744)

## Columns

- CONTEXT_ID
- BASE_CONTEXT_NAME
- CONTEXT_NAME
- DESCRIPTION
- DATA_TYPE
- USER_CONTEXT_FLAG
- ENTERPRISE_ID
- LEGISLATIVE_DATA_GROUP_ID
- LEGISLATION_CODE
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATED_BY
- CREATION_DATE
- MODULE_ID

## Query

```sql
SELECT B.CONTEXT_ID CONTEXT_ID, B.BASE_CONTEXT_NAME BASE_CONTEXT_NAME, TL.CONTEXT_NAME CONTEXT_NAME, TL.DESCRIPTION DESCRIPTION, B.DATA_TYPE DATA_TYPE, B.USER_CONTEXT_FLAG USER_CONTEXT_FLAG, B.ENTERPRISE_ID ENTERPRISE_ID, B.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, B.LEGISLATION_CODE LEGISLATION_CODE, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.MODULE_ID MODULE_ID FROM FF_CONTEXTS_B B, FF_CONTEXTS_TL TL WHERE B.CONTEXT_ID = TL.CONTEXT_ID AND TL.LANGUAGE = USERENV('LANG')
```
