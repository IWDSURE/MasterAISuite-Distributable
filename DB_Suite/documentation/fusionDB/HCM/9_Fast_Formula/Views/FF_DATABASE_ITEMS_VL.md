# FF_DATABASE_ITEMS_VL

## Details

**Schema:** FUSION

**Object owner:** FF

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffdatabaseitemsvl-6479.html#ffdatabaseitemsvl-6479](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ffdatabaseitemsvl-6479.html#ffdatabaseitemsvl-6479)

## Columns

- DATABASE_ITEM_ID
- BASE_USER_NAME
- USER_NAME
- DESCRIPTION
- USER_ENTITY_ID
- DATA_TYPE
- NULL_ALLOWED_FLAG
- DEFINITION_TEXT
- OBJECT_VERSION_NUMBER
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- CREATION_DATE
- CREATED_BY
- MODULE_ID
- ENTERPRISE_ID
- SGUID

## Query

```sql
SELECT B.DATABASE_ITEM_ID DATABASE_ITEM_ID, B.BASE_USER_NAME BASE_USER_NAME, B.BASE_USER_NAME USER_NAME, TL.DESCRIPTION DESCRIPTION, B.USER_ENTITY_ID USER_ENTITY_ID, B.DATA_TYPE DATA_TYPE, B.NULL_ALLOWED_FLAG NULL_ALLOWED_FLAG, B.DEFINITION_TEXT DEFINITION_TEXT, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.MODULE_ID MODULE_ID, B.ENTERPRISE_ID ENTERPRISE_ID, B.SGUID SGUID FROM FF_DATABASE_ITEMS_B B, FF_DATABASE_ITEMS_TL TL WHERE B.DATABASE_ITEM_ID = TL.DATABASE_ITEM_ID AND TL.LANGUAGE = USERENV('LANG')
```
