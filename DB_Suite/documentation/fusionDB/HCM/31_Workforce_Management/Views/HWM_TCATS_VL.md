# HWM_TCATS_VL

## Details

**Schema:** FUSION

**Object owner:** HWM

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwmtcatsvl-4719.html#hwmtcatsvl-4719](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hwmtcatsvl-4719.html#hwmtcatsvl-4719)

## Columns

- TCAT_ID
- TCAT_NAME
- DESCRIPTION
- TCAT_CD
- MODULE_ID
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- LAST_UPDATED_BY
- OBJECT_VERSION_NUMBER
- ENTERPRISE_ID
- PERSISTENT_FLAG
- SGUID
- UOM
- ACR_RULE_ID

## Query

```sql
SELECT HWM_TCATS_B.TCAT_ID TCAT_ID, HWM_TCATS_TL.TCAT_NAME TCAT_NAME, HWM_TCATS_TL.DESCRIPTION DESCRIPTION, HWM_TCATS_B.TCAT_CD TCAT_CD, HWM_TCATS_B.MODULE_ID MODULE_ID, HWM_TCATS_B.CREATED_BY CREATED_BY, HWM_TCATS_B.CREATION_DATE CREATION_DATE, HWM_TCATS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, HWM_TCATS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, HWM_TCATS_B.LAST_UPDATED_BY LAST_UPDATED_BY, HWM_TCATS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, HWM_TCATS_B.ENTERPRISE_ID ENTERPRISE_ID, HWM_TCATS_B.PERSISTENT_FLAG PERSISTENT_FLAG, HWM_TCATS_B.SGUID SGUID, HWM_TCATS_B.UOM UOM, HWM_TCATS_B.ACR_RULE_ID FROM HWM_TCATS_B, HWM_TCATS_TL WHERE HWM_TCATS_B.TCAT_ID = HWM_TCATS_TL.TCAT_ID AND HWM_TCATS_TL.LANGUAGE = USERENV('LANG')
```
