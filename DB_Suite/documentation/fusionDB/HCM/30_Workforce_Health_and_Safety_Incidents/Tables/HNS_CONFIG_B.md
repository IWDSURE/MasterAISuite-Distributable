# HNS_CONFIG_B

HNS CONF SETUP. This is the base table for Configuration setup. This stores all system configuration 
related data.

## Details

**Schema:** FUSION

**Object owner:** HNS

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnsconfigb-8119.html#hnsconfigb-8119](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnsconfigb-8119.html#hnsconfigb-8119)

## Primary Key

| Name | Columns |
|------|----------|
| HNS_CONFIG_B_PK | CONFIG_ID |
