# HNS_SUBJECT_VL

## Details

**Schema:** FUSION

**Object owner:** HNS

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnssubjectvl-7182.html#hnssubjectvl-7182](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnssubjectvl-7182.html#hnssubjectvl-7182)

## Columns

- SUBJECT_ID
- SUBJECT_AREA_ID
- SUBJECT_CODE
- DISPLAY_SEQUENCE
- ENABLED_FLAG
- DELETED_FLAG
- SUBJECT_NAME
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT HNS_SUBJECT_B.SUBJECT_ID, HNS_SUBJECT_B.SUBJECT_AREA_ID, HNS_SUBJECT_B.SUBJECT_CODE, HNS_SUBJECT_B.DISPLAY_SEQUENCE, HNS_SUBJECT_B.ENABLED_FLAG, HNS_SUBJECT_B.DELETED_FLAG, HNS_SUBJECT_TL.SUBJECT_NAME, HNS_SUBJECT_B.CREATED_BY, HNS_SUBJECT_B.CREATION_DATE, HNS_SUBJECT_B.LAST_UPDATED_BY, HNS_SUBJECT_B.LAST_UPDATE_DATE, HNS_SUBJECT_B.LAST_UPDATE_LOGIN, HNS_SUBJECT_B.OBJECT_VERSION_NUMBER FROM HNS_SUBJECT_B INNER JOIN HNS_SUBJECT_TL ON HNS_SUBJECT_B.SUBJECT_ID = HNS_SUBJECT_TL.SUBJECT_ID AND HNS_SUBJECT_TL.LANGUAGE = USERENV('LANG')
```
