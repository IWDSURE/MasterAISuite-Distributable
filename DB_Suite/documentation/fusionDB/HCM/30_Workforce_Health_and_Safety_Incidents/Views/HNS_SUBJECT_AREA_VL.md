# HNS_SUBJECT_AREA_VL

## Details

**Schema:** FUSION

**Object owner:** HNS

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnssubjectareavl-7598.html#hnssubjectareavl-7598](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hnssubjectareavl-7598.html#hnssubjectareavl-7598)

## Columns

- SUBJECT_AREA_ID
- SUBJECT_AREA_CODE
- DISPLAY_SEQUENCE
- DELETED_FLAG
- ENABLED_FLAG
- SUBJECT_AREA_NAME
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT HNS_SUBJECT_AREA_B.SUBJECT_AREA_ID, HNS_SUBJECT_AREA_B.SUBJECT_AREA_CODE, HNS_SUBJECT_AREA_B.DISPLAY_SEQUENCE, HNS_SUBJECT_AREA_B.DELETED_FLAG, HNS_SUBJECT_AREA_B.ENABLED_FLAG, HNS_SUBJECT_AREA_TL.SUBJECT_AREA_NAME, HNS_SUBJECT_AREA_B.CREATED_BY, HNS_SUBJECT_AREA_B.CREATION_DATE, HNS_SUBJECT_AREA_B.LAST_UPDATED_BY, HNS_SUBJECT_AREA_B.LAST_UPDATE_DATE, HNS_SUBJECT_AREA_B.LAST_UPDATE_LOGIN, HNS_SUBJECT_AREA_B.OBJECT_VERSION_NUMBER FROM HNS_SUBJECT_AREA_B INNER JOIN HNS_SUBJECT_AREA_TL ON HNS_SUBJECT_AREA_B.SUBJECT_AREA_ID = HNS_SUBJECT_AREA_TL.SUBJECT_AREA_ID AND HNS_SUBJECT_AREA_TL.LANGUAGE = USERENV('LANG')
```
