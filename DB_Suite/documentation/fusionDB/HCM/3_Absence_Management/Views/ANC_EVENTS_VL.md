# ANC_EVENTS_VL

## Details

**Schema:** FUSION

**Object owner:** ANC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anceventsvl-7184.html#anceventsvl-7184](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anceventsvl-7184.html#anceventsvl-7184)

- LAST_UPDATE_DATE

## Query

```sql
SELECT ANC_EVENTS.EVENT_ID EVENT_ID, ANC_EVENTS.ENTERPRISE_ID ENTERPRISE_ID, ANC_EVENTS.EVENT_TYPE EVENT_TYPE, ANC_EVENTS.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, ANC_EVENTS.CREATED_BY CREATED_BY, ANC_EVENTS.CREATION_DATE CREATION_DATE, ANC_EVENTS.LAST_UPDATED_BY LAST_UPDATED_BY, ANC_EVENTS.LAST_UPDATE_DATE LAST_UPDATE_DATE, ANC_EVENTS.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, ANC_EVENTS.MODULE_ID MODULE_ID, ANC_EVENTS.SGUID SGUID, ANC_EVENTS.SEED_DATA_SOURCE SEED_DATA_SOURCE, ANC_EVENTS_TL.EVENT_ID EVENT_ID1, ANC_EVENTS_TL.LANGUAGE LANGUAGE, ANC_EVENTS_TL.ENTERPRISE_ID ENTERPRISE_ID1, ANC_EVENTS_TL.NAME NAME, ANC_EVENTS_TL.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER1, ANC_EVENTS_TL.DESCRIPTION DESCRIPTION, ANC_EVENTS_TL.SOURCE_LANG SOURCE_LANG, ANC_EVENTS_TL.MODULE_ID MODULE_ID1, ANC_EVENTS_TL.SEED_DATA_SOURCE SEED_DATA_SOURCE1 FROM ANC_EVENTS, ANC_EVENTS_TL WHERE ANC_EVENTS.EVENT_ID = ANC_EVENTS_TL.EVENT_ID AND ANC_EVENTS_TL.LANGUAGE = USERENV('LANG')
```
