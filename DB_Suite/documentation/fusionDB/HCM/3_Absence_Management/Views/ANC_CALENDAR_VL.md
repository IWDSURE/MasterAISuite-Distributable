# ANC_CALENDAR_VL

## Details

**Schema:** FUSION

**Object owner:** ANC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anccalendarvl-3326.html#anccalendarvl-3326](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anccalendarvl-3326.html#anccalendarvl-3326)

## Columns

- CALENDAR_ID
- NAME
- DESCRIPTION
- ENTERPRISE_ID
- START_DATE
- END_DATE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT B.CALENDAR_ID CALENDAR_ID, T.NAME NAME, T.DESCRIPTION DESCRIPTION, B.ENTERPRISE_ID ENTERPRISE_ID, B.START_DATE START_DATE, B.END_DATE END_DATE, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM ANC_CALENDAR_B B, ANC_CALENDAR_TL T WHERE B.CALENDAR_ID = T.CALENDAR_ID AND T.LANGUAGE = USERENV('LANG')
```
