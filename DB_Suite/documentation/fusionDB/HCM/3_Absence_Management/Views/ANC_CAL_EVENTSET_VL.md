# ANC_CAL_EVENTSET_VL

## Details

**Schema:** FUSION

**Object owner:** ANC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anccaleventsetvl-5694.html#anccaleventsetvl-5694](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/anccaleventsetvl-5694.html#anccaleventsetvl-5694)

## Columns

- CAL_EVENTSET_ID
- NAME
- DESCRIPTION
- ENTERPRISE_ID
- START_DATE
- END_DATE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT B.CAL_EVENTSET_ID CAL_EVENTSET_ID, T.NAME NAME, T.DESCRIPTION DESCRIPTION, B.ENTERPRISE_ID ENTERPRISE_ID, B.START_DATE START_DATE, B.END_DATE END_DATE, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM ANC_CAL_EVENTSET_B B, ANC_CAL_EVENTSET_TL T WHERE B.CAL_EVENTSET_ID = T.CAL_EVENTSET_ID AND T.LANGUAGE = USERENV('LANG')
```
