# ANC_AVAILABILITY_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** ANC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ancavailabilitytypesvl-8532.html#ancavailabilitytypesvl-8532](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ancavailabilitytypesvl-8532.html#ancavailabilitytypesvl-8532)

## Columns

- AVAILABILITY_TYPE_ID
- ENTERPRISE_ID
- NAME
- ACTIVE_STATUS
- CONTRACTED_HOURS
- WORK_SHIFT_TYPE
- REPEATED_PATTERN_TYPE
- LEGISLATION_CODE
- IS_WORK_PATTERN_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.AVAILABILITY_TYPE_ID, B.ENTERPRISE_ID, T.NAME, B.ACTIVE_STATUS, B.CONTRACTED_HOURS, B.WORK_SHIFT_TYPE, B.REPEATED_PATTERN_TYPE, B.LEGISLATION_CODE, B.IS_WORK_PATTERN_FLAG, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM ANC_AVAILABILITY_TYPES_B B, ANC_AVAILABILITY_TYPES_TL T WHERE B.AVAILABILITY_TYPE_ID = T.AVAILABILITY_TYPE_ID AND T.LANGUAGE = USERENV('LANG')
```
