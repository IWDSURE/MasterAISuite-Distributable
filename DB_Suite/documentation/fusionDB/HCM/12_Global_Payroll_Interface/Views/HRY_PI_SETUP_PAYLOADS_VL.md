## Query

```sql
SELECT Base.SETUP_PAYLOAD_ID SETUP_PAYLOAD_ID, Base.BASE_SETUP_PAYLOAD_NAME BASE_SETUP_PAYLOAD_NAME, Base.PI_PAYLOAD_DATA PI_PAYLOAD_DATA, Tras.SETUP_PAYLOAD_NAME SETUP_PAYLOAD_NAME, Base.ENTERPRISE_ID ENTERPRISE_ID, Base.LEGISLATIVE_DATA_GROUP_ID LEGISLATIVE_DATA_GROUP_ID, Base.LEGISLATION_CODE LEGISLATION_CODE, Base.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, Base.LAST_UPDATE_DATE LAST_UPDATE_DATE, Base.LAST_UPDATED_BY LAST_UPDATED_BY, Base.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, Base.CREATED_BY CREATED_BY, Base.CREATION_DATE CREATION_DATE, Base.SGUID SGUID, Base.MODULE_ID MODULE_ID, Base.PI_PAYLOAD_TYPE PI_PAYLOAD_TYPE FROM HRY_PI_SETUP_PAYLOADS_B Base, HRY_PI_SETUP_PAYLOADS_TL Tras WHERE Base.SETUP_PAYLOAD_ID = Tras.SETUP_PAYLOAD_ID AND Tras.LANGUAGE = USERENV('LANG')
```
