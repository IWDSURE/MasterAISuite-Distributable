# HRY_PI_RTI_PROCESS_LOG

To store the custom logs generated by RTI for a single person for troubleshooting purposes.

## Details

**Schema:** FUSION

**Object owner:** HRY

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrypirtiprocesslog-29211.html#hrypirtiprocesslog-29211](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrypirtiprocesslog-29211.html#hrypirtiprocesslog-29211)

## Primary Key

| Name | Columns |
|------|----------|
| hry_pi_rti_process_log_PK | PROCESS_LOG_ID |
