# HRG_GOAL_PLAN_SETS_VL

## Details

**Schema:** FUSION

**Object owner:** HRG

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrggoalplansetsvl-3136.html#hrggoalplansetsvl-3136](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrggoalplansetsvl-3136.html#hrggoalplansetsvl-3136)

## Columns

- GOAL_PLAN_SET_ID

- GOAL_PLAN_SET_EXT_ID
- GOAL_PLAN_SET_NAME

- START_DATE
- END_DATE
- MASS_REQUEST_ID
- GOAL_PLAN_SET_ACTIVE_FLAG
- REQUEST_UI_CONTEXT_CODE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- REVIEW_PERIOD_ID
- REQUEST_ID
- REQUEST_STATUS

## Query

```sql
SELECT GPSB.GOAL_PLAN_SET_ID, GPSB.BUSINESS_GROUP_ID, GPSB.GOAL_PLAN_SET_EXT_ID, GPSTL.GOAL_PLAN_SET_NAME, GPSTL.DESCRIPTION, GPSB.START_DATE, GPSB.END_DATE, GPSB.MASS_REQUEST_ID, GPSB.GOAL_PLAN_SET_ACTIVE_FLAG, GPSB.REQUEST_UI_CONTEXT_CODE, GPSB.OBJECT_VERSION_NUMBER, GPSB.CREATED_BY, GPSB.CREATION_DATE, GPSB.LAST_UPDATE_DATE, GPSB.LAST_UPDATED_BY, GPSB.LAST_UPDATE_LOGIN, GPSB.REVIEW_PERIOD_ID, GPSB.REQUEST_ID, GPSB.REQUEST_STATUS FROM HRG_GOAL_PLAN_SETS_B GPSB, HRG_GOAL_PLAN_SETS_TL GPSTL WHERE GPSB.GOAL_PLAN_SET_ID = GPSTL.GOAL_PLAN_SET_ID AND GPSTL.LANGUAGE = USERENV('LANG')
```
