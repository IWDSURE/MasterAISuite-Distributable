# HRA_EVAL_RATINGS_

This table stores the evaluation items and sections ratings

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** TABLE

**Tablespace:** APPS_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalratings-9639.html#hraevalratings-9639](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalratings-9639.html#hraevalratings-9639)

| Name | Columns |
|------|----------|
