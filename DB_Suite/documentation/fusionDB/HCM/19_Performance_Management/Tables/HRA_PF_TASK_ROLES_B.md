# HRA_PF_TASK_ROLES_B

This table stores the task roles involved in the process flow definition

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** TABLE

**Tablespace:** REFERENCE

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrapftaskrolesb-19624.html#hrapftaskrolesb-19624](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrapftaskrolesb-19624.html#hrapftaskrolesb-19624)
