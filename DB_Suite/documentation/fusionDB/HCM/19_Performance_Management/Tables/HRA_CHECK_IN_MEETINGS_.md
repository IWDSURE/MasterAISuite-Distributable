# HRA_CHECK_IN_MEETINGS_

It will store UI field values for creating check-in meeting like check-in meeting name, date etc.

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hracheckinmeetings-9501.html#hracheckinmeetings-9501](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hracheckinmeetings-9501.html#hracheckinmeetings-9501)

## Primary Key

| Name | Columns |
|------|----------|
| HRA_CHECK_IN_MEETINGS_PK_ | LAST_UPDATE_DATE, LAST_UPDATED_BY, CHECK_IN_MEETING_ID, BUSINESS_GROUP_ID |
