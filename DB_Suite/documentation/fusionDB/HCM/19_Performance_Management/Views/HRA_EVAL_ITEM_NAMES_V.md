# HRA_EVAL_ITEM_NAMES_V

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalitemnamesv-8462.html#hraevalitemnamesv-8462](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalitemnamesv-8462.html#hraevalitemnamesv-8462)

## Columns

- EVAL_ITEM_ID
- REFERENCE_ITEM_NAME

## Query

```sql
SELECT HRA_EVAL_ITEMS.EVAL_ITEM_ID, CASE WHEN HRA_EVAL_SECTIONS.SECTION_TYPE_CODE IN ('GOAL', 'ORA_DEVGOAL') THEN HRG_GOALS.GOAL_NAME WHEN HRA_EVAL_SECTIONS.SECTION_TYPE_CODE IN ('REG', 'ORA_DYNAMIC_SKILLS') THEN HRT_CONTENT_ITEMS_VL.NAME END AS REFERENCE_ITEM_NAME FROM HRA_EVAL_ITEMS JOIN HRA_EVAL_SECTIONS ON HRA_EVAL_SECTIONS.EVALUATION_ID = HRA_EVAL_ITEMS.EVALUATION_ID AND HRA_EVAL_SECTIONS.EVAL_SECTION_ID = HRA_EVAL_ITEMS.EVAL_SECTION_ID LEFT JOIN HRG_GOALS ON HRG_GOALS.GOAL_ID = HRA_EVAL_ITEMS.REFERENCE_ITEM_ID AND HRA_EVAL_SECTIONS.SECTION_TYPE_CODE IN ('GOAL', 'ORA_DEVGOAL') LEFT JOIN HRT_CONTENT_ITEMS_VL ON HRT_CONTENT_ITEMS_VL.CONTENT_ITEM_ID = HRA_EVAL_ITEMS.REFERENCE_ITEM_ID AND HRT_CONTENT_ITEMS_VL.BUSINESS_GROUP_ID = HRA_EVAL_ITEMS.BUSINESS_GROUP_ID AND HRA_EVAL_SECTIONS.SECTION_TYPE_CODE IN ('REG', 'ORA_DYNAMIC_SKILLS') WHERE HRA_EVAL_SECTIONS.SECTION_TYPE_CODE IN ('GOAL', 'ORA_DEVGOAL', 'REG', 'ORA_DYNAMIC_SKILLS')
```
