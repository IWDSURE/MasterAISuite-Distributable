# HRA_ROLE_DEFNS_VL

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraroledefnsvl-6549.html#hraroledefnsvl-6549](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraroledefnsvl-6549.html#hraroledefnsvl-6549)

## Columns

- ROLE_ID
- BUSINESS_GROUP_ID
- MODULE_ID
- ROLE_TYPE_CODE
- NAME
- DESCRIPTION
- DATE_FROM
- DATE_TO
- STATUS_CODE
- MATRIX_PARTICIPANT_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- MANAGER_TYPE

## Query

```sql
SELECT RD.ROLE_ID, RD.BUSINESS_GROUP_ID, RD.MODULE_ID, RD.ROLE_TYPE_CODE, RDTL.NAME, RDTL.DESCRIPTION, RD.DATE_FROM, RD.DATE_TO, RD.STATUS_CODE, RD.MATRIX_PARTICIPANT_FLAG, RD.OBJECT_VERSION_NUMBER, RD.CREATED_BY, RD.CREATION_DATE, RD.LAST_UPDATE_DATE, RD.LAST_UPDATED_BY, RD.LAST_UPDATE_LOGIN, RD.MANAGER_TYPE FROM HRA_ROLE_DEFNS_B RD, HRA_ROLE_DEFNS_TL RDTL WHERE RD.ROLE_ID = RDTL.ROLE_ID AND RD.BUSINESS_GROUP_ID = RDTL.BUSINESS_GROUP_ID AND RDTL.LANGUAGE = USERENV('LANG')
```
