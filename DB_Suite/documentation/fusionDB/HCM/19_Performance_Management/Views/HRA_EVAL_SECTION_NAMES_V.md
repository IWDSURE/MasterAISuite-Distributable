# HRA_EVAL_SECTION_NAMES_V

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalsectionnamesv-8898.html#hraevalsectionnamesv-8898](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalsectionnamesv-8898.html#hraevalsectionnamesv-8898)

## Columns

- EVAL_SECTION_ID
- SECTION_NAME

## Query

```sql
SELECT HRA_EVAL_SECTIONS.EVAL_SECTION_ID, COALESCE(HRG_GOAL_PLANS_VL.GOAL_PLAN_NAME, HRA_SECTION_DEFNS_VL.NAME) AS SECTION_NAME FROM HRA_EVAL_SECTIONS LEFT JOIN HRA_TMPL_SECTIONS ON HRA_EVAL_SECTIONS.TMPL_SECTION_ID = HRA_TMPL_SECTIONS.SECTION_ID AND HRA_EVAL_SECTIONS.BUSINESS_GROUP_ID = HRA_TMPL_SECTIONS.BUSINESS_GROUP_ID LEFT JOIN HRA_SECTION_DEFNS_VL ON HRA_TMPL_SECTIONS.SECTION_DEF_ID = HRA_SECTION_DEFNS_VL.SECTION_DEF_ID AND HRA_EVAL_SECTIONS.BUSINESS_GROUP_ID = HRA_SECTION_DEFNS_VL.BUSINESS_GROUP_ID LEFT JOIN HRG_GOAL_PLN_ASSIGNMENTS ON HRA_EVAL_SECTIONS.GOAL_PLAN_ASSIGNMENT_ID = HRG_GOAL_PLN_ASSIGNMENTS.GOAL_PLAN_ASSIGNMENT_ID AND HRA_EVAL_SECTIONS.BUSINESS_GROUP_ID = HRG_GOAL_PLN_ASSIGNMENTS.BUSINESS_GROUP_ID LEFT JOIN HRG_GOAL_PLANS_VL ON HRG_GOAL_PLN_ASSIGNMENTS.GOAL_PLAN_ID = HRG_GOAL_PLANS_VL.GOAL_PLAN_ID AND HRA_EVAL_SECTIONS.BUSINESS_GROUP_ID = HRG_GOAL_PLANS_VL.BUSINESS_GROUP_ID WHERE (HRA_EVAL_SECTIONS.GOAL_PLAN_ASSIGNMENT_ID IS NULL AND HRA_SECTION_DEFNS_VL.SECTION_DEF_ID IS NOT NULL) OR (HRA_EVAL_SECTIONS.GOAL_PLAN_ASSIGNMENT_ID IS NOT NULL AND HRG_GOAL_PLANS_VL.GOAL_PLAN_ID IS NOT NULL)
```
