# HRA_PF_TASK_ROLES_VL

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrapftaskrolesvl-4031.html#hrapftaskrolesvl-4031](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrapftaskrolesvl-4031.html#hrapftaskrolesvl-4031)

## Columns

- PROCESS_TASK_ROLE_ID
- BUSINESS_GROUP_ID
- PROCESS_FLOW_ID
- TASK_CODE
- MGR_TASK_NAME
- WKR_TASK_NAME
- SEQUENCE_NUMBER
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- MODULE_ID

## Query

```sql
SELECT PTRB.PROCESS_TASK_ROLE_ID, PTRB.BUSINESS_GROUP_ID, PTRB.PROCESS_FLOW_ID, PTRB.TASK_CODE, PTRL.MGR_TASK_NAME, PTRL.WKR_TASK_NAME, PTRB.SEQUENCE_NUMBER, PTRB.OBJECT_VERSION_NUMBER, PTRB.CREATED_BY, PTRB.CREATION_DATE, PTRB.LAST_UPDATED_BY, PTRB.LAST_UPDATE_DATE, PTRB.LAST_UPDATE_LOGIN, PTRB.MODULE_ID FROM HRA_PF_TASK_ROLES_B PTRB, HRA_PF_TASK_ROLES_TL PTRL WHERE PTRB.PROCESS_TASK_ROLE_ID = PTRL.PROCESS_TASK_ROLE_ID AND PTRB.BUSINESS_GROUP_ID = PTRL.BUSINESS_GROUP_ID AND PTRL.LANGUAGE = USERENV('LANG')
```
