# HRA_EVAL_SECTION_ITEMS_V

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalsectionitemsv-4932.html#hraevalsectionitemsv-4932](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalsectionitemsv-4932.html#hraevalsectionitemsv-4932)

- EVAL_ITEM_ID
- ITEM_TYPE
- BUSINESS_GROUP_ID
- EVALUATION_ID
- EVAL_SECTION_ID
- REFERENCE_ITEM_ID
- ITEM_NAME
- DESCRIPTION
- MEASUREMENT
- TARGET_PROF_REV_RATING_ID
- TARGET_PERF_REV_RATING_ID
- PERF_RATING_MODEL_ID
- PROF_RATING_MODEL_ID
- TARGET_DATE
- START_DATE
- MIN_WEIGHT
- WEIGHT

- MANDATORY_FLAG
- ADDED_IN_TASK
- NOT_RATED_FLAG
- NOT_RATED_FLAG_UPDATED_BY

## Query

```sql
SELECT EVAL_ITEM_ID, 'ITEM' ITEM_TYPE, BUSINESS_GROUP_ID, EVALUATION_ID, EVAL_SECTION_ID, REFERENCE_ITEM_ID, ITEM_NAME, DESCRIPTION, MEASUREMENT, TARGET_PROF_REV_RATING_ID, TARGET_PERF_REV_RATING_ID, PERF_RATING_MODEL_ID, PROF_RATING_MODEL_ID, TARGET_DATE, START_DATE, MIN_WEIGHT, WEIGHT, OWNED_BY, MANDATORY_FLAG, ADDED_IN_TASK, NOT_RATED_FLAG, NOT_RATED_FLAG_UPDATED_BY FROM HRA_EVAL_ITEMS UNION ALL SELECT EVAL_SECTION_ID EVAL_ITEM_ID, 'SECTION' ITEM_TYPE, BUSINESS_GROUP_ID, EVALUATION_ID, EVAL_SECTION_ID, NULL, NULL, NULL, NULL, NULL, NULL, SECTION_RATING_MODEL_ID PERF_RATING_MODEL_ID, NULL, NULL, NULL, NULL, NULL, NULL, NULL, NULL, NULL, NULL FROM HRA_EVAL_SECTIONS WHERE SECTION_TYPE_CODE IN ( 'REG', 'GOAL', 'ORA_DEVGOAL', 'ORA_DYNAMIC_SKILLS', 'OS', 'QUESTIONNAIRE' )
```
