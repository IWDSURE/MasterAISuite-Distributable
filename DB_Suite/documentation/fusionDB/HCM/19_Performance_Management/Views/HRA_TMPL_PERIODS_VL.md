# HRA_TMPL_PERIODS_VL

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hratmplperiodsvl-4440.html#hratmplperiodsvl-4440](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hratmplperiodsvl-4440.html#hratmplperiodsvl-4440)

## Columns

- TMPL_PERIOD_ID
- BUSINESS_GROUP_ID
- TEMPLATE_DEFN_ID
- CUSTOMARY_NAME
- SHORT_NAME
- NOMINAL_FROM_DATE
- NOMINAL_TO_DATE
- AVAILABLE_TO_USE
- REVIEW_PERIOD_ID
- LOCK_MGR_SHARE_FOR_CALIB_FLAG
- GOAL_ALGORITHM_TYPE_CODE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- MODULE_ID

## Query

```sql
SELECT TP.TMPL_PERIOD_ID, TP.BUSINESS_GROUP_ID, TP.TEMPLATE_DEFN_ID, TPTL.CUSTOMARY_NAME, TPTL.SHORT_NAME, TP.NOMINAL_FROM_DATE, TP.NOMINAL_TO_DATE, TP.AVAILABLE_TO_USE, TP.REVIEW_PERIOD_ID, TP.LOCK_MGR_SHARE_FOR_CALIB_FLAG, TP.GOAL_ALGORITHM_TYPE_CODE , TP.OBJECT_VERSION_NUMBER, TP.CREATED_BY, TP.CREATION_DATE, TP.LAST_UPDATE_DATE, TP.LAST_UPDATED_BY, TP.LAST_UPDATE_LOGIN, TP.MODULE_ID FROM HRA_TMPL_PERIODS_B TP, HRA_TMPL_PERIODS_TL TPTL WHERE TP.TMPL_PERIOD_ID = TPTL.TMPL_PERIOD_ID AND TP.BUSINESS_GROUP_ID = TPTL.BUSINESS_GROUP_ID AND TPTL.LANGUAGE = USERENV('LANG')
```
