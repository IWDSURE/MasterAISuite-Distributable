# HRA_SECTION_ITEMS_VL

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrasectionitemsvl-3799.html#hrasectionitemsvl-3799](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrasectionitemsvl-3799.html#hrasectionitemsvl-3799)

- CONTENT_ITEM_ID

- DESCRIPTION
- MEASUREMENT
- TARGET_PROF_REV_RATING_ID

- PERF_RATING_MODEL_ID

- DUE_DATE

- CRITICAL_FLAG

- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- MODULE_ID
- WEIGHT
- MIN_WEIGHT

## Query

```sql
SELECT SI.ITEM_ID, SI.BUSINESS_GROUP_ID, SI.SECTION_DEF_ID, SI.PARENT_ITEM_ID, SI.CONTENT_ITEM_ID, SITL.NAME, SITL.DESCRIPTION, SITL.MEASUREMENT, SI.TARGET_PROF_REV_RATING_ID, SI.TARGET_PERF_REV_RATING_ID, SI.PERF_RATING_MODEL_ID, SI.PROF_RATING_MODEL_ID, SI.DUE_DATE, SI.REMINDER_DATE, SI.OWNED_BY, SI.MANDATORY_FLAG, SI.CRITICAL_FLAG, SI.OBJECT_VERSION_NUMBER, SI.CREATED_BY, SI.CREATION_DATE, SI.LAST_UPDATE_DATE, SI.LAST_UPDATED_BY, SI.LAST_UPDATE_LOGIN, SI.MODULE_ID, SI.WEIGHT, SI.MIN_WEIGHT FROM HRA_SECTION_ITEMS_B SI, HRA_SECTION_ITEMS_TL SITL WHERE SI.ITEM_ID = SITL.ITEM_ID AND SI.BUSINESS_GROUP_ID = SITL.BUSINESS_GROUP_ID AND SITL.LANGUAGE = USERENV('LANG')
```
