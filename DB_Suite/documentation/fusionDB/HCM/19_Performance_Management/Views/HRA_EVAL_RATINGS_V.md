# HRA_EVAL_RATINGS_V

## Details

**Schema:** FUSION

**Object owner:** HRA

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalratingsv-7879.html#hraevalratingsv-7879](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hraevalratingsv-7879.html#hraevalratingsv-7879)

## Columns

- EVAL_RATING_ID
- BUSINESS_GROUP_ID
- EVALUATION_ID
- REFERENCE_ID
- REFERENCE_TYPE
- PERF_RATING_MODEL_ID
- PERFORMANCE_RATING_ID
- PROF_RATING_MODEL_ID
- PROFICIENCY_RATING_ID
- EVAL_PARTICIPANT_ID
- REVIEW_POINTS
- CALCULATED_RATING
- ROLE_TYPE_CODE
- COMMENT_TEXT
- JUSTIFICATION_TEXT
- SECTION_TYPE_CODE
- EVAL_SECTION_ID

## Query

```sql
SELECT HRA_EVAL_RATINGS.EVAL_RATING_ID, HRA_EVAL_RATINGS.BUSINESS_GROUP_ID, HRA_EVAL_RATINGS.EVALUATION_ID, HRA_EVAL_RATINGS.REFERENCE_ID, HRA_EVAL_RATINGS.REFERENCE_TYPE, HRA_EVAL_SECTION_ITEMS_V.PERF_RATING_MODEL_ID, HRA_EVAL_RATINGS.PERFORMANCE_RATING_ID, HRA_EVAL_SECTION_ITEMS_V.PROF_RATING_MODEL_ID, HRA_EVAL_RATINGS.PROFICIENCY_RATING_ID, HRA_EVAL_RATINGS.EVAL_PARTICIPANT_ID, HRA_EVAL_RATINGS.REVIEW_POINTS, HRA_EVAL_RATINGS.CALCULATED_RATING, HRA_EVAL_RATINGS.ROLE_TYPE_CODE, HRA_EVAL_RATINGS.COMMENT_TEXT, HRA_EVAL_RATINGS.JUSTIFICATION_TEXT, HRA_EVAL_SECTIONS.SECTION_TYPE_CODE, HRA_EVAL_SECTIONS.EVAL_SECTION_ID FROM HRA_EVAL_RATINGS JOIN HRA_EVAL_SECTION_ITEMS_V ON HRA_EVAL_SECTION_ITEMS_V.EVALUATION_ID = HRA_EVAL_RATINGS.EVALUATION_ID AND HRA_EVAL_SECTION_ITEMS_V.EVAL_ITEM_ID = HRA_EVAL_RATINGS.REFERENCE_ID JOIN HRA_EVAL_SECTIONS ON HRA_EVAL_SECTION_ITEMS_V.EVALUATION_ID = HRA_EVAL_SECTIONS.EVALUATION_ID AND HRA_EVAL_SECTION_ITEMS_V.EVAL_SECTION_ID = HRA_EVAL_SECTIONS.EVAL_SECTION_ID
```
