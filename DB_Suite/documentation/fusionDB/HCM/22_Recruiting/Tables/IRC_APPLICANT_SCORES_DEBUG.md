# IRC_APPLICANT_SCORES_DEBUG

This table captures the request and response payload of gen ai service for a candidate who is an applicant of a requisition.

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircapplicantscoresdebug-24790.html#ircapplicantscoresdebug-24790](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircapplicantscoresdebug-24790.html#ircapplicantscoresdebug-24790)

## Primary Key

| Name | Columns |
|------|----------|
| IRC_APPLICANT_SCORES_DEBUG_PK | APPLICANT_SCORE_DEBUG_ID |
