# IRC_TN_SUBSCRIBERS

This is the table for storing subscriber information on Talent Network.

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irctnsubscribers-8821.html#irctnsubscribers-8821](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irctnsubscribers-8821.html#irctnsubscribers-8821)

## Primary Key

| Name | Columns |
|------|----------|
| IRC_TN_SUBSCRIBERS_PK | SUBSCRIBER_ID |

## Columns

| Name | Datatype | Length | Precision | Not-null | Comments |
|---|---|---|---|---|---|
| SUBSCRIBER_ID | NUMBER |  | 18 | Yes | Primary Key of the table. System generated. |
| STATUS_CODE | VARCHAR2 | 30 |  | Yes | Used to store the status of the subscriber. |
| CODE | VARCHAR2 | 30 |  | Yes | Unique key of the table and generated by sequence IRC_TN_SUBSCRIBER_CODE_S. |
| NAME | VARCHAR2 | 60 |  |  | Used To store name of the customer |
| LOGO_URL | VARCHAR2 | 1000 |  |  | Used to store the customer logo URL. |
| SUBSCRIBER_GUID | VARCHAR2 | 64 |  |  | System generated and generated by SCIM apis. |
| OBJECT_VERSION_NUMBER | NUMBER |  | 9 | Yes | Used to implement optimistic locking. This number is incremented every time that the row is updated. The number is compared at the start and end of a transaction to detect whether another session has updated the row since it was queried. |
| CREATED_BY | VARCHAR2 | 64 |  | Yes | Who column: indicates the user who created the row. |
| CREATION_DATE | TIMESTAMP |  |  | Yes | Who column: indicates the date and time of the creation of the row. |
| LAST_UPDATED_BY | VARCHAR2 | 64 |  | Yes | Who column: indicates the user who last updated the row. |
| LAST_UPDATE_DATE | TIMESTAMP |  |  | Yes | Who column: indicates the date and time of the last update of the row. |
| LAST_UPDATE_LOGIN | VARCHAR2 | 32 |  |  | Who column: indicates the session login associated to the user who last updated the row. |

## Indexes

| Index | Uniqueness | Tablespace | Columns |
|---|---|---|---|
| IRC_TN_SUBSCRIBERS_PK | Unique | Default | SUBSCRIBER_ID |
| IRC_TN_SUBSCRIBERS_U1 | Unique | Default | CODE |
