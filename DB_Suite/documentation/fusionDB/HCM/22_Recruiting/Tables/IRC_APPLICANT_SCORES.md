# IRC_APPLICANT_SCORES

This table captures the different scores calculated by the AI engine for a candidate who is an applicant of a requisition.

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircapplicantscores-31696.html#ircapplicantscores-31696](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircapplicantscores-31696.html#ircapplicantscores-31696)

## Primary Key

| Name | Columns |
|------|----------|
| IRC_APPLICANT_SCORES_PK | APPLICANT_SCORE_ID |

## Columns

| Name | Datatype | Length | Precision | Not-null | Comments |
|---|---|---|---|---|---|
| APPLICANT_SCORE_ID | NUMBER |  | 18 | Yes | A unique key generated by system for each row that acts as a Primary Key for the table |
| ENTITY_TYPE | VARCHAR2 | 30 |  |  | Stores the context ORA_SUBMISSION or ORA_PROSPECT. |
| ENTITY_ID | NUMBER |  | 18 |  | Candidate SubmissionId or ProspectId. |

| PROFILE_AI_SCORE | NUMBER |  | 3 |  | Matching profile score of the candidate for the given requisition. |
