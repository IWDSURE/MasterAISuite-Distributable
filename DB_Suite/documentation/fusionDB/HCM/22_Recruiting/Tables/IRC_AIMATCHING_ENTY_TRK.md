# IRC_AIMATCHING_ENTY_TRK

This table is to track the injested candidate along with status of passed for successfully injested candidate and status of failed for non injested candidate while incremental injestion is being ran.

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** TABLE

**Tablespace:** Default

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircaimatchingentytrk-15050.html#ircaimatchingentytrk-15050](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircaimatchingentytrk-15050.html#ircaimatchingentytrk-15050)

## Primary Key

| Name | Columns |
|------|----------|
| IRC_AIMATCHING_ENTY_TRK_PK | TRACKING_ID |

## Columns
