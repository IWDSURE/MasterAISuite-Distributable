## Query

```sql
SELECT IRC_IS_PROPERTIES_B.PROPERTY_ID PROPERTY_ID, IRC_IS_PROPERTIES_B.OBJECT_TYPE OBJECT_TYPE, IRC_IS_PROPERTIES_B.OBJECT_ID OBJECT_ID, IRC_IS_PROPERTIES_B.PROPERTY_CODE PROPERTY_CODE, IRC_IS_PROPERTIES_B.PROPERTY_NAME PROPERTY_NAME, IRC_IS_PROPERTIES_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, IRC_IS_PROPERTIES_B.CREATED_BY CREATED_BY, IRC_IS_PROPERTIES_B.CREATION_DATE CREATION_DATE, IRC_IS_PROPERTIES_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_IS_PROPERTIES_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_IS_PROPERTIES_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, IRC_IS_PROPERTIES_TL.PROPERTY_VALUE PROPERTY_VALUE FROM IRC_IS_PROPERTIES_B, IRC_IS_PROPERTIES_TL WHERE IRC_IS_PROPERTIES_B.PROPERTY_ID = IRC_IS_PROPERTIES_TL.PROPERTY_ID AND IRC_IS_PROPERTIES_TL.LANGUAGE = USERENV('LANG')
```
