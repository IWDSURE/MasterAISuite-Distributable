# IRC_LC_REASONS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclcreasonsvl-5100.html#irclcreasonsvl-5100](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclcreasonsvl-5100.html#irclcreasonsvl-5100)

## Columns

- REASON_ID
- REASON_CODE
- REASON_STATUS_CODE
- REASON_NAME
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- SEED_DATA_SOURCE

## Query

```sql
SELECT IRC_LC_REASONS_B.REASON_ID REASON_ID, IRC_LC_REASONS_B.REASON_CODE REASON_CODE, IRC_LC_REASONS_B.REASON_STATUS_CODE REASON_STATUS_CODE, IRC_LC_REASONS_TL.REASON_NAME REASON_NAME, IRC_LC_REASONS_B.MODULE_ID MODULE_ID, IRC_LC_REASONS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, IRC_LC_REASONS_B.CREATED_BY CREATED_BY, IRC_LC_REASONS_B.CREATION_DATE CREATION_DATE, IRC_LC_REASONS_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_LC_REASONS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_LC_REASONS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, IRC_LC_REASONS_B.SEED_DATA_SOURCE SEED_DATA_SOURCE FROM IRC_LC_REASONS_B, IRC_LC_REASONS_TL WHERE IRC_LC_REASONS_B.REASON_ID = IRC_LC_REASONS_TL.REASON_ID AND IRC_LC_REASONS_TL.LANGUAGE = USERENV('LANG')
```
