# IRC_GEO_NODE_ACTIVE_LATEST_V

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgeonodeactivelatestv-8802.html#ircgeonodeactivelatestv-8802](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgeonodeactivelatestv-8802.html#ircgeonodeactivelatestv-8802)

## Columns

- GEOGRAPHY_NODE_ID
- GEOGRAPHY_ID
- HIERARCHY_ID
- PARENT_GEO_NODE_ID

## Query

```sql
SELECT GHN.GEOGRAPHY_NODE_ID AS GEOGRAPHY_NODE_ID, GHN.GEOGRAPHY_ID as GEOGRAPHY_ID, GHN.HIERARCHY_ID, GHN.PARENT_GEO_NODE_ID FROM IRC_GEO_HIER_NODES GHN WHERE GHN.GEOGRAPHY_NODE_ID IN (SELECT GEOGRAPHY_NODE_ID FROM IRC_GEO_HIER_HIRING_V) UNION ALL SELECT GHN.GEOGRAPHY_NODE_ID AS GEOGRAPHY_NODE_ID, GHN.GEOGRAPHY_ID as GEOGRAPHY_ID, GHN.HIERARCHY_ID, GHN.PARENT_GEO_NODE_ID FROM IRC_GEO_HIER_NODES GHN WHERE GHN.HIERARCHY_ID = (SELECT MAX(GHN1.HIERARCHY_ID) FROM IRC_GEO_HIER_NODES GHN1 WHERE GHN1.GEOGRAPHY_ID = GHN.GEOGRAPHY_ID) AND GHN.GEOGRAPHY_ID NOT IN (SELECT GEOGRAPHY_ID FROM IRC_GEO_HIER_HIRING_V)
```
