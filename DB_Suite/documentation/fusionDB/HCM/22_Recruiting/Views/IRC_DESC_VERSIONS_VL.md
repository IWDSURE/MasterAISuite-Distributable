# IRC_DESC_VERSIONS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdescversionsvl-8516.html#ircdescversionsvl-8516](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdescversionsvl-8516.html#ircdescversionsvl-8516)

## Columns

- DESC_VERSION_ID
- DESCRIPTION_ID
- VERSION_START_DATE
- STATUS_CODE
- WA_TEMPLATE_ID
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_ID
- OBJECT_TYPE
- DESCRIPTION
- SHORT_DESCRIPTION
- TXT_DESCRIPTION
- QUALIFICATIONS
- RESPONSIBILITIES

## Query

```sql
SELECT B.DESC_VERSION_ID DESC_VERSION_ID, B.DESCRIPTION_ID DESCRIPTION_ID, B.VERSION_START_DATE VERSION_START_DATE, B.STATUS_CODE STATUS_CODE, B.WA_TEMPLATE_ID, B.MODULE_ID MODULE_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.OBJECT_ID OBJECT_ID, B.OBJECT_TYPE OBJECT_TYPE, TL.DESCRIPTION DESCRIPTION, TL.SHORT_DESCRIPTION SHORT_DESCRIPTION, TL.TXT_DESCRIPTION TXT_DESCRIPTION, TL.QUALIFICATIONS QUALIFICATIONS, TL.RESPONSIBILITIES RESPONSIBILITIES FROM IRC_DESC_VERSIONS_B B, IRC_DESC_VERSIONS_TL TL WHERE TL.DESC_VERSION_ID (+)= B.DESC_VERSION_ID AND TL.LANGUAGE (+)= USERENV('LANG')
```
