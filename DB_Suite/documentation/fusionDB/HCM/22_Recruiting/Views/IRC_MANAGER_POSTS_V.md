# IRC_MANAGER_POSTS_V

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmanagerpostsv-5349.html#ircmanagerpostsv-5349](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmanagerpostsv-5349.html#ircmanagerpostsv-5349)

## Columns

- ACTIVATED_ON_DATE
- CAMPAIGN_CODE
- CAMPAIGN_DURATION_CODE
- CAMPAIGN_ID
- CAMPAIGN_LANGUAGE_CODE
- CAMPAIGN_STATUS_CODE
- CAMPAIGN_TYPE_CODE
- CANCELLED_ON_DATE
- CLOSED_ON_DATE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT EMPC.ACTIVATED_ON_DATE, EMPC.CAMPAIGN_CODE, EMPC.CAMPAIGN_DURATION_CODE, EMPC.CAMPAIGN_ID, EMPC.CAMPAIGN_LANGUAGE_CODE, EMPC.CAMPAIGN_STATUS_CODE, EMPC.CAMPAIGN_TYPE_CODE, EMPC.CANCELLED_ON_DATE, EMPC.CLOSED_ON_DATE, EMPC.CREATED_BY, EMPC.CREATION_DATE, EMPC.LAST_UPDATED_BY, EMPC.LAST_UPDATE_DATE, EMPC.LAST_UPDATE_LOGIN, EMPC.OBJECT_VERSION_NUMBER FROM IRC_CAMPAIGNS_B EMPC WHERE CAMPAIGN_TYPE_CODE = 'ORA_MANAGER_POST'
```
