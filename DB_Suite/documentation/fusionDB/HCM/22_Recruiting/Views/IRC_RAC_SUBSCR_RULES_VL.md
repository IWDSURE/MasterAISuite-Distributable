## Query

```sql
SELECT RULEB.SUBSCRIBER_RULE_ID SUBSCRIBER_RULE_ID, RULEB.SUBSCRIBER_ID SUBSCRIBER_ID, RULEB.SUBSCRIBER_RULE_CODE SUBSCRIBER_RULE_CODE, RULEB.RULE_PRIORITY_CODE RULE_PRIORITY_CODE, RULEB.ENABLED_FLAG ENABLED_FLAG, RULEB.ESS_RULE_FLAG ESS_RULE_FLAG, RULEB.MESSAGE MESSAGE, RULEB.APPLY_AFTER_AGE APPLY_AFTER_AGE, RULETL.NAME NAME, RULEB.MODULE_ID MODULE_ID, RULEB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, RULEB.CREATED_BY CREATED_BY, RULEB.CREATION_DATE CREATION_DATE, RULEB.LAST_UPDATED_BY LAST_UPDATED_BY, RULEB.LAST_UPDATE_DATE LAST_UPDATE_DATE, RULEB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_RAC_SUBSCR_RULES_B RULEB, IRC_RAC_SUBSCR_RULES_TL RULETL WHERE RULEB.SUBSCRIBER_RULE_ID = RULETL.SUBSCRIBER_RULE_ID AND RULETL.LANGUAGE = USERENV('LANG')
```
