# IRC_AF_BLOCKS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircafblocksvl-6739.html#ircafblocksvl-6739](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircafblocksvl-6739.html#ircafblocksvl-6739)

## Columns

- BLOCK_ID
- BLOCK_CODE
- BLOCK_USAGE_CODE
- BLOCK_TYPE_CODE
- PROFILE_CONTENT_TYPE_ID
- BLOCK_NAME
- BLOCK_TITLE
- BLOCK_INSTRUCTIONS
- REUSABLE_FLAG
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.BLOCK_ID BLOCK_ID, B.BLOCK_CODE BLOCK_CODE, B.BLOCK_USAGE_CODE BLOCK_USAGE_CODE, B.BLOCK_TYPE_CODE BLOCK_TYPE_CODE, B.PROFILE_CONTENT_TYPE_ID PROFILE_CONTENT_TYPE_ID, TL.BLOCK_NAME BLOCK_NAME, TL.BLOCK_TITLE BLOCK_TITLE, TL.BLOCK_INSTRUCTIONS BLOCK_INSTRUCTIONS, B.REUSABLE_FLAG REUSABLE_FLAG, B.MODULE_ID MODULE_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_AF_BLOCKS_B B, IRC_AF_BLOCKS_TL TL WHERE B.BLOCK_ID = TL.BLOCK_ID AND TL.LANGUAGE = USERENV('LANG')
```
