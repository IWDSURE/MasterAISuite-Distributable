# IRC_BOT_QUESTIONS_CONFIG_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircbotquestionsconfigvl-7186.html#ircbotquestionsconfigvl-7186](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircbotquestionsconfigvl-7186.html#ircbotquestionsconfigvl-7186)

## Columns

- BOT_QUESTION_ID
- QUESTION_FLOW_CODE
- QUESTION_CATEGORY
- BOT_INTENT_CODE
- FIELD_NAME
- SEQUENCE_NUMBER
- IS_ENABLED_FLAG
- QUESTION
- ANSWER
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT BOT_QS_CONFIG_B.BOT_QUESTION_ID BOT_QUESTION_ID, BOT_QS_CONFIG_B.QUESTION_FLOW_CODE QUESTION_FLOW_CODE, BOT_QS_CONFIG_B.QUESTION_CATEGORY QUESTION_CATEGORY, BOT_QS_CONFIG_B.BOT_INTENT_CODE BOT_INTENT_CODE, BOT_QS_CONFIG_B.FIELD_NAME FIELD_NAME, BOT_QS_CONFIG_B.SEQUENCE_NUMBER SEQUENCE_NUMBER, BOT_QS_CONFIG_B.IS_ENABLED_FLAG IS_ENABLED_FLAG, BOT_QS_CONFIG_TL.QUESTION QUESTION, BOT_QS_CONFIG_TL.ANSWER ANSWER, BOT_QS_CONFIG_B.MODULE_ID MODULE_ID, BOT_QS_CONFIG_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, BOT_QS_CONFIG_B.CREATED_BY CREATED_BY, BOT_QS_CONFIG_B.CREATION_DATE CREATION_DATE, BOT_QS_CONFIG_B.LAST_UPDATED_BY LAST_UPDATED_BY, BOT_QS_CONFIG_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, BOT_QS_CONFIG_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_BOT_QUESTIONS_CONFIG_B BOT_QS_CONFIG_B, IRC_BOT_QUESTIONS_CONFIG_TL BOT_QS_CONFIG_TL WHERE BOT_QS_CONFIG_B.BOT_QUESTION_ID = BOT_QS_CONFIG_TL.BOT_QUESTION_ID AND BOT_QS_CONFIG_TL.LANGUAGE = USERENV('LANG')
```
