## Query

```sql
SELECT B.GEOGRAPHY_ID, (CASE WHEN B.CITY_ID IS NOT NULL THEN 'CITY' WHEN B.STATE_ID IS NOT NULL THEN 'STATE' ELSE 'COUNTRY' END) GEOGRAPHY_TYPE, B.GEOGRAPHY_NAME, B.COUNTRY_ID, B.COUNTRY, B.STATE_ID, B.STATE, B.CITY_ID, B.CITY, B.COUNTY_ID, B.COUNTY, TL.NAME, TL_US.FQN FQN_ID, TL.FQN, TL_US.FQN_COUNTRY FQN_COUNTRY_ID, TL.FQN_COUNTRY, TL_US.FQN_STATE FQN_STATE_ID, TL.FQN_STATE, TL_US.FQN_CITY FQN_CITY_ID, TL.FQN_CITY, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM irc_cand_geographies_b B, irc_cand_geographies_tl TL, IRC_CAND_GEOGRAPHIES_TL TL_US WHERE B.GEOGRAPHY_ID = TL.GEOGRAPHY_ID AND B.GEOGRAPHY_ID = TL_US.GEOGRAPHY_ID AND TL_US.LANGUAGE = 'US' AND TL.LANGUAGE = USERENV('LANG')
```
