# IRC_CAMP_POST_CONTENT_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccamppostcontentvl-7972.html#irccamppostcontentvl-7972](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccamppostcontentvl-7972.html#irccamppostcontentvl-7972)

## Columns

- POST_CONTENT_ID
- ASSET_ID
- PRIMARY_LINK_TYPE_CODE
- PRIMARY_ASSET_URL_ID
- SECONDARY_LINK_TYPE_CODE
- SECONDARY_ASSET_URL_ID
- IMAGE_URL
- TITLE
- DESCRIPTION
- PRIMARY_LABEL
- SECONDARY_LABEL
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.POST_CONTENT_ID, B.ASSET_ID, B.PRIMARY_LINK_TYPE_CODE, B.PRIMARY_ASSET_URL_ID, B.SECONDARY_LINK_TYPE_CODE, B.SECONDARY_ASSET_URL_ID, B.IMAGE_URL, TL.TITLE, TL.DESCRIPTION, TL.PRIMARY_LABEL, TL.SECONDARY_LABEL, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM IRC_CAMP_POST_CONTENT_B B, IRC_CAMP_POST_CONTENT_TL TL WHERE B.POST_CONTENT_ID = TL.POST_CONTENT_ID AND TL.LANGUAGE = USERENV('LANG')
```
