# IRC_CX_HEADERS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccxheadersvl-6066.html#irccxheadersvl-6066](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccxheadersvl-6066.html#irccxheadersvl-6066)

## Columns

- HEADER_ID
- THEME_ID
- TYPE
- NAME
- LOGO_URL
- PROFILE_TEXT
- EMPLOYEE_TEXT
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE

## Query

```sql
SELECT IRC_CX_HEADERS_B.HEADER_ID, IRC_CX_HEADERS_B.THEME_ID, IRC_CX_HEADERS_B.TYPE, IRC_CX_HEADERS_B.NAME, IRC_CX_HEADERS_TL.LOGO_URL, IRC_CX_HEADERS_TL.PROFILE_TEXT, IRC_CX_HEADERS_TL.EMPLOYEE_TEXT, IRC_CX_HEADERS_B.OBJECT_VERSION_NUMBER, IRC_CX_HEADERS_B.CREATED_BY, IRC_CX_HEADERS_B.CREATION_DATE, IRC_CX_HEADERS_B.LAST_UPDATED_BY, IRC_CX_HEADERS_B.LAST_UPDATE_LOGIN, IRC_CX_HEADERS_B.LAST_UPDATE_DATE FROM IRC_CX_HEADERS_B, IRC_CX_HEADERS_TL WHERE IRC_CX_HEADERS_B.HEADER_ID = IRC_CX_HEADERS_TL.HEADER_ID AND IRC_CX_HEADERS_TL.LANGUAGE = USERENV('LANG')
```
