# IRC_MEDIA_ASSETS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmediaassetsvl-4098.html#ircmediaassetsvl-4098](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmediaassetsvl-4098.html#ircmediaassetsvl-4098)

## Columns

- MEDIA_ASSET_ID
- MEDIA_ASSET_GUID
- MEDIA_ASSET_NAME
- MEDIA_ASSET_STATUS
- CONTEXT
- LINKED_ASSET_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.MEDIA_ASSET_ID MEDIA_ASSET_ID, B.MEDIA_ASSET_GUID MEDIA_ASSET_GUID, TL.MEDIA_ASSET_NAME MEDIA_ASSET_NAME, B.MEDIA_ASSET_STATUS MEDIA_ASSET_STATUS, B.CONTEXT CONTEXT, B.LINKED_ASSET_ID LINKED_ASSET_ID, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_MEDIA_ASSETS_B B, IRC_MEDIA_ASSETS_TL TL WHERE B.MEDIA_ASSET_ID = TL.MEDIA_ASSET_ID AND TL.LANGUAGE = USERENV('LANG')
```
