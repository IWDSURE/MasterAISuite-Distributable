# IRC_CX_SITE_TEMPLATES_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccxsitetemplatesvl-4162.html#irccxsitetemplatesvl-4162](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irccxsitetemplatesvl-4162.html#irccxsitetemplatesvl-4162)

## Columns

- TEMPLATE_ID
- OBJECT_STATUS
- OBJECT_VERSION_NUMBER
- TEMPLATE_NUMBER
- TEMPLATE_NAME
- MAIN_CALL_FOR_ACTION
- WELCOME_TEXT
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- SPLASH_PAGE_NUMBER
- SITE_NUMBER
- IS_ACTIVE_FLAG

## Query

```sql
SELECT B.TEMPLATE_ID TEMPLATE_ID, B.OBJECT_STATUS, B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, B.TEMPLATE_NUMBER, TL.TEMPLATE_NAME TEMPLATE_NAME, TL.MAIN_CALL_FOR_ACTION MAIN_CALL_FOR_ACTION, TL.WELCOME_TEXT WELCOME_TEXT, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.SPLASH_PAGE_NUMBER, B.SITE_NUMBER, B.IS_ACTIVE_FLAG FROM IRC_CX_SITE_TEMPLATES_B B, IRC_CX_SITE_TEMPLATES_TL TL WHERE B.TEMPLATE_ID = TL.TEMPLATE_ID AND TL.LANGUAGE = USERENV('LANG')
```
