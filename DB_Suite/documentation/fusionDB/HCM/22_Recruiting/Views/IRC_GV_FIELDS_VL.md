# IRC_GV_FIELDS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvfieldsvl-4353.html#ircgvfieldsvl-4353](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvfieldsvl-4353.html#ircgvfieldsvl-4353)

## Columns

- FIELD_ID
- NAME
- DESCRIPTION
- FIELD_CODE
- CATEGORY_CODE
- FIELD_TYPE_CODE
- VO_ATTRIBUTE_PATH
- OSCS_ATTRIBUTE_PATH
- DATA_TYPE
- TRANSLATABLE_FLAG
- MULTI_ROW_FLAG
- DISPLAY_ROWS
- HIDDEN_FLAG
- CUSTOM_PROPERTIES
- GV_CONTEXT_CODE
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT FIELDS_B.FIELD_ID FIELD_ID, FIELDS_TL.NAME NAME, FIELDS_TL.DESCRIPTION DESCRIPTION, FIELDS_B.FIELD_CODE FIELD_CODE, FIELDS_B.CATEGORY_CODE CATEGORY_CODE, FIELDS_B.FIELD_TYPE_CODE FIELD_TYPE_CODE, FIELDS_B.VO_ATTRIBUTE_PATH VO_ATTRIBUTE_PATH, FIELDS_B.OSCS_ATTRIBUTE_PATH OSCS_ATTRIBUTE_PATH, FIELDS_B.DATA_TYPE DATA_TYPE, FIELDS_B.TRANSLATABLE_FLAG TRANSLATABLE_FLAG, FIELDS_B.MULTI_ROW_FLAG MULTI_ROW_FLAG, FIELDS_B.DISPLAY_ROWS DISPLAY_ROWS, FIELDS_B.HIDDEN_FLAG HIDDEN_FLAG, FIELDS_B.CUSTOM_PROPERTIES CUSTOM_PROPERTIES, FIELDS_B.GV_CONTEXT_CODE GV_CONTEXT_CODE, FIELDS_B.MODULE_ID MODULE_ID, FIELDS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, FIELDS_B.CREATED_BY CREATED_BY, FIELDS_B.CREATION_DATE CREATION_DATE, FIELDS_B.LAST_UPDATED_BY LAST_UPDATED_BY, FIELDS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, FIELDS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_GV_FIELDS_B FIELDS_B, IRC_GV_FIELDS_TL FIELDS_TL WHERE FIELDS_B.FIELD_ID = FIELDS_TL.FIELD_ID AND FIELDS_TL.LANGUAGE = USERENV('LANG')
```
