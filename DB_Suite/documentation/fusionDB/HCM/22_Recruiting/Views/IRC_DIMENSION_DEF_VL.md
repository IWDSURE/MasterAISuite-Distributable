# IRC_DIMENSION_DEF_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdimensiondefvl-3846.html#ircdimensiondefvl-3846](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdimensiondefvl-3846.html#ircdimensiondefvl-3846)

## Columns

- DIMENSION_ID
- MODULE_ID
- SOURCE_URL_VALUE
- SOURCE_MEDIUM_URL_VALUE
- SOURCE_MEDIUM
- URL_HEADER_REGEX
- PRIORITY
- SEQUENCE
- CREATION_DATE
- CREATED_BY
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE
- SOURCE_NAME
- SOURCE_DISPLAY_NAME

## Query

```sql
SELECT B.DIMENSION_ID DIMENSION_ID, B.MODULE_ID MODULE_ID, B.SOURCE_URL_VALUE SOURCE_URL_VALUE, B.SOURCE_MEDIUM_URL_VALUE SOURCE_MEDIUM_URL_VALUE, B.SOURCE_MEDIUM SOURCE_MEDIUM, B.URL_HEADER_REGEX URL_HEADER_REGEX, B.PRIORITY PRIORITY, B.SEQUENCE SEQUENCE, B.CREATION_DATE CREATION_DATE, B.CREATED_BY CREATED_BY, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, TL.SOURCE_NAME SOURCE_NAME, TL.SOURCE_DISPLAY_NAME SOURCE_DISPLAY_NAME FROM IRC_DIMENSION_DEF_B B, IRC_DIMENSION_DEF_TL TL WHERE B.DIMENSION_ID = TL.DIMENSION_ID AND TL.LANGUAGE = USERENV('LANG')
```
