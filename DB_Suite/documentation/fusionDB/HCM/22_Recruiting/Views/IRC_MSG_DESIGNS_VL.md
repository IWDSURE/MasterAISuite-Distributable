# IRC_MSG_DESIGNS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmsgdesignsvl-3640.html#ircmsgdesignsvl-3640](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircmsgdesignsvl-3640.html#ircmsgdesignsvl-3640)

## Columns

- MESSAGE_DESIGN_ID
- MESSAGE_DESIGN_CODE
- DESIGN_TYPE_CODE
- DESIGN_STATUS_CODE
- DESIGN_EDITOR_TYPE_CODE
- DESIGN_NAME
- TEMPLATE_DESCRIPTION
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- SEED_DATA_SOURCE

## Query

```sql
SELECT B.MESSAGE_DESIGN_ID, B.MESSAGE_DESIGN_CODE, B.DESIGN_TYPE_CODE, B.DESIGN_STATUS_CODE, B.DESIGN_EDITOR_TYPE_CODE, TL.DESIGN_NAME, TL.TEMPLATE_DESCRIPTION, B.MODULE_ID, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN, B.SEED_DATA_SOURCE FROM IRC_MSG_DESIGNS_B B, IRC_MSG_DESIGNS_TL TL WHERE B.MESSAGE_DESIGN_ID = TL.MESSAGE_DESIGN_ID AND TL.LANGUAGE = USERENV('LANG')
```
