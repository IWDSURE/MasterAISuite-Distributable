# IRC_PHASES_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircphasesvl-5303.html#ircphasesvl-5303](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircphasesvl-5303.html#ircphasesvl-5303)

## Columns

- PHASE_ID
- NAME
- DESCRIPTION
- CODE
- TYPE_CODE
- MODULE_ID
- SEQUENCE_NUMBER
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT PHASESB.PHASE_ID, PHASESTL.NAME, PHASESTL.DESCRIPTION, PHASESB.CODE, PHASESB.TYPE_CODE, PHASESB.MODULE_ID, PHASESB.SEQUENCE_NUMBER, PHASESB.OBJECT_VERSION_NUMBER, PHASESB.CREATED_BY, PHASESB.CREATION_DATE, PHASESB.LAST_UPDATED_BY, PHASESB.LAST_UPDATE_DATE, PHASESB.LAST_UPDATE_LOGIN FROM IRC_PHASES_B PHASESB, IRC_PHASES_TL PHASESTL WHERE PHASESB.PHASE_ID = PHASESTL.PHASE_ID AND PHASESTL.LANGUAGE = USERENV('LANG')
```
