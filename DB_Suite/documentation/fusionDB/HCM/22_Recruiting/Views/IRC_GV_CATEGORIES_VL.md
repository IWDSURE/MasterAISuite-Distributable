# IRC_GV_CATEGORIES_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvcategoriesvl-5245.html#ircgvcategoriesvl-5245](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvcategoriesvl-5245.html#ircgvcategoriesvl-5245)

## Columns

- CATEGORY_ID
- CATEGORY_CODE
- GV_CONTEXT_CODE
- MULTI_ROW_FLAG
- NAME
- DESCRIPTION
- PROFILE_CONTENT_TYPE_ID
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT CATEGORIES_B.CATEGORY_ID CATEGORY_ID, CATEGORIES_B.CATEGORY_CODE CATEGORY_CODE, CATEGORIES_B.GV_CONTEXT_CODE, CATEGORIES_B.MULTI_ROW_FLAG, CATEGORIES_TL.NAME NAME, CATEGORIES_TL.DESCRIPTION DESCRIPTION, CATEGORIES_B.PROFILE_CONTENT_TYPE_ID PROFILE_CONTENT_TYPE_ID, CATEGORIES_B.MODULE_ID MODULE_ID, CATEGORIES_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, CATEGORIES_B.CREATED_BY CREATED_BY, CATEGORIES_B.CREATION_DATE CREATION_DATE, CATEGORIES_B.LAST_UPDATED_BY LAST_UPDATED_BY, CATEGORIES_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, CATEGORIES_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_GV_CATEGORIES_B CATEGORIES_B, IRC_GV_CATEGORIES_TL CATEGORIES_TL WHERE CATEGORIES_B.CATEGORY_ID = CATEGORIES_TL.CATEGORY_ID AND CATEGORIES_TL.LANGUAGE = USERENV('LANG')
```
