## Query

```sql
SELECT IRC_LC_ACTIONS_B.ACTION_ID ACTION_ID, IRC_LC_ACTIONS_B.TYPE_CODE TYPE_CODE, IRC_LC_ACTIONS_B.ACTION_CODE ACTION_CODE, IRC_LC_ACTIONS_B.ACTION_TYPE ACTION_TYPE, IRC_LC_ACTIONS_B.ACTION_SETUP_CLASS_NAME ACTION_SETUP_CLASS_NAME, IRC_LC_ACTIONS_B.ACTION_EXEC_CLASS_NAME ACTION_EXEC_CLASS_NAME, IRC_LC_ACTIONS_B.MODULE_ID MODULE_ID, IRC_LC_ACTIONS_TL.ACTION_NAME ACTION_NAME, IRC_LC_ACTIONS_TL.ACTION_DESCRIPTION ACTION_DESCRIPTION, IRC_LC_ACTIONS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, IRC_LC_ACTIONS_B.CREATED_BY CREATED_BY, IRC_LC_ACTIONS_B.CREATION_DATE CREATION_DATE, IRC_LC_ACTIONS_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_LC_ACTIONS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_LC_ACTIONS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, IRC_LC_ACTIONS_B.OBJECT_STATUS OBJECT_STATUS, IRC_LC_ACTIONS_B.ASYNC_ENABLED FROM IRC_LC_ACTIONS_B, IRC_LC_ACTIONS_TL WHERE IRC_LC_ACTIONS_B.ACTION_ID = IRC_LC_ACTIONS_TL.ACTION_ID AND IRC_LC_ACTIONS_TL.LANGUAGE = USERENV('LANG')
```
