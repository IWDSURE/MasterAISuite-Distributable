# IRC_SYNONYMS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircsynonymsvl-5402.html#ircsynonymsvl-5402](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircsynonymsvl-5402.html#ircsynonymsvl-5402)

## Columns

- SYNONYM_ID
- TYPE
- SYNONYM_CODE
- SYNONYM_NAME
- FA_REFERENCE
- SYNONYM_VALUE
- OBJECT_STATUS
- ADDITIONAL_VALUE
- SEED_DATA_SOURCE
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT IRC_SYNONYMS_B.SYNONYM_ID SYNONYM_ID, IRC_SYNONYMS_B.TYPE TYPE, IRC_SYNONYMS_B.SYNONYM_CODE SYNONYM_CODE, IRC_SYNONYMS_B.SYNONYM_NAME SYNONYM_NAME, IRC_SYNONYMS_B.FA_REFERENCE FA_REFERENCE, IRC_SYNONYMS_TL.SYNONYM_VALUE SYNONYM_VALUE, IRC_SYNONYMS_B.OBJECT_STATUS OBJECT_STATUS, IRC_SYNONYMS_B.ADDITIONAL_VALUE ADDITIONAL_VALUE, IRC_SYNONYMS_B.SEED_DATA_SOURCE SEED_DATA_SOURCE, IRC_SYNONYMS_B.MODULE_ID MODULE_ID, IRC_SYNONYMS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, IRC_SYNONYMS_B.CREATED_BY CREATED_BY, IRC_SYNONYMS_B.CREATION_DATE CREATION_DATE, IRC_SYNONYMS_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_SYNONYMS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_SYNONYMS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_SYNONYMS_B, IRC_SYNONYMS_TL WHERE IRC_SYNONYMS_B.SYNONYM_ID = IRC_SYNONYMS_TL.SYNONYM_ID AND IRC_SYNONYMS_TL.LANGUAGE = USERENV('LANG')
```
