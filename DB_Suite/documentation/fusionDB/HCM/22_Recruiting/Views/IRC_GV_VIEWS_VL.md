# IRC_GV_VIEWS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvviewsvl-7215.html#ircgvviewsvl-7215](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvviewsvl-7215.html#ircgvviewsvl-7215)

## Columns

- VIEW_ID
- NAME
- DESCRIPTION
- VIEW_CODE
- GV_CONTEXT_CODE
- VIEW_OWNER
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT VIEWS_B.VIEW_ID VIEW_ID, VIEWS_TL.NAME NAME, VIEWS_TL.DESCRIPTION DESCRIPTION, VIEWS_B.VIEW_CODE VIEW_CODE, VIEWS_B.GV_CONTEXT_CODE, VIEWS_B.VIEW_OWNER VIEW_OWNER, VIEWS_B.MODULE_ID MODULE_ID, VIEWS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, VIEWS_B.CREATED_BY CREATED_BY, VIEWS_B.CREATION_DATE CREATION_DATE, VIEWS_B.LAST_UPDATED_BY LAST_UPDATED_BY, VIEWS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, VIEWS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_GV_VIEWS_B VIEWS_B, IRC_GV_VIEWS_TL VIEWS_TL WHERE VIEWS_B.VIEW_ID = VIEWS_TL.VIEW_ID AND VIEWS_TL.LANGUAGE = USERENV('LANG')
```
