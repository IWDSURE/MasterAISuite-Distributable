# IRC_WHATSAPP_TEMPLATE_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircwhatsapptemplatevl-6175.html#ircwhatsapptemplatevl-6175](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircwhatsapptemplatevl-6175.html#ircwhatsapptemplatevl-6175)

## Columns

- WA_TEMPLATE_ID
- WABA_TEMPLATE_NAME
- WA_TEMPLATE_DISPLAY_NAME
- TEMPLATE_STATUS
- DESCRIPTION_TYPE_CODE
- WABA_CATEGORY
- WABA_TEMPLATE_CONTENT
- LANGUAGE
- DESC_VERSION_ID
- PRODUCT_CONTEXT
- TEMPLATE_SOURCE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.WA_TEMPLATE_ID WA_TEMPLATE_ID, B.WABA_TEMPLATE_NAME WABA_TEMPLATE_NAME, TL.WA_TEMPLATE_DISPLAY_NAME WA_TEMPLATE_DISPLAY_NAME, B.TEMPLATE_STATUS TEMPLATE_STATUS, B.DESCRIPTION_TYPE_CODE DESCRIPTION_TYPE_CODE, B.WABA_CATEGORY WABA_CATEGORY, TL.WABA_TEMPLATE_CONTENT WABA_TEMPLATE_CONTENT, TL.LANGUAGE LANGUAGE, B.DESC_VERSION_ID DESC_VERSION_ID, B.PRODUCT_CONTEXT PRODUCT_CONTEXT, B.TEMPLATE_SOURCE TEMPLATE_SOURCE, B.CREATED_BY CREATED_BY, B.CREATION_DATE CREATION_DATE, B.LAST_UPDATED_BY LAST_UPDATED_BY, B.LAST_UPDATE_DATE LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_WHATSAPP_TEMPLATE_B B, IRC_WHATSAPP_TEMPLATE_TL TL WHERE B.WA_TEMPLATE_ID = TL.WA_TEMPLATE_ID AND TL.LANGUAGE = USERENV('LANG')
```
