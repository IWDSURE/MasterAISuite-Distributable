# IRC_LABELS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclabelsvl-8190.html#irclabelsvl-8190](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclabelsvl-8190.html#irclabelsvl-8190)

## Columns

- LABEL_ID
- ENTITY_TYPE
- LABEL_NAME
- LABEL_CODE
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN
- OBJECT_VERSION_NUMBER

## Query

```sql
SELECT IRC_LABELS_B.LABEL_ID LABEL_ID, IRC_LABELS_B.ENTITY_TYPE ENTITY_TYPE, IRC_LABELS_TL.LABEL_NAME LABEL_NAME, IRC_LABELS_B.LABEL_CODE LABEL_CODE, IRC_LABELS_B.CREATED_BY CREATED_BY, IRC_LABELS_B.CREATION_DATE CREATION_DATE, IRC_LABELS_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_LABELS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_LABELS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, IRC_LABELS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER FROM IRC_LABELS_B, IRC_LABELS_TL WHERE IRC_LABELS_B.LABEL_ID = IRC_LABELS_TL.LABEL_ID AND IRC_LABELS_TL.LANGUAGE = USERENV('LANG')
```
