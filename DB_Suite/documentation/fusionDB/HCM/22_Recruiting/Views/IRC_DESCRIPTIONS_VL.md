# IRC_DESCRIPTIONS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdescriptionsvl-8780.html#ircdescriptionsvl-8780](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircdescriptionsvl-8780.html#ircdescriptionsvl-8780)

## Columns

- DESCRIPTION_ID
- DESCRIPTION_CODE
- DESCRIPTION_TYPE_CODE
- VISIBILITY_CODE
- STATUS_CODE
- NAME
- MODULE_ID
- COLLABORATOR_RESP_TYPE_LIST
- CAPTURE_INTERACTION
- USE_NO_REPLY_FLAG
- INTERACTION_CONTENT
- MESSAGE_TRACKING_START_DATE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT B.DESCRIPTION_ID, B.DESCRIPTION_CODE, B.DESCRIPTION_TYPE_CODE, B.VISIBILITY_CODE, B.STATUS_CODE, TL.NAME, B.MODULE_ID MODULE_ID, B.COLLABORATOR_RESP_TYPE_LIST COLLABORATOR_RESP_TYPE_LIST, B.CAPTURE_INTERACTION CAPTURE_INTERACTION, B.USE_NO_REPLY_FLAG, TL.INTERACTION_CONTENT INTERACTION_CONTENT, B.MESSAGE_TRACKING_START_DATE, B.OBJECT_VERSION_NUMBER, B.CREATED_BY, B.CREATION_DATE, B.LAST_UPDATED_BY, B.LAST_UPDATE_DATE, B.LAST_UPDATE_LOGIN FROM IRC_DESCRIPTIONS_B B, IRC_DESCRIPTIONS_TL TL WHERE B.DESCRIPTION_ID = TL.DESCRIPTION_ID AND TL.LANGUAGE = USERENV('LANG')
```
