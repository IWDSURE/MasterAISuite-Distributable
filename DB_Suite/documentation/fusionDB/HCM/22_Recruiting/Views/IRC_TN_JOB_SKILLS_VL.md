# IRC_TN_JOB_SKILLS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irctnjobskillsvl-3982.html#irctnjobskillsvl-3982](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irctnjobskillsvl-3982.html#irctnjobskillsvl-3982)

## Columns

- TN_JOB_SKILL_ID
- TN_JOB_ID
- FA_SKILL_ID
- FA_SECTION_ID
- SECTION_NAME
- SKILL_NAME
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT TNJOBSKILLSB.TN_JOB_SKILL_ID, TNJOBSKILLSB.TN_JOB_ID, TNJOBSKILLSB.FA_SKILL_ID, TNJOBSKILLSB.FA_SECTION_ID, TNJOBSKILLSTL.SECTION_NAME, TNJOBSKILLSTL.SKILL_NAME, TNJOBSKILLSB.OBJECT_VERSION_NUMBER, TNJOBSKILLSB.CREATED_BY, TNJOBSKILLSB.CREATION_DATE, TNJOBSKILLSB.LAST_UPDATED_BY, TNJOBSKILLSB.LAST_UPDATE_DATE, TNJOBSKILLSB.LAST_UPDATE_LOGIN FROM IRC_TN_JOB_SKILLS_B TNJOBSKILLSB, IRC_TN_JOB_SKILLS_TL TNJOBSKILLSTL WHERE TNJOBSKILLSB.TN_JOB_SKILL_ID = TNJOBSKILLSTL.TN_JOB_SKILL_ID AND TNJOBSKILLSTL.LANGUAGE = USERENV('LANG')
```
