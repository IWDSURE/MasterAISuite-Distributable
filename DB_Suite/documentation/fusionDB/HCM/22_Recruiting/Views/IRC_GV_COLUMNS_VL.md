**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvcolumnsvl-4888.html#ircgvcolumnsvl-4888](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircgvcolumnsvl-4888.html#ircgvcolumnsvl-4888)

- COLUMN_ID
- VIEW_ID
- NAME
- DESCRIPTION
- COLUMN_SEQUENCE
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT COLUMNS_B.COLUMN_ID COLUMN_ID, COLUMNS_B.VIEW_ID VIEW_ID, COLUMNS_TL.NAME NAME, COLUMNS_TL.DESCRIPTION DESCRIPTION, COLUMNS_B.COLUMN_SEQUENCE COLUMN_SEQUENCE, COLUMNS_B.MODULE_ID MODULE_ID, COLUMNS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, COLUMNS_B.CREATED_BY CREATED_BY, COLUMNS_B.CREATION_DATE CREATION_DATE, COLUMNS_B.LAST_UPDATED_BY LAST_UPDATED_BY, COLUMNS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, COLUMNS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_GV_COLUMNS_B COLUMNS_B, IRC_GV_COLUMNS_TL COLUMNS_TL WHERE COLUMNS_B.COLUMN_ID = COLUMNS_TL.COLUMN_ID AND COLUMNS_TL.LANGUAGE = USERENV('LANG')
```
