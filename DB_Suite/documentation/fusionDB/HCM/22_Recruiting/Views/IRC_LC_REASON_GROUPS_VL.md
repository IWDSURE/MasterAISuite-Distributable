# IRC_LC_REASON_GROUPS_VL

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclcreasongroupsvl-5964.html#irclcreasongroupsvl-5964](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/irclcreasongroupsvl-5964.html#irclcreasongroupsvl-5964)

- REASON_GROUP_CODE
- REASON_GROUP_STATUS_CODE
- REASON_GROUP_NAME
- REASON_GROUP_COMMENTS
- OBJECT_VERSION_NUMBER

- LAST_UPDATE_DATE

## Query

```sql
SELECT IRC_LC_REASON_GROUPS_B.REASON_GROUP_ID REASON_GROUP_ID, IRC_LC_REASON_GROUPS_B.REASON_GROUP_CODE REASON_GROUP_CODE, IRC_LC_REASON_GROUPS_B.REASON_GROUP_STATUS_CODE REASON_GROUP_STATUS_CODE, IRC_LC_REASON_GROUPS_TL.REASON_GROUP_NAME REASON_GROUP_NAME, IRC_LC_REASON_GROUPS_TL.REASON_GROUP_COMMENTS REASON_GROUP_COMMENTS, IRC_LC_REASON_GROUPS_B.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, IRC_LC_REASON_GROUPS_B.CREATED_BY CREATED_BY, IRC_LC_REASON_GROUPS_B.CREATION_DATE CREATION_DATE, IRC_LC_REASON_GROUPS_B.LAST_UPDATED_BY LAST_UPDATED_BY, IRC_LC_REASON_GROUPS_B.LAST_UPDATE_DATE LAST_UPDATE_DATE, IRC_LC_REASON_GROUPS_B.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_LC_REASON_GROUPS_B, IRC_LC_REASON_GROUPS_TL WHERE IRC_LC_REASON_GROUPS_B.REASON_GROUP_ID = IRC_LC_REASON_GROUPS_TL.REASON_GROUP_ID AND IRC_LC_REASON_GROUPS_TL.LANGUAGE = USERENV('LANG')
```
