# IRC_CC_ELEMENTS_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircccelementsvl-8370.html#ircccelementsvl-8370](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircccelementsvl-8370.html#ircccelementsvl-8370)

## Columns

- ELEMENT_ID
- ELEMENT_NUMBER
- PARENT_ELEMENT_NUMBER
- ELEMENT_TYPE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- LAST_UPDATE_DATE
- ELEMENT_ORDER
- CONTENT

## Query

```sql
SELECT IRC_CC_ELEMENTS_B.ELEMENT_ID, IRC_CC_ELEMENTS_B.ELEMENT_NUMBER, IRC_CC_ELEMENTS_B.PARENT_ELEMENT_NUMBER, IRC_CC_ELEMENTS_B.ELEMENT_TYPE, IRC_CC_ELEMENTS_B.OBJECT_VERSION_NUMBER, IRC_CC_ELEMENTS_B.CREATED_BY, IRC_CC_ELEMENTS_B.CREATION_DATE, IRC_CC_ELEMENTS_B.LAST_UPDATED_BY, IRC_CC_ELEMENTS_B.LAST_UPDATE_LOGIN, IRC_CC_ELEMENTS_B.LAST_UPDATE_DATE, IRC_CC_ELEMENTS_B.ELEMENT_ORDER, IRC_CC_ELEMENTS_TL.CONTENT FROM IRC_CC_ELEMENTS_B, IRC_CC_ELEMENTS_TL WHERE IRC_CC_ELEMENTS_B.ELEMENT_ID = IRC_CC_ELEMENTS_TL.ELEMENT_ID AND IRC_CC_ELEMENTS_TL.LANGUAGE = USERENV('LANG')
```
