# IRC_STATES_VL

## Details

**Schema:** FUSION

**Object owner:** IRC

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircstatesvl-8854.html#ircstatesvl-8854](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/ircstatesvl-8854.html#ircstatesvl-8854)

## Columns

- STATE_ID
- NAME
- DESCRIPTION
- CODE
- TYPE_CODE
- IS_TERMINAL_FLAG
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT STATEB.STATE_ID STATE_ID, STATETL.NAME NAME, STATETL.DESCRIPTION DESCRIPTION, STATEB.CODE CODE, STATEB.TYPE_CODE TYPE_CODE, STATEB.IS_TERMINAL_FLAG IS_TERMINAL_FLAG, STATEB.MODULE_ID, STATEB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, STATEB.CREATED_BY CREATED_BY, STATEB.CREATION_DATE CREATION_DATE, STATEB.LAST_UPDATED_BY LAST_UPDATED_BY, STATEB.LAST_UPDATE_DATE LAST_UPDATE_DATE, STATEB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM IRC_STATES_B STATEB, IRC_STATES_TL STATETL WHERE STATEB.STATE_ID = STATETL.STATE_ID AND STATETL.LANGUAGE = USERENV('LANG')
```
