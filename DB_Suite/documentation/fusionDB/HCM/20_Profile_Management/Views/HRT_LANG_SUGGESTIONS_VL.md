# HRT_LANG_SUGGESTIONS_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtlangsuggestionsvl-5002.html#hrtlangsuggestionsvl-5002](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtlangsuggestionsvl-5002.html#hrtlangsuggestionsvl-5002)

## Columns

- SUGGESTION_ID
- ENTERPRISE_ID
- SUGGESTION_CODE
- SUPPLIER_CODE
- SUGGESTION_TEXT
- PROPER_USE_TEXT
- IMPROPER_USE_TEXT
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT LSB.SUGGESTION_ID, LSB.ENTERPRISE_ID, LSB.SUGGESTION_CODE, LSB.SUPPLIER_CODE, LSTL.SUGGESTION_TEXT, LSTL.PROPER_USE_TEXT, LSTL.IMPROPER_USE_TEXT, LSB.MODULE_ID, LSB.OBJECT_VERSION_NUMBER, LSB.CREATED_BY, LSB.CREATION_DATE, LSB.LAST_UPDATE_DATE, LSB.LAST_UPDATED_BY, LSB.LAST_UPDATE_LOGIN FROM HRT_LANG_SUGGESTIONS_B LSB, HRT_LANG_SUGGESTIONS_TL LSTL WHERE LSB.SUGGESTION_ID = LSTL.SUGGESTION_ID AND LSB.ENTERPRISE_ID = LSTL.ENTERPRISE_ID AND LSTL.LANGUAGE = USERENV('LANG')
```
