# HRT_N_B_ASGN_SECURED_LIST_V

**Schema:** FUSION

- LAST_UPDATE_LOGIN

- EFFECTIVE_END_DATE

## Query

```sql
SELECT PR.PROFILE_ID, PR.BUSINESS_GROUP_ID, PR.PROFILE_TYPE_ID, PR.PROFILE_CODE, PR.PROFILE_STATUS_CODE, PR.PROFILE_USAGE_CODE, PR.PARTY_ID, PR.OWNER_PERSON_ID, PR.OBJECT_VERSION_NUMBER, PR.CREATED_BY, PR.CREATION_DATE, PR.LAST_UPDATED_BY, PR.LAST_UPDATE_DATE, PR.LAST_UPDATE_LOGIN, ProfileTypeSectionPEO.CONTENT_TYPE_ID, ProfileTypeSectionPEO.SECTION_CONTEXT, ProfileTypeSectionPEO.SECTION_ID, PersonDPEO.PERSON_ID, PersonDPEO.EFFECTIVE_START_DATE, PersonDPEO.EFFECTIVE_END_DATE FROM HRT_PROFILES_B PR, HRT_PROFILE_TYP_SECTIONS ProfileTypeSectionPEO, PER_ALL_PEOPLE_F PersonDPEO WHERE ProfileTypeSectionPEO.PROFILE_TYPE_ID = PR.PROFILE_TYPE_ID AND PR.PROFILE_TYPE_ID = 1 AND ProfileTypeSectionPEO.BUSINESS_GROUP_ID = PR.BUSINESS_GROUP_ID AND PersonDPEO.BUSINESS_GROUP_ID = PR.BUSINESS_GROUP_ID AND PersonDPEO.PERSON_ID = PR.PERSON_ID AND ProfileTypeSectionPEO.SECTION_CONTEXT = 'PERSON_N_BOX_ASSIGNMENT' AND TRUNC(SYSDATE) BETWEEN PersonDPEO.EFFECTIVE_START_DATE AND PersonDPEO.EFFECTIVE_END_DATE
```
