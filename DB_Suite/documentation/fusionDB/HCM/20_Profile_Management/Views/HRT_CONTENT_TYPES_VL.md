# HRT_CONTENT_TYPES_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtcontenttypesvl-5860.html#hrtcontenttypesvl-5860](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtcontenttypesvl-5860.html#hrtcontenttypesvl-5860)

## Columns

- CONTENT_TYPE_ID
- BUSINESS_GROUP_ID
- CONTENT_TYPE_NAME
- CONTENT_DESCRIPTION
- CONTEXT_NAME
- MODULE_ID
- FREE_FORM_FLAG
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT CT.CONTENT_TYPE_ID, CT.BUSINESS_GROUP_ID, CTTL.CONTENT_TYPE_NAME, CTTL.CONTENT_DESCRIPTION, CT.CONTEXT_NAME, CT.MODULE_ID, CT.FREE_FORM_FLAG, CT.OBJECT_VERSION_NUMBER, CT.CREATED_BY, CT.CREATION_DATE, CT.LAST_UPDATE_DATE, CT.LAST_UPDATED_BY, CT.LAST_UPDATE_LOGIN FROM HRT_CONTENT_TYPES_B CT, HRT_CONTENT_TYPES_TL CTTL WHERE CT.CONTENT_TYPE_ID = CTTL.CONTENT_TYPE_ID AND CT.BUSINESS_GROUP_ID = CTTL.BUSINESS_GROUP_ID AND CTTL.LANGUAGE = USERENV('LANG')
```
