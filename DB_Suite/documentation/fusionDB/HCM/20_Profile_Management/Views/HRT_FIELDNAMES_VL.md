# HRT_FIELDNAMES_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtfieldnamesvl-4133.html#hrtfieldnamesvl-4133](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtfieldnamesvl-4133.html#hrtfieldnamesvl-4133)

## Columns

- FIELD_NAME_ID
- BUSINESS_GROUP_ID
- FIELD_NAME
- DEFAULT_FIELD_LABEL
- DEFAULT_VALUESET
- USED_IN_CODE
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT FLD.FIELD_NAME_ID, FLD.BUSINESS_GROUP_ID, FLD.FIELD_NAME, FTL.DEFAULT_FIELD_LABEL, FLD.DEFAULT_VALUESET, FLD.USED_IN_CODE, FLD.MODULE_ID, FLD.OBJECT_VERSION_NUMBER, FLD.CREATED_BY, FLD.CREATION_DATE, FLD.LAST_UPDATE_DATE, FLD.LAST_UPDATED_BY, FLD.LAST_UPDATE_LOGIN FROM HRT_FIELDNAMES_B FLD, HRT_FIELDNAMES_TL FTL WHERE FLD.FIELD_NAME_ID = FTL.FIELD_NAME_ID AND FLD.BUSINESS_GROUP_ID = FTL.BUSINESS_GROUP_ID AND FTL.LANGUAGE = USERENV('LANG')
```
