# HRT_CONTENT_PROPS_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtcontentpropsvl-4390.html#hrtcontentpropsvl-4390](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtcontentpropsvl-4390.html#hrtcontentpropsvl-4390)

## Columns

- CONTENT_TYPE_PROP_ID
- BUSINESS_GROUP_ID
- CONTENT_TYPE_ID
- FIELD_NAME
- ATTRIBUTE_LABEL
- COLUMN_NAME
- DEFAULT_VALUE
- REQUIRED_FLAG
- DISPLAY_CODE
- VIEW_ATTRB_NAME
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT CP.CONTENT_TYPE_PROP_ID, CP.BUSINESS_GROUP_ID, CP.CONTENT_TYPE_ID, CP.FIELD_NAME, CPTL.ATTRIBUTE_LABEL, CP.COLUMN_NAME, CP.DEFAULT_VALUE, CP.REQUIRED_FLAG, CP.DISPLAY_CODE, CP.VIEW_ATTRB_NAME, CP.MODULE_ID, CP.OBJECT_VERSION_NUMBER, CP.CREATED_BY, CP.CREATION_DATE, CP.LAST_UPDATE_DATE, CP.LAST_UPDATED_BY, CP.LAST_UPDATE_LOGIN FROM HRT_CONTENT_PROPS_B CP, HRT_CONTENT_PROPS_TL CPTL WHERE CP.CONTENT_TYPE_PROP_ID = CPTL.CONTENT_TYPE_PROP_ID AND CP.BUSINESS_GROUP_ID = CPTL.BUSINESS_GROUP_ID AND CPTL.LANGUAGE = USERENV('LANG')
```
