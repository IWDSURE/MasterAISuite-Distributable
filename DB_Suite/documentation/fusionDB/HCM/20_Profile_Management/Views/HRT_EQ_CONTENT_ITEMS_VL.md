# HRT_EQ_CONTENT_ITEMS_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrteqcontentitemsvl-4409.html#hrteqcontentitemsvl-4409](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrteqcontentitemsvl-4409.html#hrteqcontentitemsvl-4409)

## Columns

- EQ_CONTENT_ITEM_ID
- SGUID
- BUSINESS_GROUP_ID
- MODULE_ID
- GLOBAL_ID
- NAME
- CONTENT_TYPE_ID
- CONTENT_VALUE_SET_ID
- CONTENT_SUPPLIER_CODE
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT VB.EQ_CONTENT_ITEM_ID, VB.SGUID, VB.BUSINESS_GROUP_ID, VB.MODULE_ID, VB.GLOBAL_ID, VTL.NAME, VB.CONTENT_TYPE_ID, VB.CONTENT_VALUE_SET_ID, VB.CONTENT_SUPPLIER_CODE, VB.OBJECT_VERSION_NUMBER, VB.CREATED_BY, VB.CREATION_DATE, VB.LAST_UPDATED_BY, VB.LAST_UPDATE_DATE, VB.LAST_UPDATE_LOGIN FROM HRT_EQ_CONTENT_ITEMS_B VB, HRT_EQ_CONTENT_ITEMS_TL VTL WHERE VB.EQ_CONTENT_ITEM_ID = VTL.EQ_CONTENT_ITEM_ID AND VB.BUSINESS_GROUP_ID = VTL.BUSINESS_GROUP_ID AND VTL.LANGUAGE = USERENV('LANG')
```
