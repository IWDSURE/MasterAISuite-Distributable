# HRT_EXTERNAL_FIELDS_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtexternalfieldsvl-5034.html#hrtexternalfieldsvl-5034](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtexternalfieldsvl-5034.html#hrtexternalfieldsvl-5034)

## Columns

- EXTERNAL_FIELD_ID
- BUSINESS_GROUP_ID
- SOURCE_ID
- FIELD_CODE
- FIELD_DATATYPE
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATE_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_LOGIN
- FIELD_NAME
- FIELD_DESCRIPTION

## Query

```sql
SELECT SD.EXTERNAL_FIELD_ID EXTERNAL_FIELD_ID, SD.BUSINESS_GROUP_ID BUSINESS_GROUP_ID, SD.SOURCE_ID SOURCE_ID, SD.FIELD_CODE FIELD_CODE, SD.FIELD_DATATYPE FIELD_DATATYPE, SD.MODULE_ID MODULE_ID, SD.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, SD.CREATED_BY CREATED_BY, SD.CREATION_DATE CREATION_DATE, SD.LAST_UPDATE_DATE LAST_UPDATE_DATE, SD.LAST_UPDATED_BY LAST_UPDATED_BY, SD.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN, SDTL.FIELD_NAME FIELD_NAME, SDTL.FIELD_DESCRIPTION FIELD_DESCRIPTION FROM HRT_EXTERNAL_FIELDS_B SD, HRT_EXTERNAL_FIELDS_TL SDTL WHERE SD.EXTERNAL_FIELD_ID = SDTL.EXTERNAL_FIELD_ID AND SD.BUSINESS_GROUP_ID = SDTL.BUSINESS_GROUP_ID AND SDTL.LANGUAGE = USERENV('LANG')
```
