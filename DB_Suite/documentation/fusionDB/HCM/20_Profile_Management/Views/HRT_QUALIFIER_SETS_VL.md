# HRT_QUALIFIER_SETS_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtqualifiersetsvl-8391.html#hrtqualifiersetsvl-8391](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtqualifiersetsvl-8391.html#hrtqualifiersetsvl-8391)

## Columns

- QUALIFIER_SET_ID
- BUSINESS_GROUP_ID
- DESCRIPTION
- CONTENT_TYPE_ID
- SECTION_ID
- QUALIFIER_SET_CODE
- QUALIFIER_TYPE_CODE
- KEY_TABLE_NAME
- MODULE_ID
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT QS.QUALIFIER_SET_ID, QS.BUSINESS_GROUP_ID, QST.DESCRIPTION, QS.CONTENT_TYPE_ID, QS.SECTION_ID, QS.QUALIFIER_SET_CODE, QS.QUALIFIER_TYPE_CODE, QS.KEY_TABLE_NAME, QS.MODULE_ID, QS.OBJECT_VERSION_NUMBER, QS.CREATED_BY, QS.CREATION_DATE, QS.LAST_UPDATED_BY, QS.LAST_UPDATE_DATE, QS.LAST_UPDATE_LOGIN FROM HRT_QUALIFIER_SETS_B QS, HRT_QUALIFIER_SETS_TL QST WHERE QS.QUALIFIER_SET_ID = QST.QUALIFIER_SET_ID AND QS.BUSINESS_GROUP_ID = QST.BUSINESS_GROUP_ID AND QST.LANGUAGE = USERENV('LANG')
```
