# HRT_PROFILE_TYPE_TEXT_VL

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** VIEW

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtprofiletypetextvl-6609.html#hrtprofiletypetextvl-6609](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrtprofiletypetextvl-6609.html#hrtprofiletypetextvl-6609)

## Columns

- TEXT_ID
- BUSINESS_GROUP_ID
- PROFILE_TYPE_ID
- TEXT_CODE
- MODULE_ID
- DESCRIPTION
- OBJECT_VERSION_NUMBER
- CREATED_BY
- CREATION_DATE
- LAST_UPDATED_BY
- LAST_UPDATE_DATE
- LAST_UPDATE_LOGIN

## Query

```sql
SELECT PTTB.TEXT_ID TEXT_ID, PTTB.BUSINESS_GROUP_ID BUSINESS_GROUP_ID, PTTB.PROFILE_TYPE_ID PROFILE_TYPE_ID, PTTB.TEXT_CODE TEXT_CODE, PTTB.MODULE_ID MODULE_ID, PTTTL.DESCRIPTION DESCRIPTION, PTTB.OBJECT_VERSION_NUMBER OBJECT_VERSION_NUMBER, PTTB.CREATED_BY CREATED_BY, PTTB.CREATION_DATE CREATION_DATE, PTTB.LAST_UPDATED_BY LAST_UPDATED_BY, PTTB.LAST_UPDATE_DATE LAST_UPDATE_DATE, PTTB.LAST_UPDATE_LOGIN LAST_UPDATE_LOGIN FROM HRT_PROFILE_TYPE_TEXT_B PTTB, HRT_PROFILE_TYPE_TEXT_TL PTTTL WHERE PTTB.TEXT_ID = PTTTL.TEXT_ID AND PTTB.BUSINESS_GROUP_ID = PTTTL.BUSINESS_GROUP_ID AND PTTTL.LANGUAGE = USERENV('LANG')
```
