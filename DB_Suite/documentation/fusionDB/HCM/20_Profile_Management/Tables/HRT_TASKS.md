# HRT_TASKS

This table is for storing the tasks and is part of HCM Tasks Functionality.

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** TABLE

**Tablespace:** FUSION_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttasks-23903.html#hrttasks-23903](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttasks-23903.html#hrttasks-23903)

## Primary Key

| Name | Columns |
|------|----------|
| HRT_TASKS_PK | TASK_ID, ENTERPRISE_ID |
