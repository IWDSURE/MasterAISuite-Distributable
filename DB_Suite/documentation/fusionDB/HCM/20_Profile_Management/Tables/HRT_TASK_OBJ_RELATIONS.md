# HRT_TASK_OBJ_RELATIONS

This Table stores the relations among the objects involved in  HCM Tasks Functionality.

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** TABLE

**Tablespace:** FUSION_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttaskobjrelations-16037.html#hrttaskobjrelations-16037](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttaskobjrelations-16037.html#hrttaskobjrelations-16037)

| Name | Columns |
