# HRT_TASK_USERS

This table is for storing the users assigned to specific tasks involved in HCM Tasks Functionality.

## Details

**Schema:** FUSION

**Object owner:** HRT

**Object type:** TABLE

**Tablespace:** FUSION_TS_TX_DATA

**Source:** [https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttaskusers-21050.html#hrttaskusers-21050](https://docs.oracle.com/en/cloud/saas/human-resources/oedmh/hrttaskusers-21050.html#hrttaskusers-21050)

## Primary Key

| Name | Columns |
|------|----------|
| HRT_TASKS_USER_PK | TASK_USER_ID, ENTERPRISE_ID |
